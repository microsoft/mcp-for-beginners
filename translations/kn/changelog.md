# ಬದಲಾವಣೆಗಳ ಚರಿತ್ರೆ: ಆರಂಭಿಕರಿಗಾಗಿ MCP ಪಠ್ಯಕ್ರಮ

ಈ ದಾಖಲೆ ಆರಂಭಿಕರಿಗಾಗಿ ಮಾದರಿ ಸನ್ನಿವೇಶ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಪಠ್ಯಕ್ರಮದಲ್ಲಿ ಮಾಡಲಾದ ಎಲ್ಲಾ ಪ್ರಮುಖ ಬದಲಾವಣೆಗಳ ದಾಖಲೆ ಆಗಿದೆ. ಬದಲಾವಣೆಗಳನ್ನು ವಿಲಂಬಿತ ಕಾಲಕ್ರಮದಲ್ಲಿ (ಹೊಸ ಬದಲಾವಣೆಗಳು ಮೊದಲು) ದಾಖಲಿಸಲಾಗಿದೆ.

## ಡಿಸೆಂಬರ್ 18, 2025

### ಭದ್ರತಾ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ನವೀಕರಣ - MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25

#### MCP ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು (02-Security/mcp-best-practices.md) - ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಆವೃತ್ತಿ ನವೀಕರಣ
- **ಪ್ರೋಟೋಕಾಲ್ ಆವೃತ್ತಿ ನವೀಕರಣ**: ಇತ್ತೀಚಿನ MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25 (ನವೆಂಬರ್ 25, 2025 ಬಿಡುಗಡೆ) ಗೆ ಉಲ್ಲೇಖ ನವೀಕರಿಸಲಾಗಿದೆ
  - ಎಲ್ಲಾ ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಆವೃತ್ತಿ ಉಲ್ಲೇಖಗಳನ್ನು 2025-06-18 ರಿಂದ 2025-11-25 ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
  - ದಾಖಲೆ ದಿನಾಂಕ ಉಲ್ಲೇಖಗಳನ್ನು ಆಗಸ್ಟ್ 18, 2025 ರಿಂದ ಡಿಸೆಂಬರ್ 18, 2025 ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
  - ಎಲ್ಲಾ ಸ್ಪೆಸಿಫಿಕೇಶನ್ URL ಗಳು ಪ್ರಸ್ತುತ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಕಡೆಗೆ ಸೂಚಿಸುತ್ತವೆ ಎಂದು ಪರಿಶೀಲಿಸಲಾಗಿದೆ
- **ವಿಷಯ ಪರಿಶೀಲನೆ**: ಇತ್ತೀಚಿನ ಮಾನದಂಡಗಳ ವಿರುದ್ಧ ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳ ಸಮಗ್ರ ಪರಿಶೀಲನೆ
  - **ಮೈಕ್ರೋಸಾಫ್ಟ್ ಭದ್ರತಾ ಪರಿಹಾರಗಳು**: ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್‌ಗಳು (ಹಿಂದೆ "ಜೆಲ್ಬ್ರೇಕ್ ಅಪಾಯ ಪತ್ತೆ" ಎಂದು ಕರೆಯಲ್ಪಟ್ಟವು), ಅಜೂರ್ ವಿಷಯ ಸುರಕ್ಷತೆ, ಮೈಕ್ರೋಸಾಫ್ಟ್ ಎಂಟ್ರಾ ID, ಮತ್ತು ಅಜೂರ್ ಕೀ ವಾಲ್ಟ್ ನ ಪ್ರಸ್ತುತ ಪದಬಳಕೆ ಮತ್ತು ಲಿಂಕ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸಲಾಗಿದೆ
  - **OAuth 2.1 ಭದ್ರತೆ**: ಇತ್ತೀಚಿನ OAuth ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳೊಂದಿಗೆ ಹೊಂದಾಣಿಕೆ ದೃಢೀಕರಿಸಲಾಗಿದೆ
  - **OWASP ಮಾನದಂಡಗಳು**: LLM ಗಾಗಿ OWASP ಟಾಪ್ 10 ಉಲ್ಲೇಖಗಳು ಪ್ರಸ್ತುತವಾಗಿವೆ ಎಂದು ಪರಿಶೀಲಿಸಲಾಗಿದೆ
  - **ಅಜೂರ್ ಸೇವೆಗಳು**: ಎಲ್ಲಾ ಮೈಕ್ರೋಸಾಫ್ಟ್ ಅಜೂರ್ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಲಿಂಕ್‌ಗಳು ಮತ್ತು ಉತ್ತಮ ಅಭ್ಯಾಸಗಳನ್ನು ಪರಿಶೀಲಿಸಲಾಗಿದೆ
- **ಮಾನದಂಡಗಳ ಹೊಂದಾಣಿಕೆ**: ಎಲ್ಲಾ ಉಲ್ಲೇಖಿತ ಭದ್ರತಾ ಮಾನದಂಡಗಳು ಪ್ರಸ್ತುತವಾಗಿವೆ ಎಂದು ದೃಢೀಕರಿಸಲಾಗಿದೆ
  - NIST AI ಅಪಾಯ ನಿರ್ವಹಣಾ ಚಟುವಟಿಕೆ
  - ISO 27001:2022
  - OAuth 2.1 ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು
  - ಅಜೂರ್ ಭದ್ರತೆ ಮತ್ತು ಅನುಕೂಲತಾ ಚಟುವಟಿಕೆಗಳು
- **ಕಾರ್ಯಗತಗೊಳಿಸುವ ಸಂಪನ್ಮೂಲಗಳು**: ಎಲ್ಲಾ ಕಾರ್ಯಗತಗೊಳಿಸುವ ಮಾರ್ಗದರ್ಶಿ ಲಿಂಕ್‌ಗಳು ಮತ್ತು ಸಂಪನ್ಮೂಲಗಳನ್ನು ಪರಿಶೀಲಿಸಲಾಗಿದೆ
  - ಅಜೂರ್ API ನಿರ್ವಹಣಾ ಪ್ರಮಾಣೀಕರಣ ಮಾದರಿಗಳು
  - ಮೈಕ್ರೋಸಾಫ್ಟ್ ಎಂಟ್ರಾ ID ಏಕೀಕರಣ ಮಾರ್ಗದರ್ಶಿಗಳು
  - ಅಜೂರ್ ಕೀ ವಾಲ್ಟ್ ರಹಸ್ಯ ನಿರ್ವಹಣೆ
  - ಡೆವ್‌ಸೆಕ್‌ಆಪ್ಸ್ ಪೈಪ್ಲೈನ್ಗಳು ಮತ್ತು ಮೇಲ್ವಿಚಾರಣಾ ಪರಿಹಾರಗಳು

### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಗುಣಮಟ್ಟ ಖಚಿತತೆ
- **ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಅನುಕೂಲತೆ**: ಎಲ್ಲಾ ಅಗತ್ಯ MCP ಭದ್ರತಾ ಅವಶ್ಯಕತೆಗಳು (ಮಸ್ಟ್/ಮಸ್ಟ್ ನಾಟ್) ಇತ್ತೀಚಿನ ಸ್ಪೆಸಿಫಿಕೇಶನ್‌ಗೆ ಹೊಂದಾಣಿಕೆ ಹೊಂದಿವೆ ಎಂದು ಖಚಿತಪಡಿಸಲಾಗಿದೆ
- **ಸಂಪನ್ಮೂಲ ಪ್ರಸ್ತುತತೆ**: ಮೈಕ್ರೋಸಾಫ್ಟ್ ಡಾಕ್ಯುಮೆಂಟೇಶನ್, ಭದ್ರತಾ ಮಾನದಂಡಗಳು ಮತ್ತು ಕಾರ್ಯಗತಗೊಳಿಸುವ ಮಾರ್ಗದರ್ಶಿಗಳ ಎಲ್ಲಾ ಬಾಹ್ಯ ಲಿಂಕ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸಲಾಗಿದೆ
- **ಉತ್ತಮ ಅಭ್ಯಾಸಗಳ ವ್ಯಾಪ್ತಿ**: ಪ್ರಮಾಣೀಕರಣ, ಪ್ರಾಧಿಕಾರ, AI-ನಿರ್ದಿಷ್ಟ ಬೆದರಿಕೆಗಳು, ಸರಬರಾಜು ಸರಪಳಿ ಭದ್ರತೆ ಮತ್ತು ಎಂಟರ್‌ಪ್ರೈಸ್ ಮಾದರಿಗಳ ಸಮಗ್ರ ವ್ಯಾಪ್ತಿಯನ್ನು ದೃಢೀಕರಿಸಲಾಗಿದೆ

## ಅಕ್ಟೋಬರ್ 6, 2025

### ಪ್ರಾರಂಭಿಸುವ ವಿಭಾಗ ವಿಸ್ತರಣೆ – ಉನ್ನತ ಮಟ್ಟದ ಸರ್ವರ್ ಬಳಕೆ ಮತ್ತು ಸರಳ ಪ್ರಮಾಣೀಕರಣ

#### ಉನ್ನತ ಮಟ್ಟದ ಸರ್ವರ್ ಬಳಕೆ (03-GettingStarted/10-advanced)
- **ಹೊಸ ಅಧ್ಯಾಯ ಸೇರಿಸಲಾಗಿದೆ**: ಉನ್ನತ MCP ಸರ್ವರ್ ಬಳಕೆಯ ಸಮಗ್ರ ಮಾರ್ಗದರ್ಶಿ ಪರಿಚಯಿಸಲಾಗಿದೆ, ಸಾಮಾನ್ಯ ಮತ್ತು ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್ ವಾಸ್ತುಶಿಲ್ಪಗಳನ್ನು ಒಳಗೊಂಡಿದೆ.
  - **ಸಾಮಾನ್ಯ ಮತ್ತು ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್**: ಎರಡೂ ವಿಧಾನಗಳ ಪೈಥಾನ್ ಮತ್ತು ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್ ಕೋಡ್ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ ವಿವರವಾದ ಹೋಲಿಕೆ.
  - **ಹ್ಯಾಂಡ್ಲರ್ ಆಧಾರಿತ ವಿನ್ಯಾಸ**: ವಿಸ್ತಾರಗೊಳ್ಳುವ, ಲವಚಿಕ ಸರ್ವರ್ ಅನುಷ್ಠಾನಗಳಿಗೆ ಹ್ಯಾಂಡ್ಲರ್ ಆಧಾರಿತ ಉಪಕರಣ/ಸಂಪನ್ಮೂಲ/ಪ್ರಾಂಪ್ಟ್ ನಿರ್ವಹಣೆಯ ವಿವರಣೆ.
  - **ಪ್ರಾಯೋಗಿಕ ಮಾದರಿಗಳು**: ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್ ಮಾದರಿಗಳು ಉನ್ನತ ವೈಶಿಷ್ಟ್ಯಗಳು ಮತ್ತು ವಾಸ್ತುಶಿಲ್ಪಕ್ಕೆ ಹೇಗೆ ಲಾಭದಾಯಕವಾಗುತ್ತವೆ ಎಂಬ ನೈಜ ಜಗತ್ತಿನ ದೃಶ್ಯಗಳು.

#### ಸರಳ ಪ್ರಮಾಣೀಕರಣ (03-GettingStarted/11-simple-auth)
- **ಹೊಸ ಅಧ್ಯಾಯ ಸೇರಿಸಲಾಗಿದೆ**: MCP ಸರ್ವರ್‌ಗಳಲ್ಲಿ ಸರಳ ಪ್ರಮಾಣೀಕರಣವನ್ನು ಅನುಷ್ಠಾನಗೊಳಿಸುವ ಹಂತ ಹಂತ ಮಾರ್ಗದರ್ಶಿ.
  - **ಪ್ರಮಾಣೀಕರಣ ಸಂಪ್ರದಾಯಗಳು**: ಪ್ರಮಾಣೀಕರಣ ಮತ್ತು ಪ್ರಾಧಿಕಾರ, ಮತ್ತು ಪ್ರಮಾಣಪತ್ರ ನಿರ್ವಹಣೆಯ ಸ್ಪಷ್ಟ ವಿವರಣೆ.
  - **ಮೂಲಭೂತ ಪ್ರಮಾಣೀಕರಣ ಅನುಷ್ಠಾನ**: ಪೈಥಾನ್ (ಸ್ಟಾರ್ಲೆಟ್) ಮತ್ತು ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್ (ಎಕ್ಸ್‌ಪ್ರೆಸ್) ನಲ್ಲಿ ಮಧ್ಯವರ್ತಿ ಆಧಾರಿತ ಪ್ರಮಾಣೀಕರಣ ಮಾದರಿಗಳು, ಕೋಡ್ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ.
  - **ಉನ್ನತ ಭದ್ರತೆ ಕಡೆಗೆ ಪ್ರಗತಿ**: ಸರಳ ಪ್ರಮಾಣೀಕರಣದಿಂದ ಪ್ರಾರಂಭಿಸಿ OAuth 2.1 ಮತ್ತು RBAC ಗೆ ಮುಂದುವರಿಯುವ ಮಾರ್ಗದರ್ಶನ, ಉನ್ನತ ಭದ್ರತಾ ಘಟಕಗಳಿಗೆ ಉಲ್ಲೇಖಗಳೊಂದಿಗೆ.

ಈ ಸೇರ್ಪಡೆಗಳು ಹೆಚ್ಚು ಬಲಿಷ್ಠ, ಭದ್ರ ಮತ್ತು ಲವಚಿಕ MCP ಸರ್ವರ್ ಅನುಷ್ಠಾನಗಳನ್ನು ನಿರ್ಮಿಸಲು ಪ್ರಾಯೋಗಿಕ, ಕೈಯಲ್ಲಿ ಮಾರ್ಗದರ್ಶನವನ್ನು ಒದಗಿಸುತ್ತವೆ, ಮೂಲಭೂತ ಸಂಪ್ರದಾಯಗಳನ್ನು ಉನ್ನತ ಉತ್ಪಾದನಾ ಮಾದರಿಗಳೊಂದಿಗೆ ಸೇರ್ಪಡೆ ಮಾಡುತ್ತವೆ.

#### 11-MCPServerHandsOnLabs - ಹೊಸ ಸಂಪೂರ್ಣ ಡೇಟಾಬೇಸ್ ಏಕೀಕರಣ ಪಠ್ಯಕ್ರಮ

  - **ನೈಜ ಜಗತ್ತಿನ ಅನುಷ್ಠಾನ**: ಎಂಟರ್‌ಪ್ರೈಸ್-ಗ್ರೇಡ್ ಮಾದರಿಗಳನ್ನು ತೋರಿಸುವ Zava ರಿಟೇಲ್ ವಿಶ್ಲೇಷಣೆ ಬಳಕೆ ಪ್ರಕರಣ
  - **ಸಂರಚಿತ ಕಲಿಕೆ ಪ್ರಗತಿ**:
    - **ಪ್ರಯೋಗಾಲಯಗಳು 00-03: ಮೂಲಭೂತಗಳು** - ಪರಿಚಯ, ಕೋರ್ ವಾಸ್ತುಶಿಲ್ಪ, ಭದ್ರತೆ ಮತ್ತು ಬಹು-ಭಾಡಿಗೆ, ಪರಿಸರ ಸ್ಥಾಪನೆ

    - **ಪ್ರಯೋಗಾಲಯಗಳು 07-09: ಉನ್ನತ ವೈಶಿಷ್ಟ್ಯಗಳು** - ಅರ್ಥಪೂರ್ಣ ಹುಡುಕಾಟ ಏಕೀಕರಣ, ಪರೀಕ್ಷೆ ಮತ್ತು ದೋಷನಿರ್ವಹಣೆ, VS ಕೋಡ್ ಏಕೀಕರಣ
    - **ಪ್ರಯೋಗಾಲಯಗಳು 10-12: ಉತ್ಪಾದನೆ ಮತ್ತು ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು** - ನಿಯೋಜನೆ ತಂತ್ರಗಳು, ಮೇಲ್ವಿಚಾರಣೆ ಮತ್ತು ಗಮನಾರ್ಹತೆ, ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು ಮತ್ತು ಆಪ್ಟಿಮೈಜೆಷನ್

  - **ಅಜೂರ್ ಈವೆಂಟ್ ಗ್ರಿಡ್ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್**: ಸಮಗ್ರ ಸರ್ವರ್‌ಲೆಸ್ ಈವೆಂಟ್ ಚಾಲಿತ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಅನುಷ್ಠಾನ
    - C#, ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್ ಮತ್ತು ಪೈಥಾನ್ ಉದಾಹರಣೆಗಳು ಅಜೂರ್ ಫಂಕ್ಷನ್‌ಗಳ ಏಕೀಕರಣದೊಂದಿಗೆ
    - ವಿಸ್ತಾರಗೊಳ್ಳುವ MCP ಪರಿಹಾರಗಳಿಗೆ ಈವೆಂಟ್ ಚಾಲಿತ ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳು
    - ವೆಬ್‌ಹುಕ್ ಸ್ವೀಕರಿಸುವಿಕೆ ಮತ್ತು ಪುಷ್ ಆಧಾರಿತ ಸಂದೇಶ ನಿರ್ವಹಣೆ
  - **ಅಜೂರ್ ಈವೆಂಟ್ ಹಬ್ಸ್ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್**: ಹೆಚ್ಚಿನ ಪ್ರಮಾಣದ ಸ್ಟ್ರೀಮಿಂಗ್ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಅನುಷ್ಠಾನ
    - ಕಡಿಮೆ ವಿಳಂಬದ ದೃಶ್ಯಗಳಿಗೆ ನೈಜ-ಸಮಯ ಸ್ಟ್ರೀಮಿಂಗ್ ಸಾಮರ್ಥ್ಯಗಳು
    - ವಿಭಾಗೀಕರಣ ತಂತ್ರಗಳು ಮತ್ತು ಚೆಕ್‌ಪಾಯಿಂಟ್ ನಿರ್ವಹಣೆ

    - ಅನೇಕ ಅಜೂರ್ ಫಂಕ್ಷನ್‌ಗಳ ನಡುವೆ ವಿತರಿತ MCP ಪ್ರಕ್ರಿಯೆ
    - ಹಲವಾರು ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಪ್ರಕಾರಗಳನ್ನು ಸಂಯೋಜಿಸುವ ಸಂಯುಕ್ತ ವಾಸ್ತುಶಿಲ್ಪಗಳು
    - ಸಂದೇಶ ಸ್ಥಿರತೆ, ವಿಶ್ವಾಸಾರ್ಹತೆ ಮತ್ತು ದೋಷ ನಿರ್ವಹಣಾ ತಂತ್ರಗಳು
  - **ಭದ್ರತೆ ಮತ್ತು ಮೇಲ್ವಿಚಾರಣೆ**: ಅಜೂರ್ ಕೀ ವಾಲ್ಟ್ ಏಕೀಕರಣ ಮತ್ತು ಗಮನಾರ್ಹತೆ ಮಾದರಿಗಳು
    - ನಿರ್ವಹಿತ ಗುರುತು ಪ್ರಮಾಣೀಕರಣ ಮತ್ತು ಕನಿಷ್ಠ привಿಲೇಜ್ ಪ್ರವೇಶ
    - ಅಪ್ಲಿಕೇಶನ್ ಇನ್ಸೈಟ್ಸ್ ಟೆಲಿಮೆಟ್ರಿ ಮತ್ತು ಕಾರ್ಯಕ್ಷಮತೆ ಮೇಲ್ವಿಚಾರಣೆ
    - ಸರ್ಕ್ಯೂಟ್ ಬ್ರೇಕರ್‌ಗಳು ಮತ್ತು ದೋಷ ಸಹಿಷ್ಣುತೆ ಮಾದರಿಗಳು

    - ಟೆಸ್ಟ್ ಡಬಲ್ಸ್ ಮತ್ತು ಮಾಕಿಂಗ್ ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳೊಂದಿಗೆ ಯುನಿಟ್ ಪರೀಕ್ಷೆ
    - ಅಜೂರ್ ಟೆಸ್ಟ್ ಕಂಟೈನರ್‌ಗಳೊಂದಿಗೆ ಏಕೀಕರಣ ಪರೀಕ್ಷೆ

#### ಮೂಲ ತತ್ವಗಳು (01-CoreConcepts/) - ಸಂಪೂರ್ಣ ಆಧುನೀಕರಣ  
- **ಪ್ರೋಟೋಕಾಲ್ ಆವೃತ್ತಿ ನವೀಕರಣ**: ಪ್ರಸ್ತುತ MCP Specification 2025-06-18 ಗೆ ದಿನಾಂಕ ಆಧಾರಿತ ಆವೃತ್ತಿ (YYYY-MM-DD ಸ್ವರೂಪ) ಉಲ್ಲೇಖವನ್ನು ನವೀಕರಿಸಲಾಗಿದೆ  
- **ವಾಸ್ತುಶಿಲ್ಪ ಸುಧಾರಣೆ**: ಪ್ರಸ್ತುತ MCP ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳನ್ನು ಪ್ರತಿಬಿಂಬಿಸುವ Hosts, Clients, ಮತ್ತು Servers ವಿವರಣೆಗಳನ್ನು ಸುಧಾರಿಸಲಾಗಿದೆ  
  - Hosts ಈಗ ಬಹು MCP ಕ್ಲೈಂಟ್ ಸಂಪರ್ಕಗಳನ್ನು ಸಂಯೋಜಿಸುವ AI ಅಪ್ಲಿಕೇಶನ್‌ಗಳಾಗಿ ಸ್ಪಷ್ಟವಾಗಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ  
  - Clients ಅನ್ನು ಪ್ರೋಟೋಕಾಲ್ ಸಂಪರ್ಕಕಾರಿಗಳಾಗಿ ವರ್ಣಿಸಲಾಗಿದೆ, ಒಂದರಿಂದ ಒಂದರ ಸರ್ವರ್ ಸಂಬಂಧಗಳನ್ನು ನಿರ್ವಹಿಸುವಂತೆ  
  - Servers ಅನ್ನು ಸ್ಥಳೀಯ ಮತ್ತು ದೂರಸ್ಥ ನಿಯೋಜನೆ ದೃಶ್ಯಾವಳಿಗಳೊಂದಿಗೆ ಸುಧಾರಿಸಲಾಗಿದೆ  
- **ಪ್ರಿಮಿಟಿವ್ ಪುನರ್‌ರಚನೆ**: ಸರ್ವರ್ ಮತ್ತು ಕ್ಲೈಂಟ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳ ಸಂಪೂರ್ಣ ಪುನರ್‌ರಚನೆ  
  - ಸರ್ವರ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳು: ಸಂಪನ್ಮೂಲಗಳು (ಡೇಟಾ ಮೂಲಗಳು), ಪ್ರಾಂಪ್ಟ್‌ಗಳು (ಟೆಂಪ್ಲೇಟುಗಳು), ಟೂಲ್‌ಗಳು (ಕಾರ್ಯನಿರ್ವಹಣಾ ಕಾರ್ಯಗಳು) ವಿವರವಾದ ವಿವರಣೆ ಮತ್ತು ಉದಾಹರಣೆಗಳೊಂದಿಗೆ  
  - ಕ್ಲೈಂಟ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳು: ಸ್ಯಾಂಪ್ಲಿಂಗ್ (LLM ಪೂರ್ಣಗೊಳಿಸುವಿಕೆ), ಎಲಿಸಿಟೇಶನ್ (ಬಳಕೆದಾರ ಇನ್‌ಪುಟ್), ಲಾಗಿಂಗ್ (ಡಿಬಗ್/ಮೇಲ್ವಿಚಾರಣೆ)  
  - ಪ್ರಸ್ತುತ ಡಿಸ್ಕವರಿ (`*/list`), ರಿಟ್ರೀವಲ್ (`*/get`), ಮತ್ತು ಕಾರ್ಯನಿರ್ವಹಣೆ (`*/call`) ವಿಧಾನ ಮಾದರಿಗಳೊಂದಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ  
- **ಪ್ರೋಟೋಕಾಲ್ ವಾಸ್ತುಶಿಲ್ಪ**: ಎರಡು-ಮಟ್ಟದ ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಯನ್ನು ಪರಿಚಯಿಸಲಾಗಿದೆ  
  - ಡೇಟಾ ಲೇಯರ್: JSON-RPC 2.0 ಆಧಾರ, ಜೀವನಚಕ್ರ ನಿರ್ವಹಣೆ ಮತ್ತು ಪ್ರಿಮಿಟಿವ್‌ಗಳೊಂದಿಗೆ  
  - ಸಾರಿಗೆ ಲೇಯರ್: STDIO (ಸ್ಥಳೀಯ) ಮತ್ತು Streamable HTTP SSE (ದೂರಸ್ಥ) ಸಾರಿಗೆ ಯಂತ್ರಗಳು  
- **ಭದ್ರತಾ ಫ್ರೇಮ್ವರ್ಕ್**: ಸ್ಪಷ್ಟ ಬಳಕೆದಾರ ಅನುಮತಿ, ಡೇಟಾ ಗೌಪ್ಯತೆ ರಕ್ಷಣೆ, ಟೂಲ್ ಕಾರ್ಯನಿರ್ವಹಣೆ ಸುರಕ್ಷತೆ, ಮತ್ತು ಸಾರಿಗೆ ಲೇಯರ್ ಭದ್ರತೆ ಸೇರಿದಂತೆ ಸಮಗ್ರ ಭದ್ರತಾ ತತ್ವಗಳು  
- **ಸಂವಹನ ಮಾದರಿಗಳು**: ಪ್ರೋಟೋಕಾಲ್ ಸಂದೇಶಗಳನ್ನು ಪ್ರಾರಂಭ, ಡಿಸ್ಕವರಿ, ಕಾರ್ಯನಿರ್ವಹಣೆ, ಮತ್ತು ಸೂಚನೆ ಹರಿವುಗಳನ್ನು ತೋರಿಸಲು ನವೀಕರಿಸಲಾಗಿದೆ  
- **ಕೋಡ್ ಉದಾಹರಣೆಗಳು**: ಪ್ರಸ್ತುತ MCP SDK ಮಾದರಿಗಳನ್ನು ಪ್ರತಿಬಿಂಬಿಸುವ ಬಹುಭಾಷಾ ಉದಾಹರಣೆಗಳನ್ನು (.NET, Java, Python, JavaScript) ನವೀಕರಿಸಲಾಗಿದೆ

  - ನೈಜ ಜಗತ್ತಿನ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ ವಿವರವಾದ ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್ ದಾಳಿ ದೃಶ್ಯಗಳು  
  - ಟೂಲ್ ವಿಷಕಾರಕ ಯಂತ್ರಗಳು ಮತ್ತು "ರಗ್ ಪುಲ್" ದಾಳಿ ಮಾದರಿಗಳು  
  - ಸಾಂದರ್ಭಿಕ ವಿಂಡೋ ವಿಷಕಾರಕ ಮತ್ತು ಮಾದರಿ ಗೊಂದಲ ದಾಳಿಗಳು  
- **Microsoft AI ಭದ್ರತಾ ಪರಿಹಾರಗಳು**: Microsoft ಭದ್ರತಾ ಪರಿಸರದ ಸಮಗ್ರ ಆವರಣೆ  
  - ಉನ್ನತ ಪತ್ತೆ, ಸ್ಪಾಟ್‌ಲೈಟಿಂಗ್, ಮತ್ತು ಡೆಲಿಮಿಟರ್ ತಂತ್ರಜ್ಞಾನಗಳೊಂದಿಗೆ AI Prompt Shields  
  - Azure Content Safety ಏಕೀಕರಣ ಮಾದರಿಗಳು  
  - ಸರಬರಾಜು ಸರಪಳಿ ರಕ್ಷಣೆಗೆ GitHub Advanced Security  
- **ಉನ್ನತ ಬೆದರಿಕೆ ನಿವಾರಣೆ**:  
  - MCP-ವಿಶೇಷ ದಾಳಿ ದೃಶ್ಯಗಳೊಂದಿಗೆ ಸೆಷನ್ ಹೈಜ್ಯಾಕಿಂಗ್ ಮತ್ತು ಕ್ರಿಪ್ಟೋಗ್ರಾಫಿಕ್ ಸೆಷನ್ ID ಅಗತ್ಯಗಳು  
  - ಸ್ಪಷ್ಟ ಅನುಮತಿ ಅಗತ್ಯಗಳೊಂದಿಗೆ MCP ಪ್ರಾಕ್ಸಿ ದೃಶ್ಯಗಳಲ್ಲಿ ಗೊಂದಲದ ಡೆಪ್ಯೂಟಿ ಸಮಸ್ಯೆಗಳು  
  - ಅಗತ್ಯ ಮಾನ್ಯತೆ ನಿಯಂತ್ರಣಗಳೊಂದಿಗೆ ಟೋಕನ್ ಪಾಸ್ಸ್ತ್ರೂ ದುರ್ಬಲತೆಗಳು  
- **ಸರಬರಾಜು ಸರಪಳಿ ಭದ್ರತೆ**: ಫೌಂಡೇಶನ್ ಮಾದರಿಗಳು, ಎम्बೆಡ್ಡಿಂಗ್ ಸೇವೆಗಳು, ಸಾಂದರ್ಭಿಕ ಒದಗಿಸುವವರು, ಮತ್ತು ಮೂರನೇ ಪಕ್ಷ API ಗಳನ್ನು ಒಳಗೊಂಡ AI ಸರಬರಾಜು ಸರಪಳಿಯ ವಿಸ್ತರಣೆ  
- **ಫೌಂಡೇಶನ್ ಭದ್ರತೆ**: ಶೂನ್ಯ ನಂಬಿಕೆ ವಾಸ್ತುಶಿಲ್ಪ ಮತ್ತು Microsoft ಭದ್ರತಾ ಪರಿಸರ ಸೇರಿದಂತೆ ಎಂಟರ್‌ಪ್ರೈಸ್ ಭದ್ರತಾ ಮಾದರಿಗಳ ಏಕೀಕರಣವನ್ನು ಸುಧಾರಿಸಲಾಗಿದೆ  
- **ಸಂಪನ್ಮೂಲ ಸಂಘಟನೆ**: ಪ್ರಕಾರಗಳ ಪ್ರಕಾರ (ಅಧಿಕೃತ ಡಾಕ್ಸ್, ಮಾನದಂಡಗಳು, ಸಂಶೋಧನೆ, Microsoft ಪರಿಹಾರಗಳು, ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳು) ಸಮಗ್ರ ಸಂಪನ್ಮೂಲ ಲಿಂಕ್‌ಗಳನ್ನು ವರ್ಗೀಕರಿಸಲಾಗಿದೆ  

### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಗುಣಮಟ್ಟ ಸುಧಾರಣೆಗಳು  
- **ಸಂರಚಿತ ಅಧ್ಯಯನ ಗುರಿಗಳು**: ನಿರ್ದಿಷ್ಟ, ಕಾರ್ಯನಿರ್ವಹಣೀಯ ಫಲಿತಾಂಶಗಳೊಂದಿಗೆ ಅಧ್ಯಯನ ಗುರಿಗಳನ್ನು ಸುಧಾರಿಸಲಾಗಿದೆ  
- **ಕ್ರಾಸ್-ರೆಫರೆನ್ಸುಗಳು**: ಸಂಬಂಧಿತ ಭದ್ರತಾ ಮತ್ತು ಮೂಲ ತತ್ವ ವಿಷಯಗಳ ನಡುವೆ ಲಿಂಕ್‌ಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ  
- **ಪ್ರಸ್ತುತ ಮಾಹಿತಿ**: ಎಲ್ಲಾ ದಿನಾಂಕ ಉಲ್ಲೇಖಗಳು ಮತ್ತು ನಿರ್ದಿಷ್ಟೀಕರಣ ಲಿಂಕ್‌ಗಳನ್ನು ಪ್ರಸ್ತುತ ಮಾನದಂಡಗಳಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ  
- **ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶನ**: ಎರಡೂ ವಿಭಾಗಗಳಲ್ಲಿ ನಿರ್ದಿಷ್ಟ, ಕಾರ್ಯನಿರ್ವಹಣೀಯ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶನಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
