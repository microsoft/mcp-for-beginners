# ಉದಾಹರಣೆ: ಮೂಲ ಹೋಸ್ಟ್

MCP ಸರ್ವರ್‌ಗಳಿಗೆ ಸಂಪರ್ಕಿಸುವ ಮತ್ತು ಸುರಕ್ಷಿತ ಸ್ಯಾಂಡ್‌ಬಾಕ್ಸ್‌ನಲ್ಲಿ ಟೂಲ್ UIಗಳನ್ನು ರೆಂಡರ್ ಮಾಡುವ MCP ಹೋಸ್ಟ್ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಹೇಗೆ ನಿರ್ಮಿಸಬಹುದು ಎಂಬುದನ್ನು ತೋರಿಸುವ ರೆಫರೆನ್ಸ್ ಅಮಲಾತ್ಮಕತೆ.

ಈ ಮೂಲ ಹೋಸ್ಟ್ ಅನ್ನು ಸ್ಥಳೀಯ ಅಭಿವೃದ್ಧಿಯ ಸಂದರ್ಭದಲ್ಲಿ MCP ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ಪರೀಕ್ಷಿಸಲು ಕೂಡ ಬಳಸಬಹುದು.

## ಪ್ರಮುಖ ಕಡತಗಳು

- [`index.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/index.html) / [`src/index.tsx`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/index.tsx) - ಟೂಲ್ ಆಯ್ಕೆ, ಪ್ಯಾರಾಮೀಟರ್ ಇನ್ಪುಟ್ ಮತ್ತು ಐಫ್ರೇಮ್ ನಿರ್ವಹಣೆಯೊಂದಿಗೆ ರಿಯಾಕ್ಟ್ UI ಹೋಸ್ಟ್  
- [`sandbox.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/sandbox.html) / [`src/sandbox.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/sandbox.ts) - ಭದ್ರತಾ ಪರಿಶೀಲನೆ ಮತ್ತು ದ್ವಿಮುಖಿ ಸಂದೇಶ ರಿಲೆಯನ್ನು ಮಾಡುವ ಹೊರಗಿನ ಐಫ್ರೇಮ್ ಪ್ರಾಕ್ಸಿ  
- [`src/implementation.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/implementation.ts) - ಮೂಲ ಲಾಜಿಕ್ಸ್: ಸರ್ವರ್ ಸಂಪರ್ಕ, ಟೂಲ್ ಕಾಲಿಂಗ್, ಮತ್ತು AppBridge ಸಿದ್ಧತೆ  

## ಪ್ರಾರಂಭಿಸುವುದು

```bash
npm install
npm run start
# ತೆರೆಯಿರಿ http://localhost:8080
```
  
ನಿರ್ಧಿಷ್ಟವಾಗಿ, ಹೋಸ್ಟ್ ಅಪ್ಲಿಕೇಶನ್ `http://localhost:3001/mcp` ನಲ್ಲಿ MCP ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕ próbಮಾಡುತ್ತದೆ. ನೀವು ಈ ವರ್ತನೆಯನ್ನು ಸರ್ವರ್ URLಗಳ JSON ಸರಣಿಯನ್ನು ಹೊಂದಿರುವ `SERVERS` ಪರಿಸರ ಚರವನ್ನು ಸೆಟ್ ಮಾಡುವುದು ಮೂಲಕ ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದು:

```bash
SERVERS='["http://localhost:1234/mcp", "http://localhost:5678/mcp"]' npm run start
```
  
## ಗಣತಂತ್ರ ಸಂರಚನೆ

ಈ ಉದಾಹರಣೆ ಸುರಕ್ಷಿತ UI ಪ್ರತ್ಯೇಕಿಕೆಗಾಗಿ ಡಬಲ್-ಐಫ್ರೇಮ್ ಸ್ಯಾಂಡ್‌ಬಾಕ್ಸ್ ಮಾದರಿಯನ್ನು ಬಳಸುತ್ತದೆ:

```
Host (port 8080)
  └── Outer iframe (port 8081) - sandbox proxy
        └── Inner iframe (srcdoc) - untrusted tool UI
```
  
**ಎರಡು ಐಫ್ರೇಮ್‌ಗಳಿರುವುದು ಏಕೆ?**

- ಹೊರಗಿನ ಐಫ್ರೇಮ್ ಪ್ರತ್ಯೇಕ ಮೂಲ (ಪೋರ್ಟ್ 8081) ನಲ್ಲಿ ಓಡುತ್ತಿದ್ದು ಹೋಸ್ಟ್‌ಗೆ ನೇರ ಪ್ರವೇಶವನ್ನು ತಡೆಯುತ್ತದೆ  
- ಒಳಗಿನ ಐಫ್ರೇಮ್ `srcdoc` ಮೂಲಕ HTML ಸ್ವೀಕರಿಸುತ್ತದೆ ಮತ್ತು ಸ್ಯಾಂಡ್‌ಬಾಕ್ಸ್ ಗುಣಲಕ್ಷಣಗಳಿಂದ ನಿಯಂತ್ರಿತವಾಗಿದೆ  
- ಸಂದೇಶಗಳು ಹೊರಗಿನ ಐಫ್ರೇಮ್ ಮೂಲಕ ಹರಿದು ಹೋಗುತ್ತವೆ, ಇದು ಅವುಗಳನ್ನು ಪರಿಶೀಲಿಸಿ ದ್ವಿಮುಖವಾಗಿ ರಿಲೇ ಮಾಡುತ್ತದೆ  

ಈ ಗಣತಂತ್ರ ಸಂರಚನೆಯು ಟೂಲ್ UI ಕೋಡ್ ದುರುದ್ದೇಶಪೂರ್ಣವಾದರೂ ಸಹ, ಅದು ಹೋಸ್ಟ್ ಅಪ್ಲಿಕೇಶನ್‌ನ DOM, ಕೂಕೀಸ್ ಅಥವಾ ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್ ಸನ್ನಿವೇಶಕ್ಕೆ ಪ್ರವೇಶಿಸುವುದಿಲ್ಲ ಎಂದು ಖಚಿತಪಡಿಸುತ್ತದೆ.
