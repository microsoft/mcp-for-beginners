## ಈ ಪರಿಹಾರ ಹೇಗೆ ರಚಿಸಲಾಗಿದೆ

ಪರಿಹಾರವು ಕೇವಲ ಪ್ರಮುಖ ಭಾಗಗಳನ್ನೊಳಗೊಂಡಿದೆ, ಅಂದರೆ ಮಾರ್ಕ್ಅಪ್, ಈವೆಂಟ್ ವೈರ್ ಅಪ್ ಮತ್ತು ಸರ್ವರ್ ವೈಶಿಷ್ಟ್ಯಗಳು.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
