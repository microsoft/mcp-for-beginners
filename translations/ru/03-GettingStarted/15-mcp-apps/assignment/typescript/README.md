Пример камень, ножницы, бумага

## Как построено это решение

Решение состоит только из важной части кода, то есть из разметки, привязки событий и серверных функций.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
