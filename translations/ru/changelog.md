- **Добавлена новая глава**: Введено подробное руководство по расширенному использованию сервера MCP, охватывающее как обычные, так и низкоуровневые архитектуры серверов.

  - **Дизайн на основе обработчиков**: Объяснение управления инструментами/ресурсами/подсказками на основе обработчиков для масштабируемых и гибких реализаций серверов.
  - **Практические шаблоны**: Реальные сценарии, где шаблоны низкоуровневого сервера полезны для расширенных функций и архитектуры.

#### Простая аутентификация (03-GettingStarted/11-simple-auth)
- **Добавлена новая глава**: Пошаговое руководство по реализации простой аутентификации в серверах MCP.
  - **Концепции аутентификации**: Чёткое объяснение аутентификации и авторизации, а также обработки учётных данных.
  - **Реализация базовой аутентификации**: Шаблоны аутентификации на основе middleware в Python (Starlette) и TypeScript (Express) с примерами кода.
  - **Переход к расширенной безопасности**: Руководство по началу с простой аутентификации и переходу к OAuth 2.1 и RBAC с ссылками на модули расширенной безопасности.

Эти дополнения предоставляют практические, пошаговые рекомендации для создания более надёжных, безопасных и гибких реализаций серверов MCP, связывая базовые концепции с продвинутыми производственными шаблонами.
