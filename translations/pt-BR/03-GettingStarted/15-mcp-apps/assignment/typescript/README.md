## Como esta solução está estruturada

A solução consiste apenas na parte do código que importa, ou seja, a marcação, conexão de eventos e recursos do servidor.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
