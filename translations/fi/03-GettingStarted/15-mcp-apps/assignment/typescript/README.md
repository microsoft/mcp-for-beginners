Rock paper scissor -esimerkki

## Ratkaisun rakenne

Ratkaisu sisältää vain olennaisen osan koodista, eli merkinnän, tapahtumien kytkennän ja palvelinominaisuudet.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

## Kuinka suorittaa

Katso [Code](../../code/typescript/README.md) ja lisää näiden tiedostojen sisältö kuhunkin vastaavaan tiedostoon.
