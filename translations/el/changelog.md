Αυτό το έγγραφο χρησιμεύει ως αρχείο όλων των σημαντικών αλλαγών που έγιναν στο πρόγραμμα σπουδών Model Context Protocol (MCP) για Αρχάριους. Οι αλλαγές καταγράφονται σε αντίστροφη χρονολογική σειρά (οι πιο πρόσφατες αλλαγές πρώτες).

#### Προχωρημένη Χρήση Διακομιστή (03-GettingStarted/10-advanced)
- **Προσθήκη Νέου Κεφαλαίου**: Εισαγωγή ολοκληρωμένου οδηγού για προχωρημένη χρήση διακομιστή MCP, καλύπτοντας τόσο κανονικές όσο και χαμηλού επιπέδου αρχιτεκτονικές διακομιστών.
  - **Κανονικός έναντι Χαμηλού Επιπέδου Διακομιστή**: Λεπτομερής σύγκριση και παραδείγματα κώδικα σε Python και TypeScript για τις δύο προσεγγίσεις.
  - **Σχεδίαση Βασισμένη σε Χειριστές**: Επεξήγηση διαχείρισης εργαλείων/πόρων/προτροπών βασισμένη σε χειριστές για κλιμακούμενες, ευέλικτες υλοποιήσεις διακομιστών.
  - **Πρακτικά Πρότυπα**: Σενάρια πραγματικού κόσμου όπου τα πρότυπα χαμηλού επιπέδου διακομιστή είναι ωφέλιμα για προχωρημένα χαρακτηριστικά και αρχιτεκτονική.

#### Απλή Αυθεντικοποίηση (03-GettingStarted/11-simple-auth)
- **Προσθήκη Νέου Κεφαλαίου**: Οδηγός βήμα προς βήμα για την υλοποίηση απλής αυθεντικοποίησης σε διακομιστές MCP.
  - **Έννοιες Αυθεντικοποίησης**: Σαφής επεξήγηση της αυθεντικοποίησης έναντι της εξουσιοδότησης και της διαχείρισης διαπιστευτηρίων.
  - **Υλοποίηση Βασικής Αυθεντικοποίησης**: Πρότυπα αυθεντικοποίησης βασισμένα σε middleware σε Python (Starlette) και TypeScript (Express), με παραδείγματα κώδικα.
  - **Προώθηση σε Προχωρημένη Ασφάλεια**: Οδηγίες για εκκίνηση με απλή αυθεντικοποίηση και προώθηση σε OAuth 2.1 και RBAC, με αναφορές σε προχωρημένα μοντέλα ασφαλείας.

Αυτές οι προσθήκες παρέχουν πρακτική, εμπειρική καθοδήγηση για την κατασκευή πιο ανθεκτικών, ασφαλών και ευέλικτων υλοποιήσεων διακομιστών MCP, γεφυρώνοντας θεμελιώδεις έννοιες με προχωρημένα πρότυπα παραγωγής.

  - Οι Hosts ορίζονται πλέον σαφώς ως εφαρμογές AI που συντονίζουν πολλαπλές συνδέσεις πελατών MCP
  - Οι Clients περιγράφονται ως συνδετήρες πρωτοκόλλου που διατηρούν σχέσεις ένας προς έναν με servers

  - Ενημερωμένα με τρέχοντα πρότυπα μεθόδων ανακάλυψης (`*/list`), ανάκτησης (`*/get`) και εκτέλεσης (`*/call`)

  - Επίπεδο Δεδομένων: Βάση JSON-RPC 2.0 με διαχείριση κύκλου ζωής και πρωτογενή στοιχεία
  - Επίπεδο Μεταφοράς: STDIO (τοπικό) και Streamable HTTP με SSE (απομακρυσμένο) μηχανισμούς μεταφοράς
- **Πλαίσιο Ασφαλείας**: Ολοκληρωμένες αρχές ασφαλείας συμπεριλαμβανομένης ρητής συγκατάθεσης χρήστη, προστασίας απορρήτου δεδομένων, ασφάλειας εκτέλεσης εργαλείων και ασφάλειας επιπέδου μεταφοράς
