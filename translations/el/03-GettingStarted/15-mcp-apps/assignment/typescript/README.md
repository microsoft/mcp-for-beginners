Rock paper scissor παράδειγμα

## Πώς είναι δομημένη αυτή η λύση

Η λύση αποτελείται μόνο από το μέρος του κώδικα που έχει σημασία, δηλαδή τη σήμανση, τη σύνδεση γεγονότων και τις λειτουργίες του διακομιστή.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
