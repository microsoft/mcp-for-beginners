Rock paper scissor voorbeeld

## Hoe deze oplossing is opgebouwd

De oplossing bestaat alleen uit het deel van de code dat ertoe doet, namelijk de markup, event koppeling en serverfuncties.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
