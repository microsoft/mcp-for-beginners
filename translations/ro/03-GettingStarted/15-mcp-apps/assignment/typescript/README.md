## Cum este structurată această soluție

Soluția constă doar în partea de cod care contează, adică marcajul, legarea evenimentelor și funcțiile serverului.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
