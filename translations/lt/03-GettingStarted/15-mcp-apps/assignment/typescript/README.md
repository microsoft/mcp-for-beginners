## Kaip ši sprendimas yra struktūruotas

Sprendimas susideda tik iš svarbios kodo dalies, t.y. žymėjimo, įvykių susiejimo ir serverio funkcijų.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
