# 🚀 Model Context Protocol (MCP) အခြေခံသူများအတွက် သင်ရိုးညွှန်းတမ်း

## **C#, Java, JavaScript, Rust, Python နှင့် TypeScript မှာ လက်တွေ့ကိရိယာနောက်ခံ ကုဒ်နမူနာများဖြင့် MCP ကို သင်ယူပါ**

## 🧠 Model Context Protocol သင်ရိုးညွှန်းတမ်း အကြောင်းအကျဥ်း

Model Context Protocol အတွင်း ခရီးစဉ်သို့ ကြိုဆိုပါသည်! AI အပလီကေးရှင်းများက ဘယ်လို ကိရိယာများနှင့် ဝန်ဆောင်မှုများနှင့် ဆက်သွယ်သည်ကို ဘယ်တော့မှ စိတ်ဝင်စားခဲ့ပြီးဖြစ်ပါက၊ ဒီလှပသော ဖြေရှင်းချက်ကို သင်ရှာတွေ့မည်ဖြစ်ပြီး အဲဒါက အတက်အကျ ဆောင်ရွက်နည်းကို ပြောင်းလဲနေပါပြီ။

MCP ကို AI အပလီကေးရှင်းများအတွက် အပြည့်အဝ ဘာသာပြန်ကိရိယာတစ်ခုအဖြစ် စဉ်းစားပါ - USB ပေါက်များက မည်သည့်ကိရိယာမျိုးမဆို သင့်ကွန်ပျူတာနှင့် ချိတ်ဆက်နိုင်သလို၊ MCP က AI မော်ဒယ်များကို မည်သည့်ကိရိယာ သို့မဟုတ် ဝန်ဆောင်မှုမဆို စံနှုန်းတူနည်းဖြင့် ချိတ်ဆက်ရန် ခွင့်ပြုသည်။ သင့် ပထမဆုံး ချတ်ဘော့ အပလီကေးရှင်းကို ဆောက်ရန် ဖြစ်စေ၊ ရှုပ်ထွေးသော AI စနစ်များအတွက် အလုပ်လုပ်ရန်ဖြစ်စေ၊ MCP ကို နားလည်ခြင်းဖြင့် ပိုမိုစွမ်းဆောင်ရည်ရှိပြီး လွယ်ကူသော အပလီကေးရှင်းများ ဖန်တီးနိုင်မယ်။

ဒီ သင်ရိုးညွှန်းတမ်းကို သင်ယူရေးခရီးမှာ  သည်းခံမှုနဲ့ သေချာစွာ ကိုယ်စားပြုထားသည်။ သင့်လက်တွင် ရှိပြီးသား ရိုးရှင်းသော အယူအဆများကနေ စတင်ပြီး မိမိနှစ်သက်ရာ Programming ဘာသာစကားဖြင့် လက်တွေ့လုပ်ကာ ကျွမ်းကျင်မှုတိုးတက်မှုကို တဖြည်းဖြည်း တိုးတက်စေမယ်။ အဆင့်တိုင်းမှာရှင်းလင်းသော ရှင်းပြချက်များ၊ လက်တွေ့ ဥပမာများနှင့် စိတ်အားထက်သန်စေနိုင်သော အားပေးမှုများ ပါဝင်သည်။

ဒီခရီးကို ပြီးမြောက်သောအခါ သင့်ကိုယ်ပိုင် MCP ဆာဗာများ ဖန်တီးနိုင်မယ်၊ လူကြိုက်အများဆုံး AI ပလက်ဖောင်းများနှင့် ပေါင်းစည်းနိုင်မယ်၊ ဒီနည်းပညာသည် AI ဖွံ့ဖြိုးတိုးတက်မှု၏ အနာဂတ်ကို မည်သို့ ပြောင်းလဲနေသည်ကို နားလည်ကြမယ်။ စိတ်လှုပ်ရှားဖွယ် ဒီ ခရီးစဉ်ကို အတူတကွ စတင်ကြရအောင်!

### တရားဝင်စာရွက်စာတမ်းနှင့် ဖော်ပြချက်များ

ဒီသင်ရိုးညွှန်းတမ်းသည် **MCP Specification 2025-11-25** (နောက်ဆုံးမပြောင်းလဲသေးသော ဗားရှင်း) နှင့် ကိုက်ညီထားသည်။ MCP specification သည် ရက်စွဲအခြေပြု ဗားရှင်း (YYYY-MM-DD ဖော်မတ်) ကို အသုံးပြု၍ စနစ်တကျ protocol ဗားရှင်းများကို ခွဲခြားထားသည်။

အဆိုပါအရင်းအမြစ်များကို သင့်နားလည်မှု တိုးလာ၍ ဆာလငယ်မြင်က ရတနာတန်ဆာများလို ဖြစ်လာမည် သို့မဟုတ် မဖြတ်တောက်ဘဲ စိတ်ညစ်စရာမဖြစ်အောင် မျက်မှောက် စတင်စိုက်ထုတ်ပါ။
- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – ဒီနေရာမှာ လမ်းညွှန်ချက်များ၊ အသုံးပြုသူလမ်းညွှန်စာအုပ်များကို ပုံမှန်စနစ်တကျ ရှာတွေ့နိုင်ပါသည်။ စတင်သင်ယူသူများအတွက် ရိုးရှင်းသော ဥပမာများကို ထည့်သွင်းရေးသားထားပြီး ကိုယ့်အလျောက် တိုးတက်စေမည်။
- 📜 [MCP Specification](https://modelcontextprotocol.io/specification/2025-11-25) – ဒီစာတမ်းကို သင့်ရဲ့ တကယ့်လမ်းညွှန် များအဖြစ်ယူနိုင်သည်။ သင့်သင်ရိုးညွှန်းတမ်းကို လေ့လာသွားစဉ် အထူးအသေးစိတ်များနှင့် တိုးတက်နည်းများကို ရှာဖွေရန် ဒီနေရာကို ပြန်လာရောက်ကြည့်ပါလိမ့်မယ်။
- 📜 [MCP Specification Versioning](https://modelcontextprotocol.io/specification/versioning) – protocol ဗားရှင်းသမိုင်းနဲ့ MCP ရက်စွဲအခြေပြု ဗားရှင်း အသုံးပြုခြင်းအကြောင်း ပါဝင်သည်။
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – အမျိုးမျိုးသော programming ဘာသာစကားများဖြင့် SDK များ၊ ကိရိယာများနှင့် ကုဒ်နမူနာများကို တွေ့နိုင်သည့် သိုလှောင်ကွန်တော့ ဖြစ်ပါသည်။
- 🌐 [MCP Community](https://github.com/orgs/modelcontextprotocol/discussions) – MCP နည်းပညာ စင်တာလူ့အသိုင်းအဝိုင်းတွင်တွဲဖက်လေ့လာသူများနှင့် အတွေ့အကြုံရှိ developer များနှင့် ဆွေးနွေးပါ။ မေးခွန်းများကို ကြိုဆိုပြီး အတူတကွ သိမှုမျှဝေသော အဖွဲ့အစည်းတစ်ခု ဖြစ်ပါသည်။
  
## သင်ယူမှု ရည်မှန်းချက်များ

ဒီသင်ရိုးညွှန်းတမ်းပြီးဆုံးသောအခါ သင်သည် သင့်ရဲ့ နယူးကျွမ်းကျင်မှုများကို ယုံကြည်မှုရှိပြီး စိတ်လှုပ်ရှားနေတာကို ခံစားမယ်။ ရရှိမယ့် ပညာရပ်တွေက:

• **MCP အခြေခံများ နားလည်ခြင်း**: Model Context Protocol ဆိုတာဘာလဲ၊ AI အပ်ပလီကေးရှင်းများ ပြောဆိုဆက်သွယ်မှုအတွက် ဘယ်လို ရာဇဝင်ဖန်တီးနေသည် ဆိုတာ နားလည်သဖြင့် ရိုးရာအတိုင်းအတာနဲ့ ဥပမာများဖြင့် ဖော်ပြပါမယ်။

• **သင့် ပထမဆုံး MCP ဆာဗာတည်ဆောက်ခြင်း**: မိမိနှစ်သက်ရာ Programming ဘာသာစကားဖြင့် လက်တွေ့ MCP ဆာဗာတစ်ခု တည်ဆောက်နိုင်ရန်၊ ရိုးရှင်းသော နမူနာများဖြင့် စတင်ကာ တစ်ဆင့်ချင်းတိုးတက်စေမယ်။

• **AI မော်ဒယ်များကို တကယ့်ကိရိယာများနှင့် ချိတ်ဆက်ခြင်း**: AI မော်ဒယ်များနဲ့ တကယ့်ဝန်ဆောင်မှုများကြား လမ်းကြောင်း တည်ဆောက်ခြင်းကို သင်ယူမည်၊ မိမိဤရည်မှန်းချက်ဖြင့် အပလီကေးရှင်းများကို လုပ်ဆောင်ချက်များပိုမိုစွမ်းဆောင်နိုင်စေမယ်။

• **လုံခြုံရေးအကောင်းဆုံး လေ့လာမှုများ ကျင့်သုံးခြင်း**: MCP ရဲ့ လုံခြုံမှုနောက်ခံများကို နားလည်ပြီး မိမိအပလီကေးရှင်း နဲ့ အသုံးပြုသူများကို မလုံခြုံမှုမှ ကာကွယ်ရန် နည်းလမ်းတွေတည်ဆောက်မယ်။

• **ယုံကြည်စိတ်ချမှုနဲ့ ထုတ်လွှင့်ခြင်း**: MCP ပရောဂျက်များကို ဖွံ့ဖြိုးမှုပြီးလို့ ထုတ်လုပ်မှုဆီသို့ ရောက်အောင် အပြုသဘောဆောင်တဲ့ deployment နည်းလမ်းများကို ကိုင်တွယ်နိုင်မယ်။

• **MCP လူမှုအသိုင်းအဝိုင်းအသက်ဝင်ခြင်း**: AI အပ်ပလီကေးရှင်း ဖန်တီးရေးနယ်ပယ်မှာ အနာဂတ်ကို ပုံသေနိုင်သူ စွမ်းအားကြီးသော developer များအဖွဲ့အစည်းတစ်ခု ဖြစ်လာမယ်။ 

## အရေးကြီးသော နောက်ခံ

MCP အသေးစိတ်သို့ ကြိုးပမ်းသွားရန်မတိုင်မီ အခြေခံအတွေးအခေါ်များကို သေချာ နားလည်ထားရမည်။ မိမိမကျွမ်းကျင်သော်လည်း ရိုးရှင်းစွာ ရှင်းပြပေးသွားမည်ဖြစ်သောကြောင့် စိုးရိမ်ရန်မလိုပါ။

### Protocols နားလည်ခြင်း (အခြေခံအုတ်မြစ်)

Protocol ဆိုတာ ဆက်သွယ်မှု စည်းမျဉ်းစည်းကမ်းများလို ဖြစ်သည်။ မိတ်ဆွေတစ်ယောက်ကို ခေါ်မယ်ဆိုရင် "ဟယ်လို" လို့ ပြောတတ်ရမယ်၊ စကားပြောခေါင်းဖြတ်ကောငျ့များရှိပြီး "ပျော်ပါစေ" ဆိုပြီး နှုတ်ဆက်ရမည်။ ကွန်ပျူတာပရိုဂရမ်များလည်း မတူညီတဲ့ စည်းမျဉ်းများဖြင့် ဆက်သွယ်ကြတယ်။

MCP က protocol တစ်ခုဖြစ်ပြီး AI မော်ဒယ်နှင့် အပြင်ကိရိယာများ၊ ဝန်ဆောင်မှုများတွင် သဘောတူညီချက်ရှိသော စည်းကမ်းများဖြစ်သည်။ လူတွေ စကားပြောရာမှာ စည်းကမ်းတွေရှိခြင်းနဲ့ တူသည်။ MCP က AI အပလီကေးရှင်းလမ်းကြောင်းအသုံးပြုပြီး ထိရောက်စွာ ပြောဆိုဆက်သွယ်နိုင်စေသည်။

### Client-Server ဆက်ဆံရေး (ပရိုဂရမ်တွေ ဘယ်လိုပူးပေါင်းအလုပ်လုပ်သလဲ)

လူတိုင်းသည် client-server ဆက်ဆံရေးကို နေ့စဉ်အသုံးပြုနေပါပြီ။ Web browser (client) ဖြင့် website ကို သွားရောက်ကြည့်သောအခါ၊ မိမိတောင်းဆိုသည့် page စာများကို ဝဘ်ဆာဗာမှ လက်ခံရရှိသည်။ Browser က ဒေတာတောင်းယူ၊ ဆာဗာက တုံ့ပြန်တယ်။

MCP မှာလည်း ဒီလိုပဲ client-server ဆက်ဆံရေးရှိသည်။ AI မော်ဒယ်တွေ client အဖြစ်လုပ်ဆောင်ပြီး အချက်အလက် သို့မဟုတ် အချက်အလက်တစ်စုံတစ်ရာကို တောင်းဆိုသည်။ MCP ဆာဗာများက ဒီတာကို ဖြေကြားပေးသည်။ ဒါဟာ helper assistant (ဆာဘား) တစ်ယောက်လို ဖြစ်ပြီး AI ကို အကူအညီပေးသွားသည်။

### စံနှုန်းထားမှု အရေးပါမှု (အရာအားလုံးကို ပူးပေါင်းအလုပ်လုပ်စေခြင်း)

တစ်ချို့ကားထုတ်လုပ်သူတစ်ဦးချင်းစီက ကြိုးတံပို့ကွက်ပုံစံကွာခြားမှ ကားတစ်စီးစီအတွက် အဆက်အသွယ်ကိရိယာ အမျိုးမျိုးလိုအပ်ရမည်။ စံနှုန်းထားမှုဆိုသည်မှာ အရာအားလုံးကို ပူးပေါင်းအောင် အချက်အလက်များကို သဘောတူညီခြင်းဖြစ်သည်။

MCP သည် AI အပ်ပလီကေးရှင်းများအတွက် အဲဒီ စံနှုန်းထားမှု ဖြစ်သည်။ AI မော်ဒယ်တိုင်းသည် ကိရိယာအသစ် တစ်ခုတီထွင်ရန် မလိုအပ်ပဲ MCP ကို အသုံးပြု၍ စံပြတစ်ခုအတိုင်း ဆက်သွယ်နိုင်သည်။ ဒါကြောင့် developer တွေက တစ်ကြိမ်ကိရိယာ ဖန်တီးရုံနဲ့ မည်သည့် AI စနစ်နဲ့မဆို အလုပ်လုပ်နိုင်သည်။

## 🧭 သင်ယူရေး ခရီးစဉ် အနှစ်ချုပ်

MCP ခရီးစဉ်ကို စိတ်ချယုံကြည်စွာ နှင့် ကျွမ်းကျင်မှု ပြုလုပ်နိုင်စေဖို့ အဆင့်ဆင့် အခြေခံကနေ တိုးတက်လာအောင် စီစဉ်ထားသည်။ အဆင့်တိုင်းမှာ အသစ်အဟောင်း ထပ်မံ အကြောင်းအရာများ နားလည်စေသေးသည်။

### 🌱 အခြေခံ အဆင့်: အခြေခံ အယူအဆ နားလည်ခြင်း (Module 0-2)

ဒီနေရာမှာ သင့်စွန့်စားခန်းစတင်! MCP နည်းပညာများကို သေချာ သိအောင် ရိုးရှင်း ဥပမာများနှင့် အသုံးချ ဖော်ပြမည်။ MCP ဆိုတာ ဘာလဲ၊ ဘာကြောင့် လိုအပ်သလဲ၊ AI အဖွဲ့အစည်းမှာ ဘယ်လို ပါဝင်လဲ ဆိုတာနားလည်ရမည်။

• **Module 0 - MCP အကျဉ်းချုပ်**: MCP ရဲ့ အရေးကြီးချက်နှင့်  AI အပ်လီကေးရှင်းခေတ်သစ်မှာ ဘယ်လို အသုံးဝင်သလဲ၊ MCP ဖြေရှင်းနိုင်တဲ့ ပြဿနာများကို မြင်ရမည်။

• **Module 1 - အဓိက အယူအဆများ ရှင်းပြချက်**: MCP အခြေခံ အစိတ်အပိုင်းများကို ကျယ်ပြန့်စွာ သိအောင် ဥပမာနှင့် ပုံပြင်ပေါ်တွင် ရှင်းလင်းမည်။

• **Module 2 - MCP တွင် လုံခြုံရေး**: လုံခြုံရေးဆိုတာ ကြောက်တတ်စရာ မဟုတ်ပါဘူး၊ MCP မှာ လုံခြုံမှု ကောင်းမွန်စေရန် ဖန်တီးထားသော လုပ်ထုံးလုပ်နည်းများ ထည့်သွင်းထားတာနဲ့ လုံခြုံရေး ကောင်းမွန်စေဖို့ အကောင်းဆုံးနည်းလမ်းများကို သင်ယူမယ်။

### 🔨 ဆောက်လုပ်ခြင်း အဆင့်: သင့် ပထမဆုံး Implementation များ ဖန်တီးခြင်း (Module 3)

အခုမှ အစစ်အမှန် အခန်းကဏ္ဍ စတင်ပါပြီ! သင် လက်တွေ့ MCP ဆာဗာများနှင့် client များ ဖန်တီးရန် အတွေ့အကြုံ ရယူမည်။ စိုးရိမ်စရာ မရှိပဲ ရိုးရှင်းစတင်ပြီး တစ်ဆင့်ချင်း ဦးတည် လမ်းညွှန်မှု ရယူပါမည်။
ဤမော်ဒျူးတွင် သင်နှစ်သက်သည့် ပရိုဂရမ်မင်းဘာသာစကားတွင် လေ့ကျင့်နိုင်ရန် လက်တွေ့လမ်းညွှန်များစွာ ပါဝင်သည်။ သင်သည် သင်၏ ပထမဆုံးဆာဗာကို ဖန်တီးမည်၊ ၎င်းနှင့် ဆက်သွယ်ရန် ကလေးအင်တီးတစ်ရပ် တည်ဆောက်မည်၊ နှင့်မည်သည့် development tools ပေါ်လစီများဖြင့်ပင် ဆက်စပ် ထည့်သွင်းနိုင်မည်ဆိုသည်ကိုလည်း စမ်းသပ်နိုင်သည်။

လမ်းညွှန်တိုင်းတွင် ကုဒ်နမူနာ ပြည့်စုံ၊ ပြဿနာဖြေရှင်းရေးအကြံပြုချက်များနှင့် ဤဒီဇိုင်းရွေးချယ်မှုများကို မည်ကြောင့် ရွေးလိုက်သလဲ ဆိုသည့် ရှင်းပြချက်များ ပါဝင်သည်။ ဤအစိတ်အပိုင်းအဆုံးတွင် သင်သည် အလုပ်ဖြစ်စေသော MCP အကောင်အထည်ဖော်မှုများကို မည်သည့်အရာမှ ဂုဏ်ယူနိုင်ကြောင်း ထိမ်းသိမ်းထားမည်ဖြစ်သည်။

### 🚀 တိုးတက်မှုအဆင့် - ချိုးပွဲခံအယူအဆများနှင့် လက်တွေ့ လျှောက်လွှာ (Module 4-5)

အခြေခံအချက်များ ကို အတွေးပေါ် စွမ်းဆောင်နိုင်သည့်အခြေအနေတွင် သင်သည် MCP ၏ ပိုမိုတောင်တက်သော လုပ်ဆောင်ချက်များကို လေ့လာရန် ပြင်ဆင်ပြီးပါပြီ။ ကျွန်ုပ်တို့သည် လက်တွေ့ အကောင်အထည်ဖော်နည်းများ၊ ပြဿနာဖြေရှင်းနည်းများနှင့် မူလီမော်ဒယ် AI ပေါင်းစပ်မှု ကဲ့သို့သော အဆင့်မြင့်ခေါင်းစဉ်များကို ဖော်ပြပါမည်။

နောက်ထပ် MCP အကောင်အထည်ဖော်မှုများကို ထုတ်လုပ်မှုအတွက် မည်သို့ ပုံစံချနိုင်သည်နှင့် Azure စသည့် cloud platform များနှင့် ပေါင်းစပ် အသုံးပြုနိုင်မှုကိုလည်း သင်ယူမည်ဖြစ်သည်။ ဤမော်ဒျူးများသည် သဘာဝပတ်ဝန်းကျင် လိုအပ်ချက်များကို ကိုင်တွယ်နိုင်သည့် MCP ဖြေရှင်းချက်များကို တည်ဆောက်ရန် သင်ကို ပြင်ဆင်ပေးမည် ဖြစ်သည်။

### 🌟 ကျွမ်းကျင်မှုအဆင့် - လူမှုအသိုင်းအဝိုင်းနှင့် အထူးပြုမှု (Module 6-11)

နောက်ဆုံးအဆင့်သည် MCP လူမှုအသိုင်းအဝိုင်းသို့ ပူးပေါင်းဝင်ရောက်ရန်နှင့် သင့်အားစိတ်ဝင်စားရာနယ်ပယ်များတွင် အထူးပြုရန် အာရုံစိုက်သည်။ သင်သည် open-source MCP ပရောဂျက်များတွင် ကူညီပံ့ပိုးရာ၊ မြင့်မားသော အတည်ပြုမူ ပုံစံများ ကို အကောင်အထည်ဖော်ခြင်းနှင့် အချက်အလက် ဒေတာဘေ့စ်ပေါင်းစပ်ထားသည့် လုံးဝဉာဏ်ပညာဖြေရှင်းချက်များ တည်ဆောက်ရာတွင် လေ့လာသင်ယူမည်ဖြစ်သည်။

Module 11 သည် အထူးသဖြင့် အသိပညာဖြင့်လည်း ထူးခြားသည်။ ၎င်းသည် PostgreSQL ပေါင်းစပ်ပြီး ထုတ်လုပ်မှုအသင့် MCP ဆာဗာများ တည်ဆောက်ရန် သင်ခန်းစာ lab ၁၃ ခုပါဝင်သည့် လက်တွေ့ထက်မြက် သင်ယူမှုလမ်းကြောင်းဖြစ်သည်။ ၎င်းသည် သင်အတူတူ သင်ယူခဲ့သော အရာအားလုံးကို ထည့်သွင်းပြသသည့် အစိတ်အပိုင်း စီမံကိန်းတစ်ခုကဲ့သို့ ဖြစ်ပါသည်။

## 📚 လေ့လာသင်ယူလမ်းညွှန်နှင့် ရင်းမြစ်များ

ဤ repository တွင် သင်ကြားလမ်းညွှန်နှင့် ထိရောက်စွာ လေ့လာနိုင်ရန် အထောက်အကူဖြစ်မည့် ရင်းမြစ်များ ပါဝင်သည်။

### လေ့လာသင်ယူလမ်းညွှန်

ပြည့်စုံသော [လေ့လာသင်ယူလမ်းညွှန်](./study_guide.md) တစ်ခုမှာ ဤ repository ကို ထိရောက်စွာ လမ်းပြရန် ရရှိနိုင်သည်။ ဤ မြင်ကွင်းအပြည့်အစုံ သင်ရိုးညွှန်းတမ်းတွင် အကြောင်းအရာအားလုံး ချိတ်ဆက်ပုံကို ပြသပြီး၊ နမူနာပရောဂျက်များကို ဘာသာရပ်မှတ် အတိုင်း သုံးစွဲနည်း ပေးသည်။ သင်သည် ဗွီဇွရယ် လေ့လာသူ ဖြစ်ပါက ဤသည်သည် အထူးအကျိုးရှိမည်ဖြစ်သည်။

လမ်းညွှန်တွင်ပါဝင်ခြင်းများမှာ - 
- အကြောင်းအရာများအားလုံး ဖော်ပြထားသော မြင်ကွင်းအပြည့်အစုံ သင်ရိုးညွှန်းတမ်း
- repository အစိတ်အပိုင်းများ အသေးစိတ် ဖျက်ခွဲခြင်း
- နမူနာပရောဂျက်များ အသုံးချနည်း လမ်းညွှန်မှု
- အရည်အချင်းအဆင့်များ အလိုက် လေ့လာမှု လမ်းကြောင်းများ 추천
- သင်ယူခြင်းလမ်းကြောင်းကို ကူညီပံ့ပိုးရန် ထပ်ဆောင်းရင်းမြစ်များ

### ဖော်ပြချက်ပြောင်းလဲမှုမှတ်တမ်း

ကျွန်ုပ်တို့သည် သင်ရိုးညွှန်းတမ်း အရေးကြီးပြင်ဆင်မှုအားလုံးကို မှတ်တမ်းတင်ထားသော [ဖော်ပြချက်ပြောင်းလဲမှုမှတ်တမ်း](./changelog.md) ကို ထိန်းသိမ်းထားပါသည်၊ မကြာခဏအဆင့်မြှင့်မှုများ၊ အသစ်ထည့်သွင်းမှုများကို နောက်ဆုံးထိ သိရှိနားလည်နိုင်စေရန်ဖြစ်သည်။
- အသစ်ထည့်သွင်းသော အကြောင်းအရာများ
- ဖွဲ့စည်းပုံပြောင်းလဲမှုများ
- လုပ်ဆောင်ချက်တိုးတက်စေမှုများ

## 🎓 သင်၏ MCP ခရီးစဉ် စတင်သည်

အောင်မြင်မှုများကို ဂုဏ်ယူပါ! သင်သည် programming ကျွမ်းကျင်မှုများ တိုးချဲ့ရန် နှင့် AI ဖွံ့ဖြိုးတိုးတက်မှု နောက်ဆုံးရပ်စက်မှ အာရုံစိုက်နိုင်မည့် ခရီးတွင် ပထမနေရာဆုကို လုယူလိုက်သည်။

သင့် MCP ခရီးစဉ် ယခုစတင်ပါပြီ! Module 0 မှ စပြီး သင့်ပထမဆုံး အတွေ့အကြုံများကို အရိုက်အနှက်အထိ ဝင်ရောက်လိုက်ပါ၊ ဒါမှမဟုတ် သင်ဖန်တီးမည့် ပရောဂျက်နမူနာများကို ကြည့်ရှု၊ ရယူပါ။ ကျွမ်းကျင်သူ အားလုံးသည် သင်ရှိရာနေရာမှ စတင်ခဲ့သည်၊ သည်းခံမှုနှင့် လေ့ကျင့်မှုဖြင့် သင်သည် မင်းရဲကို တွေ့မြင်လိမ့်မည်။

Model Context Protocol ဖွံ့ဖြိုးတိုးတက်ရေးကမ္ဘာသို့ ကြိုဆိုပါသည်။ အတူတကွ အံ့သြဖွယ်အရာတစ်ခုဖန်တီးကြမယ်!

## 🤝 သင်ယူမှု အသိုင်းအဝိုင်းအတွက် ဝန်းရံခြင်း

ဤသင်ရိုးညွှန်းတမ်းကို သင်လိုသည့် သင်ယူသူများမှ ဝန်းရံမှုများဖြင့် ပိုမိုကောင်းမွန်စေသည်။ သင်ပါးစပ်မှား၊ ရှင်းလင်းပြသမှု ထက်ကောင်းခြင်း သို့မဟုတ် နမူနာအသစ် ထည့်သွင်းခြင်းတိုင်းသည် အခြားသူ အသစ်များ အောင်မြင်စေရန် ကူညီပံ့ပိုးမှုဖြစ်သည်။

Microsoft Valued Professional [Shivam Goyal](https://www.linkedin.com/in/shivam2003/) ကို ကုဒ်နမူနာများ ပံ့ပိုးပေးမှုအတွက် ကျေးဇူးတင်ပါသည်။

ဝန်းရံမှုလုပ်ငန်းစဉ်များသည် ကြိုဆိုလက်ခံ မြင်သာစေသည့် ပုံစံဖြစ်သည်။ ဝန်းရံမှုများ၏ အများစုမှာ Contributor License Agreement (CLA) လိုအပ်သည်၊ သို့သော် Automated tools များက လမ်းပြမှုပေးမည်ဖြစ်သည်။

## 📜 ဖွင့်လှစ်နိုင်သော သင်ယူမှု

ဤသင်ရိုးညွှန်းတမ်း အားလုံးသည် MIT [LICENSE](../../LICENSE) ရွေးချယ်မှုအောက်ရှိဖြစ်ကာ မည်သူမဆို အခမဲ့အသုံးပြု၊ ပြင်ဆင်၊ မျှဝေနိုင်သည်။ ၎င်းသည် MCP ပညာကို လူတိုင်းအဆောင်အအုံရှိစေရန် ကျွန်ုပ်တို့၏သတ္တိတရားဖြစ်သည်။

## 🤝 ဝန်းရံမှု ညွှန်ကြားချက်များ

ဤပရောဂျက်သည် ဝန်းရံမှုများနှင့် အကြံပြုချက်များကို ကြိုဆိုသည်။ ဝန်းရံမှုအများစုတွင် CLA သဘောတူညီချက် တစ်ရပ်နှင့်အညီ သင်၏ ဝန်းရံခွင့်ရှိကြောင်း ဖော်ပြရမည်ဖြစ်သည်။ အသေးစိတ်အချက်အလက်များအတွက် <https://cla.opensource.microsoft.com> ကို ကြည့်ရှုပါ။

Pull request တင်သည့်အခါ၊ CLA bot သည် သင့်ကို CLA တင်ရမည်မဟုတ်မဟုတ်ကို လုပ်ဆောင်ပြီး PR ကို သီးသန့် အခြေအနေပြပေးပါမည် (ဥပမာ - status check, comment)။ Bot သည် လိုက်နာရန် လမ်းညွှန်ချက်များ ပေးမည်ဖြစ်သည်။ သင်သည် ဤ CLA ကို တစ်ကြိမ်တည်းသာ အပေါ်ပါ repository အားလုံးအတွက် လိုအပ်ပြီး ပြုလုပ်ရမည်ဖြစ်သည်။

ဤပရောဂျက်သည် [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/) ကို လက်ခံအသုံးပြုသည်။ အသေးစိတ်အချက်အလက်များအတွက် [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) ကို သို့မဟုတ် [opencode@microsoft.com](mailto:opencode@microsoft.com) သို့ ဆက်သွယ်နိုင်သည်။
