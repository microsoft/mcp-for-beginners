## ဒီဖြေရှင်းချက်သည် မည်သို့ဖွဲ့စည်းထားသည်

ဒီဖြေရှင်းချက်သည် အရေးကြီးသော ကုဒ်ပိုင်းအစသာ ပါဝင်သည်၊ ဆိုလိုသည်မှာ markup, event wire up နှင့် server အင်္ဂါရပ်များ ဖြစ်သည်။

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
