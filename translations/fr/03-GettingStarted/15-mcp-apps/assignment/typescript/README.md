## Comment cette solution est structurée

La solution ne contient que la partie du code qui importe, c’est-à-dire le balisage, la liaison des événements et les fonctionnalités du serveur.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
