# ఉదాహరణ: ఆధారం హోస్ట్

MCP సర్వర్లను కలుపుకొని సెక్యూర్ సాండ్‌బాక్స్‌లో టూల్ UIలను వర్ణించే MCP హోస్ట్ అనువర్తనం నిర్మించడానికి సూచన అమలు.

ఈ ప్రాథమిక హోస్ట్ స్థానిక అభివృద్ధి సమయంలో MCP యాప్స్‌ను పరీక్షించడానికి కూడా ఉపయోగించవచ్చు.

## ముఖ్య ఫైళ్లు

- [`index.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/index.html) / [`src/index.tsx`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/index.tsx) - టూల్ ఎంపిక, పారామితులు ఇన్‌పుట్, మరియు ఐఫ్రేం నిర్వహణతో React UI హోస్ట్
- [`sandbox.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/sandbox.html) / [`src/sandbox.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/sandbox.ts) - భద్రతా ధృవీకరణ మరియు ద్విముఖ సందేశ రీలేతో బయటి ఐఫ్రేం ప్రాక్సీ
- [`src/implementation.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/implementation.ts) - కోర్ లాజిక్: సర్వర్ కనెక్షన్, టూల్ కాలింగ్, మరియు AppBridge సెటప్

## ప్రారంభించటం

```bash
npm install
npm run start
# http://localhost:8080 ని తెరవండి
```

డిఫాల్ట్‌గా, హోస్ట్ అనువర్తనం `http://localhost:3001/mcp` వద్ద MCP సర్వర్‌తో కనెక్ట్ అవ్వడానికి ప్రయత్నిస్తుంది. మీరు ఈ ప్రవర్తనను సర్వర్ URLల JSON అర్రేపై `SERVERS` ఎన్విరాన్మెంట్ వేరియబుల్ సెట్ చేసి కస్టమైజ్ చేయవచ్చు:

```bash
SERVERS='["http://localhost:1234/mcp", "http://localhost:5678/mcp"]' npm run start
```

## నిర్మాణం

ఈ ఉదాహరణ భద్ర UI పరీక్ష కోసం డబుల్-ఐఫ్రేం సాండ్‌బాక్స్ నమూనాను ఉపయోగిస్తుంది:

```
Host (port 8080)
  └── Outer iframe (port 8081) - sandbox proxy
        └── Inner iframe (srcdoc) - untrusted tool UI
```

**రెండు ఐఫ్రేం ఎందుకు?**

- బయటివైపు ఐఫ్రేం వేరే ఆరిజిన్ (పోర్టు 8081) పై నడుస్తుంది, హోస్ట్‌కు ప్రత్యక్ష ప్రాప్తిని నిరోధిస్తుంది
- లోపలి ఐఫ్రేం `srcdoc` ద్వారా HTML అందుకుంటుంది మరియు సాండ్‌బాక్స్ లక్షణాలు దాటి పరిమితం చేయబడి ఉంటుంది
- సందేశాలు బయటివైపు ఐఫ్రేం ద్వారా ప్రసారం అవుతాయి, ఇది వాటిని ధృవీకరించి ద్విముఖ రీవే చేస్తుంది

ఈ నిర్మాణం టూల్ UI కోడ్ దుర్వినియోగకరంగా ఉన్నప్పటికీ, అది హోస్ట్ అనువర్తనం DOM, కుకీలను, లేదా జావాస్క్రిప్ట్ పరిసరాన్ని ప్రాప్తి చేయలేరని నిశ్చయిస్తోంది.
