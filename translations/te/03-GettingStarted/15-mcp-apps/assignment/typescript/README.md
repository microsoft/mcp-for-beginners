## ఈ పరిష్కారం ఎలా నిర్మించబడింది

పరిష్కారం ఆ కోడ్ భాగం మాత్రమే ఉంటుంది, అంటే మార్కప్, ఈవెంట్ వాటర్-అప్ మరియు సర్వర్ ఫీచర్లను.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
