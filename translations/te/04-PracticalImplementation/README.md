# ఉపయోగకరమైన అన్వయము

[![ఎలా MCP యాప్స్‌ని రియల్ టూల్స్ మరియు వర్క్‌ఫ్లోలతో నిర్మించడం, పరీక్షించడం, మరియు డిప్లాయ్ చేయటం](../../../translated_images/te/05.64bea204e25ca891.webp)](https://youtu.be/vCN9-mKBDfQ)

_(ఈ పాఠం యొక్క వీడియోను చూడడానికి పైన్ల చిత్రాన్ని క్లిక్ చేయండి)_

ఉపయోగకరమైన అన్వయము అనేది మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP) శక్తి స్పష్టంగా కనపడే చోటు. MCP వెనుక స సిద్ధాంతం మరియు ఆర్కిటెక్చర్ అర్థం చేసుకోవడం ముఖ్యమైంటే, ఈ సూత్రాలను ఉపయోగించి నిజ జీవిత సమస్యలను పరిష్కరించే పరిష్కారాలను నిర్మించడం, పరీక్షించడం, మరియు అమలు చేయడం ద్వారా నిజమైన విలువ వస్తుంది. ఈ అధ్యాయం సैద్ధాంతిక জ్ఞానం మరియు ప్రాక్టికల్ అభివృద్ధి మధ్య మర్మాన్ని సమీకరిస్తుంది, MCP ఆధారిత యాప్లికేషన్లను జీవించటానికి దారితీస్తుంది.

మీరు జ్ఞానవంతులైన సహాయకులను అభివృద్ధి చేస్తున్నారా, వ్యాపార వర్క్‌ఫ్లోలలో AI ను సమ్మిళితం చేస్తున్నారా, లేదా డేటా ప్రాసెసింగ్ కోసం అనుకూల టూల్స్ నిర్మిస్తున్నారా, MCP ఒక సవ్యమైన పునాది అందిస్తుంది. దాని లాంగ్వేజ్-నిరపేక్షమైన డిజైన్ మరియు ప్రముఖ ప్రోగ్రామింగ్ భాషల కోసం అధికారిక SDKలు వృద్ధి దారులకు విస్తృతంగా అందుబాటులో ఉంటాయి. ఈ SDKలను ఉపయోగించి, మీరు త్వరగా ప్రోటోటైప్ చేయవచ్చు, పునఃరోచనా చేయవచ్చు, మరియు వివిధ ప్లాట్‌ఫారమ్‌లు మరియు పర్యావరణాల్లో మీ పరిష్కారాలను స్కేలు చేయవచ్చు.

తదుపరి విభాగాలలో, మీరు ప్రాక్టికల్ ఉదాహరణలు, నమూనా కోడ్‌లు, మరియు అమలు విధానాలను కనుగొంటారు, ఇవి C#, Java with Spring, TypeScript, JavaScript, మరియు Pythonలో ఎలా MCPని అమలు చేయాలో చూపుతాయి. మీరు మీ MCP సర్వర్‌లను డీబగ్ చేయడం మరియు పరీక్షించడం, APIలను నిర్వహించడం, మరియు ఆజ్యూర్ ఉపయోగించి క్లౌడ్‌కు పరిష్కారాలను డిప్లాయ్ చేయడం నేర్చుకుంటారు. ఈ హ్యాండ్స్-ఆన్ వనరులు మీ అభ్యాసాన్ని వేగవంతం చేసి, మీకు నమ్మకంగా, బలమైన, ప్రొడక్షన్-సిద్ధమైన MCP యాప్లికేషన్లను నిర్మించడంలో సహాయపడతాయి.

## అవలోకనం

ఈ పాఠం వివిధ ప్రోగ్రామింగ్ భాషలలో MCP అమలుకు సంబంధించి ప్రాక్టికల్ అంశాలపై దృష్టి సారిస్తుంది. ఇక్కడి ఫోకస్ MCP SDKలను C#, Java with Spring, TypeScript, JavaScript, మరియు Pythonలో ఉపయోగించి బలమైన అప్లికేషన్లు నిర్మించడం, MCP సర్వర్‌లను డీబగ్ చేయడం మరియు పరీక్షించడం, మరియు పునఃఉపయోగం అయ్యే వనరులు, ప్రాంప్ట్స్, మరియు టూల్స్ సృష్టించడం ఆధారంగా ఉంటుంది.

## అభ్యాస లక్ష్యాలు

ఈ పాఠం ముగిసే వరకు, మీరు చేయగలుగుతారు:

- వివిధ ప్రోగ్రామింగ్ భాషలలో అధికారిక SDKలు ఉపయోగించి MCP పరిష్కారాలను అమలు చేయడం
- MCP సర్వర్‌లను సతతంగా డీబగ్ మరియు పరీక్షించడం
- సర్వర్ ఫీచర్లు (వనరులు, ప్రాంప్ట్స్, మరియు టూల్స్) సృష్టించడం మరియు ఉపయోగించడం
- క్లిష్టమైన పనుల కోసం సమర్థవంతమైన MCP వర్క్‌ఫ్లోలను డిజైన్ చేయడం
- పనితీరు మరియు నమ్మకదరుడత కోసం MCP అమలులను ఆప్టిమైజ్ చేయడం

## మౌలిక సర్వర్ ఫీచర్లు

MCP సర్వర్‌లు ఈ క్రింది ఫీచర్ల ఏదైనా కలిపి అమలు చేయవచ్చు:

### వనరులు

వనరులు వినియోగదారు లేదా AI మోడల్ ఉపయోగించే సందర్భం మరియు డేటా అందిస్తాయి:

- డాక్యుమెంట్ నిల్వలు
- జ్ఞాన ఆధారాలు
- నిర్మిత డేటా మూలాలు
- ఫైల్ సిస్టమ్లు

### ప్రాంప్ట్స్

ప్రాంప్ట్స్ వినియోగదారులకు టెంప్లేటెడ్ సందేశాలు మరియు వర్క్‌ఫ్లోలు:

- ముందుగా నిర్వచించబడిన సంభాషణテンプレట్లు
- మార్గనిర్దేశక ఇంటరాక్షన్ నమూనాలు
- ప్రత్యేక డైలాగ్ నిర్మాణాలు

### టూల్స్

టూల్స్ AI మోడల్ అమలు చేసే కార్యాలయాలు:

- డేటా ప్రాసెసింగ్ యుటిలిటీస్
- బాహ్య API సమ్మేళనాలు
- గణనాత్మక శక్తులు
- శోధన కార్యాచరణ

## నమూనా అమలు: C# అమలు

ప్రదర్శక C# SDK రిపోజిటరీ అనేక MCP పాక్షికాల అమలుతో కూడిన నమూనాలను కలిగి ఉంది:

- **అధారమై MCP క్లయింట్**: MCP క్లయింట్ సృష్టించడం మరియు టూల్స్‌ను పిలవడం చూపే సింపుల్ ఉదాహరణ
- **అధారమై MCP సర్వర్**: మినిమల్ సర్వర్ అమలి తో ప్రాథమిక టూల్ నమోదుకు
- **అధునాతన MCP సర్వర్**: పూర్తి సౌకర్యాలతో టూల్ నమోదు, ఆథెంటికేషన్, మరియు లోప నిర్వహణ
- **ASP.NET సమ్మేళనం**: ASP.NET కోర్‌తో సమ్మేళనం చూపే ఉదాహరణలు
- **టూల్ అమలు నమూనాలు**: వేర్వేరు క్లిష్టత స్థాయిలతో టూల్స్ అమలు కోసం వివిధ నమూనాలు

MCP C# SDK ప్రివ్యూ లో ఉంది మరియు APIలు మారవచ్చు. SDK అభివృద్ధి జరుగుతుండగా మేము ఈ బ్లాగ్‌ను సతతంగా నవీకరిస్తాము.

### ముఖ్య ఫీచర్లు

- [C# MCP Nuget ModelContextProtocol](https://www.nuget.org/packages/ModelContextProtocol)
- మీ [మొదటి MCP సర్వర్‌](https://devblogs.microsoft.com/dotnet/build-a-model-context-protocol-mcp-server-in-csharp/) నిర్మాణం.

పూర్తి C# అమలు నమూనాలకు, [అధిత C# SDK నమూనాలు](https://github.com/modelcontextprotocol/csharp-sdk) రిపోజిటరీని సందర్శించండి.

## API నిర్వహణ

Azure API నిర్వహణ MCP సర్వర్‌లను ఎలా భద్రపరచగలమో ఒక అద్భుత పరిష్కారం. MCP సర్వర్ ముందు Azure API నిర్వహణ సంస్థాపించటం ద్వారా మీరు కోరుకునే లక్షణాలను నిర్వహించవచ్చు:

- రేట్ పరిమితి
- టోకెన్ నిర్వహణ
- మానిటరింగ్
- లోడ్ బ్యాలెన్సింగ్

### ఆజ్యూర్ నమూనా

కింద పేర్కొన్న దృశ్యంలో ఆథరైజేషన్ ఫ్లో ఎలా జరుగుతుందో చూడండి:

ఈ దృశ్యంలో జరుగుతోంది:

- Azure API నిర్వహణ గేట్వేగా పనిచేస్తుంది మరియు ట్రాఫిక్‌ను నడిపించడానికి పాలసీలను ఉపయోగిస్తుంది.
- Azure మానిటర్ అన్ని అభ్యర్థనల లాగ్‌లను మరింత విశ్లేషన్ కోసం నమోదు చేస్తుంది.

## రిమోట్ MCP సర్వర్‌ను Azureకు డిప్లాయ్ చేయడం

ముందు చెప్పిన నమూనాను డిప్లాయ్ చేయగలమో చూద్దాం:

1. రిపోజిటరీని క్లోన్ చేయండి

    ```bash
    git clone https://github.com/Azure-Samples/remote-mcp-apim-functions-python.git
    cd remote-mcp-apim-functions-python
    ```

1. `Microsoft.App` వనరు ప్రొవైడర్‌ను నమోదు చేయండి.

   - మీరు Azure CLI ఉపయోగిస్తుంటే, `az provider register --namespace Microsoft.App --wait` ఎన్టర్ చేయండి.
   - మీరు Azure PowerShell ఉపయోగిస్తుంటే, `Register-AzResourceProvider -ProviderNamespace Microsoft.App` అమలు చేయండి. తరువాత కొంత సమయం తరువాత `(Get-AzResourceProvider -ProviderNamespace Microsoft.App).RegistrationState` ను అమలు చేసి నమోదు పూర్తయిందా చూడండి.

1. ఈ [azd](https://aka.ms/azd) కమాండ్ ద్వారా API నిర్వహణ సేవ, ఫంక్షన్ యాప్ (కోడ్ తో సహా), మరియు ఇతర అవసరమైన Azure వనరులను ప్రావిజన్ చేయండి

    ```shell
    azd up
    ```

    ఈ కమాండ్లు Azureపై అన్ని క్లౌడ్ వనరులను డిప్లాయ్ చేయాలి

### MCP ఇన్స్పెక్టర్ తో మీ సర్వర్‌ను పరీక్షించడం

1. **కొత్త టర్మినల్ విండోలో**, MCP ఇన్స్పెక్టర్ ను ఇన్‌స్టాల్ చేసి అమలు చేయండి

    ```shell
    npx @modelcontextprotocol/inspector
    ```

    ఈ క్రింది వలె ఇంటర్‌ఫేస్ కనిపించాలని:

    ![Connect to Node inspector](../../../translated_images/te/connect.141db0b2bd05f096.webp)

1. MCP ఇన్స్పెక్టర్ వెబ్ యాప్‌ను URL (ఉదా: [http://127.0.0.1:6274/#resources](http://127.0.0.1:6274/#resources)) నుండి CTRL క్లిక్ చేయండి
1. ట్రాన్స్‌పోర్ట్ రకం `SSE`గా సెట్ చేయండి
1. `azd up` తరువాత కనిపించే మీ రన్నింగ్ API నిర్వహణ SSE ఎండ్‌పాయింట్ URL సెట్ చేసి **Connect** క్లిక్ చేయండి:

    ```shell
    https://<apim-servicename-from-azd-output>.azure-api.net/mcp/sse
    ```

1. **List Tools** క్లిక్ చేసి, ఒక టూల్‌పై క్లిక్ చేసి **Run Tool** నొక్కండి.

మీరు అన్ని దశలను సరిగా అమలు చేసినట్లయితే, మీరు ఇప్పుడు MCP సర్వర్‌తో కనెక్ట్ అయి ఒక టూల్‌ను పిలవగలుగుతూ ఉంటారు.

## Azureకు MCP సర్వర్‌లు

[Remote-mcp-functions](https://github.com/Azure-Samples/remote-mcp-functions-dotnet): Python, C# .NET లేదా Node/TypeScriptతో Azure Functions ఉపయోగించి అనుకూల రిమోట్ MCP (మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్) సర్వర్‌లను నిర్మించడం మరియు డిప్లాయ్ చేయడానికి క్విక్‌స్టార్ట్ టెంప్లేట్‌ల సెట్.

ఈ నమూనా అభివృద్ధి దారులకు క్రింది సాధనాలు అందిస్తుంది:

- స్థానికంగా నిర్మించడం మరియు నడపడం: స్థానిక పరికరంపై MCP సర్వర్ అభివృద్ధి మరియు డీబగ్ చేయడం
- Azureకు డిప్లాయ్ చేయడం: సింపుల్ `azd up` కమాండ్‌తో క్లౌడ్‌కు సులభ డిప్లాయ్
- క్లయింట్ల నుంచి కనెక్ట్ అవడం: VS Code యొక్క Copilot ఏజెంట్ మోడ్ మరియు MCP ఇన్స్పెక్టర్ టూల్ సహా అనేక క్లయింట్ల ద్వారా MCP సర్వర్‌కు కనెక్ట్ కావడం

### ముఖ్య ఫీచర్లు

- రూపకల్పనలో భద్రత: MCP సర్వర్ కీస్ మరియు HTTPS ఉపయోగించి భద్రపరచబడింది
- ధృవీకరణ ఎంపికలు: బిల్ట్-ఇన్ ఆథ్ లేదా API నిర్వహణ ద్వారా OAuth మద్దతు
- నెట్‌వర్క్ వేరుశాఖ: Azure వర్చువల్ నెట్వర్క్స్ (VNET) ఉపయోగించి నెట్‌వర్క్ వేరుశాఖ అనుమతిస్తుంది
- సర్వర్‌లెస్ ఆర్కిటెక్చర్: Azure Functions ఉపయోగించి స్కేలబుల్, ఈవెంట్-ఆధారిత అమలు
- స్థానిక అభివృద్ధి: పూర్తి స్థాయి స్థానిక అభివృద్ధి మరియు డీబగ్ మద్దతు
- సులభమైన డిప్లాయ్: Azureకు సులభమైన డిప్లాయ్ ప్రక్రియ

రిపోజిటరీలో అన్ని అవసరమైన కాన్ఫిగరేషన్ ఫైళ్లు, మూల కోడ్, మరియు ఇన్‌ఫ్రాస్ట్రక్చర్ నిర్వచనలు ఉన్నాయి, ఉత్పత్తి సిద్ధమైన MCP సర్వర్ అమలుకు త్వరగా ప్రారంభమయ్యేందుకు.

- [Azure Remote MCP Functions Python](https://github.com/Azure-Samples/remote-mcp-functions-python) - Pythonతో Azure Functions ఉపయోగించి MCP యొక్క నమూనా అమలు

- [Azure Remote MCP Functions .NET](https://github.com/Azure-Samples/remote-mcp-functions-dotnet) - C# .NETతో Azure Functions ఉపయోగించి MCP యొక్క నమూనా అమలు

- [Azure Remote MCP Functions Node/Typescript](https://github.com/Azure-Samples/remote-mcp-functions-typescript) - Node/TypeScriptతో Azure Functions ఉపయోగించి MCP యొక్క నమూనా అమలు.

## ముఖ్యాంశాలు

- MCP SDKలు బలమైన MCP పరిష్కారాలను అమలు చేయడానికి భాష-ఆధారిత సాధనాలను అందిస్తాయి
- డీబగ్గింగ్ మరియు పరీక్షించే ప్రక్రియ నమ్మదగిన MCP అప్లికేషన్లకు కీలకం
- పునఃఉపయోగం అయ్యే ప్రాంప్ట్ టెంప్లెట్ల ద్వారా సవివరమైన AI ఇంటరాక్షన్లు సాధ్యమవుతాయి
- బాగా డిజైన్ చేయబడిన వర్క్‌ఫ్లోలు పలు టూల్స్ ఉపయోగించి క్లిష్టమైన పనులను సమన్వయపరుస్తాయి
- MCP పరిష్కారాలను అమలు చేయడంలో భద్రత, పనితీరు, మరియు లోప నిర్వహణ దృష్టిలో ఉంచాలి

## వ్యాయామం

మీ డొమైన్‌లోని ఒక నిజమైన సమస్యను పరిష్కరించే ప్రాక్టికల్ MCP వర్క్‌ఫ్లోను డిజైన్ చేయండి:

1. ఈ సమస్యను పరిష్కరించడానికి ఉపయోగకరమైన 3-4 టూల్స్ గుర్తించండి
2. ఈ టూల్స్ ఎలా పరస్పరం పనిచేసేవో చూపించే వర్క్‌ఫ్లో డయాగ్రామ్ సృష్టించండి
3. మీ ఇష్టమైన భాషలో ఒక టూల్ యొక్క ప్రాథమిక వెర్షన్‌ను అమలు చేయండి
4. మోడల్ మీ టూల్‌ను సమర్థవంతంగా ఉపయోగించేందుకు సహాయపడే ప్రాంప్ట్ టెంప్లెట్ సృష్టించండి
