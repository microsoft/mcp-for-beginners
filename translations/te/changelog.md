# మార్పుల చరిత్ర: MCP ప్రారంభికుల కోసం పాఠ్యక్రమం

ఈ డాక్యుమెంట్ మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP) ప్రారంభికుల కోసం పాఠ్యక్రమంలో చేసిన అన్ని ముఖ్యమైన మార్పుల రికార్డ్‌గా పనిచేస్తుంది. మార్పులు తిరుగుబాటు కాలానుక్రమంలో (తాజా మార్పులు మొదట) డాక్యుమెంట్ చేయబడ్డాయి.

## డిసెంబర్ 18, 2025

### భద్రతా డాక్యుమెంటేషన్ నవీకరణ - MCP స్పెసిఫికేషన్ 2025-11-25

#### MCP భద్రతా ఉత్తమ ఆచారాలు (02-Security/mcp-best-practices.md) - స్పెసిఫికేషన్ వెర్షన్ నవీకరణ
- **ప్రోటోకాల్ వెర్షన్ నవీకరణ**: తాజా MCP స్పెసిఫికేషన్ 2025-11-25 (నవంబర్ 25, 2025 విడుదల) కు సూచనలను నవీకరించబడింది
  - అన్ని స్పెసిఫికేషన్ వెర్షన్ సూచనలను 2025-06-18 నుండి 2025-11-25 కు నవీకరించబడింది
  - డాక్యుమెంట్ తేదీ సూచనలను ఆగస్టు 18, 2025 నుండి డిసెంబర్ 18, 2025 కు నవీకరించబడింది
  - అన్ని స్పెసిఫికేషన్ URLలు ప్రస్తుత డాక్యుమెంటేషన్ వైపు చూపిస్తున్నాయని ధృవీకరించబడింది
- **కంటెంట్ ధృవీకరణ**: తాజా ప్రమాణాల ప్రకారం భద్రతా ఉత్తమ ఆచారాల సమగ్ర ధృవీకరణ
  - **Microsoft భద్రతా పరిష్కారాలు**: ప్రాంప్ట్ షీల్డ్స్ (మునుపటి "జైల్‌బ్రేక్ రిస్క్ డిటెక్షన్"), Azure కంటెంట్ సేఫ్టీ, Microsoft Entra ID, మరియు Azure కీ వాల్ట్ కోసం ప్రస్తుత పదజాలం మరియు లింకులను ధృవీకరించబడింది
  - **OAuth 2.1 భద్రత**: తాజా OAuth భద్రతా ఉత్తమ ఆచారాలతో సరిపోల్చబడింది
  - **OWASP ప్రమాణాలు**: LLMs కోసం OWASP టాప్ 10 సూచనలు ప్రస్తుతంగా ఉన్నాయని ధృవీకరించబడింది
  - **Azure సేవలు**: అన్ని Microsoft Azure డాక్యుమెంటేషన్ లింకులు మరియు ఉత్తమ ఆచారాలు ధృవీకరించబడ్డాయి
- **ప్రమాణాల సరిపోలిక**: అన్ని సూచించిన భద్రతా ప్రమాణాలు ప్రస్తుతంగా ఉన్నాయని ధృవీకరించబడింది
  - NIST AI రిస్క్ మేనేజ్‌మెంట్ ఫ్రేమ్‌వర్క్
  - ISO 27001:2022
  - OAuth 2.1 భద్రతా ఉత్తమ ఆచారాలు
  - Azure భద్రత మరియు అనుగుణత ఫ్రేమ్‌వర్క్లు
- **అమలు వనరులు**: అన్ని అమలు గైడ్ లింకులు మరియు వనరులు ధృవీకరించబడ్డాయి
  - Azure API మేనేజ్‌మెంట్ ఆథెంటికేషన్ నమూనాలు
  - Microsoft Entra ID ఇంటిగ్రేషన్ గైడ్లు
  - Azure కీ వాల్ట్ సీక్రెట్స్ మేనేజ్‌మెంట్
  - DevSecOps పైప్లైన్లు మరియు మానిటరింగ్ పరిష్కారాలు

### డాక్యుమెంటేషన్ నాణ్యత హామీ
- **స్పెసిఫికేషన్ అనుగుణత**: అన్ని తప్పనిసరి MCP భద్రతా అవసరాలు (MUST/MUST NOT) తాజా స్పెసిఫికేషన్‌కు సరిపోల్చబడ్డాయి
- **వనరు ప్రస్తుతత**: Microsoft డాక్యుమెంటేషన్, భద్రతా ప్రమాణాలు, మరియు అమలు గైడ్లకు అన్ని బాహ్య లింకులు ధృవీకరించబడ్డాయి
- **ఉత్తమ ఆచారాల కవరేజ్**: ఆథెంటికేషన్, ఆథరైజేషన్, AI-స్పెసిఫిక్ ముప్పులు, సరఫరా గొలుసు భద్రత, మరియు ఎంటర్‌ప్రైజ్ నమూనాల సమగ్ర కవరేజ్ నిర్ధారించబడింది

## అక్టోబర్ 6, 2025

### ప్రారంభం విభాగం విస్తరణ – అధునాతన సర్వర్ వినియోగం & సులభ ఆథెంటికేషన్

#### అధునాతన సర్వర్ వినియోగం (03-GettingStarted/10-advanced)
- **కొత్త అధ్యాయం జోడించబడింది**: సాధారణ మరియు లో-లెవల్ సర్వర్ ఆర్కిటెక్చర్లను కవర్ చేసే MCP సర్వర్ అధునాతన వినియోగానికి సమగ్ర గైడ్ పరిచయం చేయబడింది
  - **సాధారణ vs. లో-లెవల్ సర్వర్**: రెండు విధానాలపై పాఠ్య వివరణ మరియు Python, TypeScript కోడ్ ఉదాహరణలు
  - **హ్యాండ్లర్-ఆధారిత డిజైన్**: స్కేలబుల్, ఫ్లెక్సిబుల్ సర్వర్ అమలులకు టూల్/వనరు/ప్రాంప్ట్ నిర్వహణ వివరణ
  - **ప్రాయోగిక నమూనాలు**: లో-లెవల్ సర్వర్ నమూనాలు అధునాతన ఫీచర్లు మరియు ఆర్కిటెక్చర్ కోసం ఉపయోగకరమైన వాస్తవ ప్రపంచ సందర్భాలు

#### సులభ ఆథెంటికేషన్ (03-GettingStarted/11-simple-auth)
- **కొత్త అధ్యాయం జోడించబడింది**: MCP సర్వర్లలో సులభ ఆథెంటికేషన్ అమలుకు దశల వారీ గైడ్
  - **ఆథ్ కాన్సెప్ట్‌లు**: ఆథెంటికేషన్ vs. ఆథరైజేషన్, క్రెడెన్షియల్ హ్యాండ్లింగ్ స్పష్టమైన వివరణ
  - **బేసిక్ ఆథ్ అమలు**: Python (Starlette) మరియు TypeScript (Express) లో మిడిల్‌వేర్-ఆధారిత ఆథెంటికేషన్ నమూనాలు, కోడ్ ఉదాహరణలతో
  - **అధునాతన భద్రతకు ప్రగతి**: సులభ ఆథ్ నుండి OAuth 2.1 మరియు RBAC కు మార్గదర్శనం, అధునాతన భద్రతా మాడ్యూల్స్ సూచనలతో

ఈ జోడింపులు మరింత బలమైన, భద్రతా పరమైన, మరియు ఫ్లెక్సిబుల్ MCP సర్వర్ అమలులకు ప్రాయోగిక, చేతితో నేర్చుకునే మార్గదర్శకాన్ని అందిస్తాయి, ప్రాథమిక భావనలను అధునాతన ఉత్పత్తి నమూనాలతో కలుపుతూ.

  - సర్వర్ ప్రిమిటివ్‌లు: వనరులు (డేటా మూలాలు), ప్రాంప్ట్‌లు (టెంప్లేట్లు), టూల్స్ (నిర్వహించదగిన ఫంక్షన్లు) వివరణలు మరియు ఉదాహరణలతో  
  - క్లయింట్ ప్రిమిటివ్‌లు: శాంప్లింగ్ (LLM పూర్తి), ఎలిసిటేషన్ (వినియోగదారు ఇన్‌పుట్), లాగింగ్ (డీబగ్గింగ్/పర్యవేక్షణ)  
  - ప్రస్తుత డిస్కవరీ (`*/list`), రిట్రీవల్ (`*/get`), మరియు ఎగ్జిక్యూషన్ (`*/call`) పద్ధతుల నమూనాలతో నవీకరణ  
- **ప్రోటోకాల్ ఆర్కిటెక్చర్**: రెండు-పట్టాల ఆర్కిటెక్చర్ మోడల్ పరిచయం  
  - డేటా లేయర్: JSON-RPC 2.0 ఆధారం, లైఫ్‌సైకిల్ నిర్వహణ మరియు ప్రిమిటివ్‌లు  
  - ట్రాన్స్‌పోర్ట్ లేయర్: STDIO (స్థానిక) మరియు స్ట్రీమబుల్ HTTP తో SSE (రిమోట్) ట్రాన్స్‌పోర్ట్ మెకానిజమ్స్  
- **భద్రతా ఫ్రేమ్‌వర్క్**: స్పష్టమైన వినియోగదారు అనుమతి, డేటా గోప్యతా రక్షణ, టూల్ ఎగ్జిక్యూషన్ భద్రత, మరియు ట్రాన్స్‌పోర్ట్ లేయర్ భద్రతతో సమగ్ర భద్రతా సూత్రాలు  
- **సంవాద నమూనాలు**: ప్రారంభం, డిస్కవరీ, ఎగ్జిక్యూషన్, మరియు నోటిఫికేషన్ ప్రవాహాలను చూపించే ప్రోటోకాల్ సందేశాలు నవీకరించబడ్డాయి  
- **కోడ్ ఉదాహరణలు**: ప్రస్తుత MCP SDK నమూనాలను ప్రతిబింబించే బహుభాషా ఉదాహరణలు (.NET, Java, Python, JavaScript) తాజా చేయబడ్డాయి  

#### భద్రత (02-Security/) - సమగ్ర భద్రతా పునఃరూపకల్పన  
- **ప్రమాణాల అనుగుణత**: MCP స్పెసిఫికేషన్ 2025-06-18 భద్రతా అవసరాలతో పూర్తి అనుగుణత  
- **ధృవీకరణ అభివృద్ధి**: కస్టమ్ OAuth సర్వర్ల నుండి బాహ్య గుర్తింపు ప్రొవైడర్ డెలిగేషన్ (Microsoft Entra ID) వరకు అభివృద్ధి డాక్యుమెంటేషన్  
- **AI-స్పెసిఫిక్ ముప్పు విశ్లేషణ**: ఆధునిక AI దాడి వెక్టర్ల మెరుగైన కవరేజ్  
  - వాస్తవ ప్రపంచ ఉదాహరణలతో ప్రాంప్ట్ ఇంజెక్షన్ దాడి సన్నివేశాలు  
  - టూల్ విషపూరణ యంత్రాంగాలు మరియు "రగ్ పుల్" దాడి నమూనాలు  
  - కాంటెక్స్ట్ విండో విషపూరణ మరియు మోడల్ గందరగోళ దాడులు  
- **Microsoft AI భద్రతా పరిష్కారాలు**: Microsoft భద్రతా ఎకోసిస్టమ్ యొక్క సమగ్ర కవరేజ్  
  - అధునాతన గుర్తింపు, స్పాట్‌లైటింగ్, మరియు డెలిమిటర్ సాంకేతికతలతో AI Prompt Shields  
  - Azure Content Safety ఇంటిగ్రేషన్ నమూనాలు  
  - సరఫరా గొలుసు రక్షణ కోసం GitHub Advanced Security  
- **అధునాతన ముప్పు నివారణ**:  
  - MCP-స్పెసిఫిక్ దాడి సన్నివేశాలు మరియు క్రిప్టోగ్రాఫిక్ సెషన్ ID అవసరాలతో సెషన్ హైజాకింగ్  
  - స్పష్టమైన అనుమతి అవసరాలతో MCP ప్రాక్సీ సన్నివేశాలలో గందరగోళం సమస్యలు  
  - తప్పనిసరి ధృవీకరణ నియంత్రణలతో టోకెన్ పాస్స్త్రూ లోపాలు  
- **సరఫరా గొలుసు భద్రత**: ఫౌండేషన్ మోడల్స్, ఎంబెడ్డింగ్స్ సేవలు, కాంటెక్స్ట్ ప్రొవైడర్లు, మరియు మూడవ పక్ష APIలతో విస్తరించిన AI సరఫరా గొలుసు కవరేజ్  
- **ఫౌండేషన్ భద్రత**: జీరో ట్రస్ట్ ఆర్కిటెక్చర్ మరియు Microsoft భద్రతా ఎకోసిస్టమ్‌తో ఎంటర్ప్రైజ్ భద్రతా నమూనాల మెరుగైన ఇంటిగ్రేషన్  
- **వనరు సంస్థాపన**: రకం ప్రకారం విభజించిన సమగ్ర వనరు లింకులు (అధికారిక డాక్స్, ప్రమాణాలు, పరిశోధన, Microsoft పరిష్కారాలు, అమలు మార్గదర్శకాలు)  

### డాక్యుమెంటేషన్ నాణ్యత మెరుగుదలలు  
- **సంరచిత అభ్యాస లక్ష్యాలు**: నిర్దిష్ట, కార్యాచరణాత్మక ఫలితాలతో మెరుగైన అభ్యాస లక్ష్యాలు

- **ప్రస్తుత సమాచారం**: అన్ని తేదీ సూచనలు మరియు స్పెసిఫికేషన్ లింకులు ప్రస్తుత ప్రమాణాలకు నవీకరించబడ్డాయి  
- **అమలు మార్గదర్శకాలు**: రెండు విభాగాలలో నిర్దిష్ట, కార్యాచరణాత్మక అమలు మార్గదర్శకాలు జోడించబడ్డాయి
