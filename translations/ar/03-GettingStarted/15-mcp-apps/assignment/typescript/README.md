## كيف تم هيكلة هذا الحل

يتكون الحل من الجزء المهم فقط في الكود، أي العلامة، وربط الأحداث وخصائص الخادم.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

## كيف يتم التشغيل

انظر [الكود](../../code/typescript/README.md) وأضف المحتوى من هذه الملفات إلى كل ملف على التوالي.
