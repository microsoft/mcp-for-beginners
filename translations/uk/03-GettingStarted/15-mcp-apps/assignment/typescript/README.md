Rock paper scissor приклад

## Як побудоване це рішення

Рішення складається лише з тієї частини коду, яка має значення, тобто розмітки, підключення подій та серверних функцій.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

## Як запустити

Дивіться [Code](../../code/typescript/README.md) та додайте вміст цих файлів у відповідні файли.
