# Приклад: Базовий Хост

Референсна реалізація, що показує, як побудувати MCP хост-додаток, який підключається до MCP серверів та відображає інтерфейси інструментів у захищеному середовищі.

Цей базовий хост також можна використовувати для тестування MCP додатків під час локальної розробки.

## Ключові файли

- [`index.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/index.html) / [`src/index.tsx`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/index.tsx) - React UI хост з вибором інструментів, введенням параметрів та керуванням iframe
- [`sandbox.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/sandbox.html) / [`src/sandbox.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/sandbox.ts) - Зовнішній проксі iframe з валідацією безпеки та двонаправленою пересилкою повідомлень
- [`src/implementation.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/implementation.ts) - Основна логіка: підключення до сервера, виклик інструментів і налаштування AppBridge

## Початок роботи

```bash
npm install
npm run start
# Відкрийте http://localhost:8080
```

За замовчуванням хост-додаток намагатиметься підключитись до MCP сервера за адресою `http://localhost:3001/mcp`. Ви можете налаштувати цю поведінку, встановивши змінну середовища `SERVERS` з JSON масивом URL серверів:

```bash
SERVERS='["http://localhost:1234/mcp", "http://localhost:5678/mcp"]' npm run start
```

## Архітектура

Цей приклад використовує патерн подвійного iframe для безпечної ізоляції UI:

```
Host (port 8080)
  └── Outer iframe (port 8081) - sandbox proxy
        └── Inner iframe (srcdoc) - untrusted tool UI
```

**Чому два iframe?**

- Зовнішній iframe працює на окремому походженні (порт 8081), що запобігає прямому доступу до хоста
- Внутрішній iframe отримує HTML через `srcdoc` і обмежується атрибутами sandbox
- Повідомлення проходять через зовнішній iframe, який їх перевіряє та двонаправлено пересилає

Ця архітектура гарантує, що навіть якщо код UI інструменту є шкідливим, він не зможе отримати доступ до DOM хост-додатку, cookies або контексту JavaScript.
