Rock paper scissor пример

## Как е структурирано това решение

Решението съдържа само частта от кода, която е важна, т.е. маркирането, свързването на събитията и сървърните функции.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

Вижте [Code](../../code/typescript/README.md) и добавете съдържанието от тези файлове във всеки съответен файл.
