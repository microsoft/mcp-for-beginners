# 運行此範例

建議你安裝 `uv`，但非必須，詳見 [instructions](https://docs.astral.sh/uv/#highlights)

## -0- 建立虛擬環境

```bash
python -m venv venv
```

## -1- 啟動虛擬環境

```bash
venv\Scripts\activate
```

## -2- 安裝相依套件

```bash
pip install "mcp[cli]" openai
```

## -3- 執行範例


```bash
python client.py
```
