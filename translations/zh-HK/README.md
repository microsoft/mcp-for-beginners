# 🚀 模型上下文協議（MCP）初學者課程

## **透過 C#、Java、JavaScript、Rust、Python 和 TypeScript 的實作範例學習 MCP**

## 🧠 模型上下文協議課程總覽
歡迎踏上模型上下文協議的學習旅程！如果你曾好奇 AI 應用程式如何與不同工具和服務溝通，你即將發現這項正在改變開發者打造智能系統方式的優雅解決方案。

想像 MCP 是 AI 應用的通用翻譯器——就像 USB 埠讓你能將任何裝置接到電腦上，MCP 讓 AI 模型能以標準化的方式連接任何工具或服務。無論你是在打造第一個聊天機器人，或開發複雜的 AI 工作流程，了解 MCP 將讓你有能力創造更強大且更靈活的應用。

這套課程以耐心和細心設計，陪伴你的學習旅程。我們會從你已經熟悉的簡單概念開始，透過你喜愛的程式語言的實作練習，逐步增強你的專業能力。每個步驟都包含清楚的解說、實用範例，以及滿滿的鼓勵。

完成這趟旅程後，你將有信心自行構建 MCP 伺服器，把它們整合到熱門 AI 平台，並理解這項技術如何重塑 AI 開發的未來。讓我們一起開始這趟精彩的冒險吧！

這些資源隨著你理解加深會越來越有價值，但不必一開始就全讀完。先從你最感興趣的部分開始吧！

- 📜 [MCP 規範](https://modelcontextprotocol.io/specification/2025-11-25) – 這是你的完整參考手冊。隨著課程進行，你會常回來查閱特定細節及探索進階功能。
- 📜 [MCP 規範版本記錄](https://modelcontextprotocol.io/specification/versioning) – 介紹協議版本歷史與 MCP 如何使用基於日期的版本編號。

• **自信部署**：你會知道如何將 MCP 專案從開發階段推向生產環境，採用實用且可行的部署策略。

• **加入 MCP 社群**：你將成為不斷壯大的開發者社群一員，共同塑造 AI 應用開發的未來。

## 基礎背景知識

在深入 MCP 詳細內容前，先確認你對一些基礎概念感到熟悉。別擔心，即使不是專家，我們會邊學邊解說所有所需知識！

### 理解協議（基礎）

想像協議像是談話的規則。你打電話給朋友時，雙方都知道要說「哈囉」、輪流講話，完畢要說再見。電腦程式也需要相似規則才能有效溝通。

MCP 就是一套協議——一組雙方認可的規則，幫助 AI 模型和應用與工具及服務有效「對話」。正如談話規則促進人與人間順暢溝通，MCP 讓 AI 應用溝通更可靠、更強大。

### 用戶端-伺服器關係（程式如何合作）

你每天都在使用用戶端-伺服器關係！當你用瀏覽器（用戶端）瀏覽網站，就是跟網頁伺服器連線，伺服器會送出網頁內容。瀏覽器知道如何索取資料，伺服器懂得如何回應。

MCP 也是类似關係：AI 模型扮演用戶端，請求資訊或動作；MCP 伺服器則提供這些功能。就像有個幫手（伺服器）隨時聽從 AI 的需求。

### 標準化的重要（讓一切順利配合）

想像若所有車廠都使用不同規格加油槍，你就必須為每輛車準備不同轉接頭！標準化就是約定共通方式，讓東西能無縫配合。

MCP 為 AI 應用提供這種標準化。AI 模型不需為每個工具寫特別程式碼，MCP 就是通用的溝通橋樑。開發者只需寫一次工具，就能配合多種 AI 系統。

## 🧭 你的學習路徑總覽

你的 MCP 學習之旅經過精心規劃，讓你逐步建立信心與技能。每個階段引入新概念，同時鞏固已學內容。

### 🌱 基礎階段：理解基本概念（模組 0-2）

冒險從這裡開始！我們用熟悉的比喻和簡單範例介紹 MCP 概念。你會明白 MCP 是什麼、為何存在，以及它在 AI 開發大環境中的角色。

• **模組 0 - MCP 簡介**：從探索 MCP 是什麼開始，了解它對現代 AI 應用的重要性。你會看到 MCP 的真實應用範例，以及它如何解決開發者常遇問題。

• **模組 1 - 核心概念解析**：學習 MCP 的基本架構。透過大量比喻和視覺範例，讓概念自然好懂。

• **模組 2 - MCP 安全性**：安全聽起來可能令人害怕，但我們會展示 MCP 內建的安全機制，並教你從一開始就採用保障應用的最佳做法。

### 🔨 建構階段：打造你的第一個實作（模組 3）

現在正是實作的好時機！你將親自動手打造 MCP 伺服器和用戶端。別擔心，我們會從簡單開始，細心引導你完成每一步。
此模組包括多個實作指南，讓您可以使用自己偏好的程式語言進行練習。您將建立第一個伺服器，建構用於連接伺服器的用戶端，甚至整合像 VS Code 這樣受歡迎的開發工具。

每個指南都包含完整的程式範例、故障排除技巧，以及我們做出特定設計決策的原因解釋。完成本階段後，您將擁有可自豪的可運作 MCP 實作！

### 🚀 成長階段：進階概念與實務應用（模組 4-5）

掌握基礎後，您將準備探索更複雜的 MCP 功能。我們會涵蓋實務執行策略、除錯技巧，以及像多模態 AI 整合等進階主題。

您還會學習如何擴展 MCP 實作以用於生產，並整合像 Azure 這樣的雲端平台。這些模組旨在讓您能建置可應付實務需求的 MCP 解決方案。

### 🌟 精通階段：社群與專精（模組 6-11）

最後階段聚焦於加入 MCP 社群與專注於您最感興趣的領域。您將學習如何為開源 MCP 專案貢獻程式碼、實作進階的身份驗證模式，並打造完整的資料庫整合解決方案。

特別要提到模組 11 —— 這是一個包含 13 個實作實驗的完整學習路徑，教您建立具 PostgreSQL 整合的生產就緒 MCP 伺服器，就像是結合所有學習內容的壓軸專案！
