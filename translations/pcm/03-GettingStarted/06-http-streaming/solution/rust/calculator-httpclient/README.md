# How to Run Dis Sample

## -1- Run di Streamable HTTP Server

Open terminal from di root of dis project and run di commands wey dey below:

```bash
cd ../calculator-httpserver
cargo run
```

Keep dis terminal open because e go dey run di HTTP server wey di client go connect to. Open new terminal for di next steps.

## -2- Install di dependencies and build di project

```bash
cargo build
```

## -3- Run di sample

```bash
cargo run
```
