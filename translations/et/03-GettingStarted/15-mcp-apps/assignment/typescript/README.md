## Kuidas see lahendus on üles ehitatud

Lahendus koosneb ainult vajalikust koodiosast, st märgistusest, sündmuste ühendamisest ja serveri funktsioonidest. 

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
