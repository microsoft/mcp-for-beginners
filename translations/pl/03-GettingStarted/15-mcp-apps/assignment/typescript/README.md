Rock paper scissor przykład

## Jak ta solucja jest zbudowana

Solucja składa się tylko z istotnej części kodu, czyli markup, podłączenia zdarzeń i funkcji serwera. 

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
