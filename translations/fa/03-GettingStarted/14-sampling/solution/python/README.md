## -0- ایجاد یک محیط مجازی

```bash
python -m venv venv
```

## -1- فعال‌سازی محیط مجازی

```bash
venv\Scripts\activate
```

## -2- نصب وابستگی‌ها

```bash
pip install "mcp[cli]" openai
```

```bash
python client.py
```
