# تغییرات: برنامه درسی MCP برای مبتدیان

این سند به عنوان رکورد تمام تغییرات مهم ایجاد شده در برنامه درسی پروتکل مدل کانتکست (MCP) برای مبتدیان خدمت می‌کند. تغییرات به ترتیب زمانی معکوس (جدیدترین تغییرات ابتدا) مستند شده‌اند.

## ۱۸ دسامبر ۲۰۲۵

### به‌روزرسانی مستندات امنیتی - مشخصات MCP ۲۰۲۵-۱۱-۲۵

#### بهترین شیوه‌های امنیتی MCP (02-Security/mcp-best-practices.md) - به‌روزرسانی نسخه مشخصات
- **به‌روزرسانی نسخه پروتکل**: به‌روزرسانی برای ارجاع به جدیدترین مشخصات MCP ۲۰۲۵-۱۱-۲۵ (منتشر شده در ۲۵ نوامبر ۲۰۲۵)
  - به‌روزرسانی تمام ارجاعات نسخه مشخصات از ۲۰۲۵-۰۶-۱۸ به ۲۰۲۵-۱۱-۲۵
  - به‌روزرسانی ارجاعات تاریخ سند از ۱۸ اوت ۲۰۲۵ به ۱۸ دسامبر ۲۰۲۵
  - تأیید صحت تمام URLهای مشخصات که به مستندات جاری اشاره دارند
- **اعتبارسنجی محتوا**: اعتبارسنجی جامع بهترین شیوه‌های امنیتی در برابر جدیدترین استانداردها
  - **راهکارهای امنیتی مایکروسافت**: تأیید اصطلاحات و لینک‌های جاری برای Prompt Shields (قبلاً "شناسایی ریسک Jailbreak")، Azure Content Safety، Microsoft Entra ID و Azure Key Vault
  - **امنیت OAuth 2.1**: تأیید تطابق با بهترین شیوه‌های امنیتی جدید OAuth
  - **استانداردهای OWASP**: اعتبارسنجی مراجع OWASP Top 10 برای LLMها که به‌روز هستند
  - **خدمات Azure**: تأیید تمام لینک‌های مستندات مایکروسافت Azure و بهترین شیوه‌ها
- **تطابق با استانداردها**: تأیید به‌روز بودن تمام استانداردهای امنیتی ارجاع شده
  - چارچوب مدیریت ریسک هوش مصنوعی NIST
  - ISO 27001:2022
  - بهترین شیوه‌های امنیتی OAuth 2.1
  - چارچوب‌های امنیتی و انطباق Azure
- **منابع پیاده‌سازی**: تأیید تمام لینک‌ها و منابع راهنمای پیاده‌سازی
  - الگوهای احراز هویت Azure API Management
  - راهنماهای ادغام Microsoft Entra ID
  - مدیریت اسرار Azure Key Vault
  - خطوط لوله و راهکارهای نظارت DevSecOps

### تضمین کیفیت مستندات
- **تطابق با مشخصات**: اطمینان از تطابق تمام الزامات امنیتی اجباری MCP (باید/نباید) با جدیدترین مشخصات
- **به‌روزرسانی منابع**: تأیید تمام لینک‌های خارجی به مستندات مایکروسافت، استانداردهای امنیتی و راهنماهای پیاده‌سازی
- **پوشش بهترین شیوه‌ها**: تأیید پوشش جامع احراز هویت، مجوزدهی، تهدیدات خاص هوش مصنوعی، امنیت زنجیره تأمین و الگوهای سازمانی

## ۶ اکتبر ۲۰۲۵

### گسترش بخش شروع به کار – استفاده پیشرفته از سرور و احراز هویت ساده

#### استفاده پیشرفته از سرور (03-GettingStarted/10-advanced)
- **فصل جدید اضافه شده**: معرفی راهنمای جامع استفاده پیشرفته از سرور MCP، شامل معماری‌های سرور معمولی و سطح پایین.
  - **سرور معمولی در مقابل سطح پایین**: مقایسه دقیق و نمونه‌های کد در پایتون و تایپ‌اسکریپت برای هر دو رویکرد.
  - **طراحی مبتنی بر هندلر**: توضیح مدیریت ابزار/منبع/پرومپت مبتنی بر هندلر برای پیاده‌سازی‌های سرور مقیاس‌پذیر و انعطاف‌پذیر.
  - **الگوهای عملی**: سناریوهای واقعی که الگوهای سرور سطح پایین برای ویژگی‌ها و معماری پیشرفته مفید هستند.

#### احراز هویت ساده (03-GettingStarted/11-simple-auth)
- **فصل جدید اضافه شده**: راهنمای گام‌به‌گام برای پیاده‌سازی احراز هویت ساده در سرورهای MCP.
  - **مفاهیم احراز هویت**: توضیح واضح تفاوت احراز هویت و مجوزدهی و مدیریت اعتبارنامه‌ها.
  - **پیاده‌سازی احراز هویت پایه**: الگوهای احراز هویت مبتنی بر میان‌افزار در پایتون (Starlette) و تایپ‌اسکریپت (Express) با نمونه کد.
  - **پیشرفت به امنیت پیشرفته**: راهنمای شروع با احراز هویت ساده و پیشرفت به OAuth 2.1 و RBAC، با ارجاع به ماژول‌های امنیتی پیشرفته.

این افزودنی‌ها راهنمایی عملی و کاربردی برای ساخت پیاده‌سازی‌های سرور MCP مقاوم‌تر، امن‌تر و انعطاف‌پذیرتر فراهم می‌کنند و مفاهیم پایه را با الگوهای تولید پیشرفته پیوند می‌دهند.
