Rock paper scissor-eksempel

## Slik er denne løsningen strukturert

Løsningen består kun av den delen av koden som er viktig, altså markup, hendelsesoppsett og serverfunksjoner.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

## Hvordan kjøre

Se [Code](../../code/typescript/README.md) og legg inn innholdet fra disse filene i hver respektive fil.
