# Дневник промена: MCP за почетнике наставни план

Овај документ служи као запис свих значајних промена направљених у наставном плану Model Context Protocol (MCP) за почетнике. Промене су документоване у обрнутом хронолошком редоследу (најновије прво).

## 18. децембар 2025.

### Ажурирање документације о безбедности - MCP спецификација 2025-11-25

#### Најбоље праксе безбедности MCP-а (02-Security/mcp-best-practices.md) - Ажурирање верзије спецификације
- **Ажурирање верзије протокола**: Ажурирано да се реферише на најновију MCP спецификацију 2025-11-25 (објављена 25. новембра 2025)
  - Ажуриране све референце верзије спецификације са 2025-06-18 на 2025-11-25
  - Ажуриране референце датума документа са 18. августа 2025. на 18. децембар 2025.
  - Верификовано да све URL адресе спецификације показују на тренутну документацију
- **Валидација садржаја**: Комплетна валидација најбољих пракси безбедности према најновијим стандардима
  - **Microsoft безбедносна решења**: Верификовани тренутни термини и линкови за Prompt Shields (раније "детекција ризика од Jailbreak-а"), Azure Content Safety, Microsoft Entra ID и Azure Key Vault
  - **OAuth 2.1 безбедност**: Потврђена усаглашеност са најновијим најбољим праксама безбедности OAuth-а
  - **OWASP стандарди**: Верификовани референци OWASP Top 10 за LLM-ове су и даље актуелни
  - **Azure услуге**: Верификовани сви Microsoft Azure документи и најбоље праксе
- **Усклађеност са стандардима**: Сви реферисани безбедносни стандарди потврђени као актуелни
  - NIST AI Risk Management Framework
  - ISO 27001:2022
  - OAuth 2.1 Security Best Practices
  - Azure безбедносни и комплајанс оквири
- **Ресурси за имплементацију**: Верификовани сви линкови и ресурси за имплементацију
  - Azure API Management аутентификациони обрасци
  - Microsoft Entra ID интеграциони водичи
  - Azure Key Vault управљање тајнама
  - DevSecOps цевоводи и решења за мониторинг

### Осигурање квалитета документације
- **Усклађеност са спецификацијом**: Осигурано да сви обавезни MCP безбедносни захтеви (MUST/MUST NOT) буду у складу са најновијом спецификацијом

## 6. октобар 2025.

### Проширење секције Почетак – Напредна употреба сервера и једноставна аутентификација

#### Напредна употреба сервера (03-GettingStarted/10-advanced)
- **Додато ново поглавље**: Уведен свеобухватан водич за напредну употребу MCP сервера, обухватајући и редовне и ниско-нивоу серверске архитектуре.
  - **Редован у односу на ниско-нивоу сервер**: Детаљно поређење и примери кода у Python-у и TypeScript-у за оба приступа.
  - **Дизајн заснован на хендлерима**: Објашњење управљања алатима/ресурсима/промптовима засновано на хендлерима за скалабилне и флексибилне серверске имплементације.
  - **Практични обрасци**: Стварни сценарији где су ниско-нивоу серверски обрасци корисни за напредне функције и архитектуру.

#### Једноставна аутентификација (03-GettingStarted/11-simple-auth)
- **Додато ново поглавље**: Корак по корак водич за имплементацију једноставне аутентификације у MCP серверима.
  - **Концепти аутентификације**: Јасно објашњење разлике између аутентификације и ауторизације, и руковање акредитивима.
  - **Основна имплементација аутентификације**: Обрасци аутентификације засновани на middleware-у у Python-у (Starlette) и TypeScript-у (Express), са примерима кода.
  - **Напредак ка напредној безбедности**: Упутства за почетак са једноставном аутентификацијом и напредовање ка OAuth 2.1 и RBAC, са референцама на напредне безбедносне модуле.

Ови додаци пружају практичне, практичне смернице за изградњу робуснијих, безбеднијих и флексибилнијих MCP серверских имплементација, повезујући основне концепте са напредним производним обрасцима.
