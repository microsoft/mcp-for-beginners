Rock paper scissor пример

## Како је ово решење структуирано

Решење се састоји само из дела кода који је битан, односно ознака, повезивање догађаја и серверске функције.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

## Како покренути

Погледајте [Code](../../code/typescript/README.md) и додајте садржај из ових фајлова у одговарајуће фајлове.
