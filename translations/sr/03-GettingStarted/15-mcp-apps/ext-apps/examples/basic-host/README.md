# Пример: Основни Хост

Референтна имплементација која показује како изградити MCP хост апликацију која се повезује на MCP сервере и приказује корисничке интерфејсе алата у безбедном сандбоксу.

Овај основни хост се такође може користити за тестирање MCP апликација током локалног развоја.

## Кључне Датотеке

- [`index.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/index.html) / [`src/index.tsx`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/index.tsx) - React UI хост са избором алата, уносом параметара и управљањем iframe-ом
- [`sandbox.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/sandbox.html) / [`src/sandbox.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/sandbox.ts) - Спољни iframe прокси са провером безбедности и двосмерним прослеђивањем порука
- [`src/implementation.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/implementation.ts) - Основна логика: веза са сервером, позив алата и подешавање AppBridge-а

## Почетак рада

```bash
npm install
npm run start
# Отворите http://localhost:8080
```

Подразумевано, хост апликација ће покушати да се повеже на MCP сервер на `http://localhost:3001/mcp`. Ово понашање можете конфигурисати тако што ћете поставити окружење променљиву `SERVERS` са JSON низом URL-ова сервера:

```bash
SERVERS='["http://localhost:1234/mcp", "http://localhost:5678/mcp"]' npm run start
```

## Архитектура

Овај пример користи образац сандбокса са двоструким iframe-ом за сигурну изолацију корисничког интерфејса:

```
Host (port 8080)
  └── Outer iframe (port 8081) - sandbox proxy
        └── Inner iframe (srcdoc) - untrusted tool UI
```

**Зашто два iframe-а?**

- Спољни iframe ради на посебном пореклу (порт 8081) спречавајући директан приступ хосту
- Унутрашњи iframe добија HTML путем `srcdoc` и ограничен је sandbox атрибутима
- Поруке пролазе кроз спољни iframe који их верификује и двосмерно прослеђује

Ова архитектура обезбеђује да чак и ако је код корисничког интерфејса алата злонамеран, не може приступити DOM-у, колачићима или JavaScript контексту хост апликације.
