Rock paper scissor sample

## Bagaimana solusi ini disusun

Solusi ini hanya terdiri dari bagian kode yang penting, yaitu markup, pengikatan event, dan fitur server.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

## Cara menjalankan

Lihat [Code](../../code/typescript/README.md) dan tambahkan konten dari file-file ini ke masing-masing file terkait.
