## โครงสร้างของวิธีแก้ปัญหานี้

วิธีแก้ปัญหานี้ประกอบด้วยเฉพาะส่วนของโค้ดที่สำคัญ เช่น เครื่องหมาย, การเชื่อมโยงเหตุการณ์ และฟีเจอร์เซิร์ฟเวอร์

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

## วิธีการรัน

ดูได้ที่ [Code](../../code/typescript/README.md) และเพิ่มเนื้อหาจากไฟล์เหล่านี้ลงในแต่ละไฟล์ที่เกี่ยวข้อง
