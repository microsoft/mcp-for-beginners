# மாற்றங்கள்: MCP ஆரம்பக்கல்வி பாடத்திட்டம்

இந்த ஆவணம் Model Context Protocol (MCP) ஆரம்பக்கல்வி பாடத்திட்டத்தில் செய்யப்பட்ட அனைத்து முக்கிய மாற்றங்களின் பதிவாக செயல்படுகிறது. மாற்றங்கள் நேர்மறை கால வரிசையில் (புதிய மாற்றங்கள் முதலில்) பதிவுசெய்யப்பட்டுள்ளன.

  - அனைத்து விவரக்குறிப்பு பதிப்பு குறிப்புகளை 2025-06-18 இருந்து 2025-11-25 ஆக புதுப்பித்தது

  - அனைத்து விவரக்குறிப்பு URLகள் தற்போதைய ஆவணத்துக்கு செல்லும் என்பதை உறுதிப்படுத்தியது

  - **Microsoft பாதுகாப்பு தீர்வுகள்**: Prompt Shields (முந்தைய "Jailbreak risk detection"), Azure Content Safety, Microsoft Entra ID மற்றும் Azure Key Vault இற்கான தற்போதைய சொற்பொருள் மற்றும் இணைப்புகள் சரிபார்க்கப்பட்டன
  - **OAuth 2.1 பாதுகாப்பு**: சமீபத்திய OAuth பாதுகாப்பு சிறந்த நடைமுறைகளுடன் ஒத்திசைவு உறுதிப்படுத்தப்பட்டது
  - **OWASP தரநிலைகள்**: LLMகளுக்கான OWASP Top 10 குறிப்பு தற்போதையதாக உள்ளது
  - **Azure சேவைகள்**: அனைத்து Microsoft Azure ஆவண இணைப்புகள் மற்றும் சிறந்த நடைமுறைகள் சரிபார்க்கப்பட்டன
- **தரநிலை ஒத்திசைவு**: அனைத்து குறிப்பிடப்பட்ட பாதுகாப்பு தரநிலைகள் தற்போதையதாக உறுதிப்படுத்தப்பட்டன
  - NIST AI அபாய மேலாண்மை கட்டமைப்பு
  - ISO 27001:2022
  - OAuth 2.1 பாதுகாப்பு சிறந்த நடைமுறைகள்
  - Azure பாதுகாப்பு மற்றும் இணக்கமான கட்டமைப்புகள்
- **நிறைவேற்றல் வளங்கள்**: அனைத்து நிறைவேற்றல் வழிகாட்டி இணைப்புகள் மற்றும் வளங்கள் சரிபார்க்கப்பட்டன
  - Azure API மேலாண்மை அங்கீகார மாதிரிகள்
  - Microsoft Entra ID ஒருங்கிணைப்பு வழிகாட்டிகள்
  - Azure Key Vault ரகசிய மேலாண்மை

### ஆவண தரம் உறுதி
- **விவரக்குறிப்பு இணக்கம்**: அனைத்து கட்டாய MCP பாதுகாப்பு தேவைகள் (MUST/MUST NOT) சமீபத்திய விவரக்குறிப்புடன் ஒத்திசைக்கப்படுவதை உறுதிப்படுத்தியது
- **வளங்கள் தற்போதைய நிலை**: Microsoft ஆவணங்கள், பாதுகாப்பு தரநிலைகள் மற்றும் நிறைவேற்றல் வழிகாட்டிகளுக்கான அனைத்து வெளிப்புற இணைப்புகள் சரிபார்க்கப்பட்டன
- **சிறந்த நடைமுறைகள் பரப்புரை**: அங்கீகாரம், அங்கீகாரமளித்தல், AI-சார்ந்த அச்சுறுத்தல்கள், வழங்கல் சங்கிலி பாதுகாப்பு மற்றும் நிறுவன மாதிரிகள் முழுமையாக உள்ளடக்கப்பட்டன

## அக்டோபர் 6, 2025

### தொடக்கம் பகுதி விரிவாக்கம் – மேம்பட்ட சர்வர் பயன்பாடு மற்றும் எளிய அங்கீகாரம்

#### மேம்பட்ட சர்வர் பயன்பாடு (03-GettingStarted/10-advanced)
- **புதிய அத்தியாயம் சேர்க்கப்பட்டது**: வழக்கமான மற்றும் குறைந்த நிலை சர்வர் கட்டமைப்புகளை உள்ளடக்கிய மேம்பட்ட MCP சர்வர் பயன்பாட்டுக்கான விரிவான வழிகாட்டி அறிமுகப்படுத்தப்பட்டது.
  - **வழக்கமான மற்றும் குறைந்த நிலை சர்வர்**: இரு அணுகுமுறைகளுக்குமான Python மற்றும் TypeScript குறியீடு உதாரணங்களுடன் விரிவான ஒப்பீடு.
  - **ஹேண்ட்லர் அடிப்படையிலான வடிவமைப்பு**: அளவிடக்கூடிய, நெகிழ்வான சர்வர் செயலாக்கங்களுக்கு ஹேண்ட்லர் அடிப்படையிலான கருவி/வள/ப்ராம்ட் மேலாண்மை விளக்கம்.
  - **பயிற்சி மாதிரிகள்**: மேம்பட்ட அம்சங்கள் மற்றும் கட்டமைப்புக்கு குறைந்த நிலை சர்வர் மாதிரிகள் பயன்படும் உண்மையான சூழல்கள்.

#### எளிய அங்கீகாரம் (03-GettingStarted/11-simple-auth)
- **புதிய அத்தியாயம் சேர்க்கப்பட்டது**: MCP சர்வர்களில் எளிய அங்கீகாரத்தை செயல்படுத்தும் படி படி வழிகாட்டி.
  - **அங்கீகாரம் கருத்துக்கள்**: அங்கீகாரம் மற்றும் அங்கீகாரமளித்தல், மற்றும் அங்கீகாரச் சான்றிதழ் கையாளுதல் தெளிவான விளக்கம்.
  - **அடிப்படை அங்கீகாரம் செயல்படுத்தல்**: Python (Starlette) மற்றும் TypeScript (Express) இல் மிடில்வேர் அடிப்படையிலான அங்கீகாரம் மாதிரிகள், குறியீடு உதாரணங்களுடன்.
  - **மேம்பட்ட பாதுகாப்புக்கு முன்னேற்றம்**: எளிய அங்கீகாரத்துடன் தொடங்கி OAuth 2.1 மற்றும் RBACக்கு முன்னேற வழிகாட்டல், மேம்பட்ட பாதுகாப்பு தொகுதிகளுக்கான குறிப்புகளுடன்.

இந்த சேர்ப்புகள் MCP சர்வர் செயலாக்கங்களை மேலும் வலுவான, பாதுகாப்பான மற்றும் நெகிழ்வானதாக கட்டுவதற்கான நடைமுறை, கைமுறை வழிகாட்டலை வழங்குகின்றன, அடிப்படைக் கருத்துக்களை மேம்பட்ட உற்பத்தி மாதிரிகளுடன் இணைக்கின்றன.

  - சர்வர்கள் உள்ளூர் மற்றும் தொலைநிலை அமர்வு சூழல்களுடன் மேம்படுத்தப்பட்டன
- **மூலவியல் மறுசீரமைப்பு**: சர்வர் மற்றும் கிளையன்ட் மூலவியல்களின் முழுமையான மறுசீரமைப்பு
  - சர்வர் மூலவியல்: வளங்கள் (தரவு மூலங்கள்), ப்ராம்ப்ட்கள் (வார்ப்புருக்கள்), கருவிகள் (செயல்படுத்தக்கூடிய செயல்பாடுகள்) விரிவான விளக்கங்களுடன்
  - கிளையன்ட் மூலவியல்: மாதிரிப்பெறுதல் (LLM நிறைவு), கேள்வி எழுப்புதல் (பயனர் உள்ளீடு), பதிவு (தவறுதிருத்தம்/கண்காணிப்பு)
  - தற்போதைய கண்டுபிடிப்பு (`*/list`), மீட்டெடுப்பு (`*/get`), மற்றும் செயல்படுத்தல் (`*/call`) முறை மாதிரிகள் புதுப்பிக்கப்பட்டன
- **புரோட்டோக்கால் கட்டமைப்பு**: இரண்டு அடுக்கு கட்டமைப்பு மாதிரி அறிமுகம்
  - தரவு அடுக்கு: JSON-RPC 2.0 அடித்தளம் மற்றும் வாழ்க்கைசுழற்சி மேலாண்மை மற்றும் மூலவியல்
  - போக்குவரத்து அடுக்கு: STDIO (உள்ளூர்) மற்றும் Streamable HTTP மற்றும் SSE (தொலைநிலை) போக்குவரத்து முறைகள்
- **பாதுகாப்பு கட்டமைப்பு**: தெளிவான பயனர் ஒப்புதல், தரவு தனியுரிமை பாதுகாப்பு, கருவி செயல்பாட்டு பாதுகாப்பு மற்றும் போக்குவரத்து அடுக்கு பாதுகாப்பு உட்பட விரிவான பாதுகாப்பு கொள்கைகள்
- **தொடர்பு மாதிரிகள்**: தொடக்கம், கண்டுபிடிப்பு, செயல்படுத்தல் மற்றும் அறிவிப்பு ஓட்டங்களை காட்டும் புரோட்டோக்கால் செய்தி வடிவங்கள் புதுப்பிக்கப்பட்டன
- **குறியீடு உதாரணங்கள்**: தற்போதைய MCP SDK மாதிரிகளுக்கு ஏற்ப பல மொழி உதாரணங்கள் (.NET, Java, Python, JavaScript) புதுப்பிக்கப்பட்டன

#### பாதுகாப்பு (02-Security/) - விரிவான பாதுகாப்பு மறுசீரமைப்பு  
- **தரநிலை ஒத்திசைவு**: MCP Specification 2025-06-18 பாதுகாப்பு தேவைகளுடன் முழுமையான ஒத்திசைவு
- **அங்கீகாரம் முன்னேற்றம்**: தனிப்பயன் OAuth சர்வர்களிலிருந்து வெளிப்புற அடையாள வழங்குநர் பிரதிநிதித்துவம் (Microsoft Entra ID) வரை வளர்ச்சி ஆவணப்படுத்தப்பட்டது
- **AI-சிறப்பு அச்சுறுத்தல் பகுப்பாய்வு**: நவீன AI தாக்குதல் வழிகளின் விரிவான கவரேஜ்
  - உண்மையான உலக உதாரணங்களுடன் விரிவான prompt injection தாக்குதல் காட்சிகள்
  - கருவி விஷப்பொருள் செயல்முறைகள் மற்றும் "ரக் புல்" தாக்குதல் மாதிரிகள்
  - சூழல் ஜன்னல் விஷப்பொருள் மற்றும் மாதிரி குழப்புதல் தாக்குதல்கள்
- **Microsoft AI பாதுகாப்பு தீர்வுகள்**: Microsoft பாதுகாப்பு சூழல் விரிவான கவரேஜ்
  - மேம்பட்ட கண்டறிதல், விளக்குதல் மற்றும் பிரிவுபடுத்தல் தொழில்நுட்பங்களுடன் AI Prompt Shields
  - Azure Content Safety ஒருங்கிணைப்பு மாதிரிகள்
  - வழங்கல் சங்கிலி பாதுகாப்புக்கான GitHub Advanced Security
- **முன்னேற்றப்பட்ட அச்சுறுத்தல் குறைப்பு**: விரிவான பாதுகாப்பு கட்டுப்பாடுகள்
  - MCP-சிறப்பு தாக்குதல் காட்சிகளுடன் அமர்வு கைப்பற்றல் மற்றும் குறியாக்க அமர்வு ID தேவைகள்
  - தெளிவான ஒப்புதல் தேவைகளுடன் MCP பிரதிநிதி சூழல்களில் குழப்பப்பட்ட துணை பிரச்சினைகள்
  - கட்டாய சரிபார்ப்பு கட்டுப்பாடுகளுடன் டோக்கன் பாஸ்த்ரூ பாதிப்புகள்
- **வழங்கல் சங்கிலி பாதுகாப்பு**: அடித்தள மாதிரிகள், எம்பெட்டிங் சேவைகள், சூழல் வழங்குநர்கள் மற்றும் மூன்றாம் தரப்பு API களை உள்ளடக்கிய AI வழங்கல் சங்கிலி விரிவாக்கம்
- **அடித்தள பாதுகாப்பு**: பூஜ்ய நம்பிக்கை கட்டமைப்பு மற்றும் Microsoft பாதுகாப்பு சூழல் உட்பட நிறுவன பாதுகாப்பு மாதிரிகளுடன் மேம்படுத்தப்பட்ட ஒருங்கிணைப்பு
- **வள அமைப்பு**: வகைப்படுத்தப்பட்ட விரிவான வள இணைப்புகள் (அதிகாரப்பூர்வ ஆவணங்கள், தரநிலைகள், ஆராய்ச்சி, Microsoft தீர்வுகள், நடைமுறை வழிகாட்டுதல்கள்)

### ஆவணத் தரம் மேம்பாடுகள்

- **தற்போதைய தகவல்**: அனைத்து தேதி குறிப்பு மற்றும் விவரக்குறிப்பு இணைப்புகள் தற்போதைய தரநிலைகளுக்கு புதுப்பிக்கப்பட்டன
- **நடைமுறை வழிகாட்டுதல்**: இரு பிரிவுகளிலும் குறிப்பிட்ட, செயல்படுத்தக்கூடிய நடைமுறை வழிகாட்டுதல்கள் சேர்க்கப்பட்டன
