# நடைமுறை நிறைவேற்றம்

[![உண்மையான கருவிகள் மற்றும் பண்பாட்டு வழிமுறைகளுடன் MCP செயலிகளை எப்படி உருவாக்க, சோதனை செய்ய மற்றும் பரப்புவது](../../../translated_images/ta/05.64bea204e25ca891.webp)](https://youtu.be/vCN9-mKBDfQ)

_(இந்த பாடத்தின் காணொளியைப் பார்க்க மேலே உள்ள படத்தை கிளிக் செய்க)_

நடைமுறை நிறைவேற்றம் என்பது Model Context Protocol (MCP) திறனின் தெளிவான வெளிப்பாடு ஆகும். MCP-வின் கோட்பாடு மற்றும் கட்டமைப்பை புரிந்து கொள்வது முக்கியமானதாக இருந்தாலும், உண்மையான மதிப்பு நீங்கள் இந்த கருத்துக்களை பயன்படுத்தி உண்மையான உலக பிரச்சனைகளை தீர்க்கும் தீர்வுகளை உருவாக்க, சோதனை செய்ய மற்றும் பரப்பும் போது வெளிப்படுகிறது. இந்த அத்தியாயம் கருத்தியல் அறிவுக்கும் செயல்பாடு வளர்ச்சிக்கும் இடையே பாலம் போடுகிறது, MCP-ஆல் அடிப்படையாக்கப்பட்ட செயலிகளை உருவாக்கும் செயல்முறையை வழிநடத்துகிறது.

நீங்கள் புத்திசாலி உதவியாளர்களை உருவாக்கினாலும், AI-ஐ வணிக பணிமுறைகளில் ஒருங்கிணைப்பதா, அல்லது தரவு செயலாக்கத்துக்கான தனிப்பயன் கருவிகளை உருவாக்கினாலும், MCP ஒரு தகுதியான அடித்தளத்தை வழங்குகிறது. அதன் மொழி-கோளாறற்ற வடிவமைப்பும் மற்றும் பிரபலமான நிரலை மொழிகளுக்கான அதிகாரப்பூர்வ SDKகளும் இதனை பரவலான மேம்படுத்துநர்களுக்கு அணுகக்கூடியதாக செய்கின்றன. இந்த SDKகளை பயன்படுத்தி, நீங்கள் விரைவில் மாதிரிப்படையை உருவாக்கி, மீண்டும் மீண்டும் சோதித்து, பல்வேறு தளங்களிலும் சூழல்களிலும் உங்கள் தீர்வுகளை விரிவுபடுத்தலாம்.

பின்வரும் பகுதிகளில், நீங்கள் C#, Java with Spring, TypeScript, JavaScript மற்றும் Python ஆகியவற்றில் MCP-ஐ எவ்வாறு நிறைவேற்றுவது என்பதை காட்டும் நடைமுறை உதாரணங்கள், மாதிரி குறியீடுகள் மற்றும் பரப்பு மூலோபாயங்களை காணக்கூடியீர்கள். MCP சேவைகளை எப்படி டிபக் மற்றும் சோதனை செய்வது, APIகளை கையாள்வது மற்றும் Azure கொண்டு மேகத்தில் தீர்வுகளை பரப்புவது ஆகியவற்றையும் கற்றுக்கொள்வீர்கள். இந்த நடைமுறை வளங்கள் உங்கள் கற்றலை எளிதாக்கி வலுவான, தயாரிப்புக்கு தயாரான MCP செயலிகளை அடையாளம் காண உதவுகின்றன.

## ஒட்டுமொத்த பார்வை

இந்த பாடம் MCP நிறைவேற்றத்தில் பல நிரலை மொழிகளில் நடைமுறை அம்சங்களை கவனம் செலுத்துகிறது. C#, Java with Spring, TypeScript, JavaScript மற்றும் Python ஆகிய MCP SDKகளை பயன்படுத்தி வலுவான செயலிகளை உருவாக்குவது, MCP சேவைகளை டிபக் மற்றும் சோதனை செய்வது மற்றும் மீண்டும் பயன்படுத்தக்கூடிய வளங்கள், தூண்டுகோற்கள் மற்றும் கருவிகள் உருவாக்குவது ஆகியவற்றைப் பரிசீலிப்போம்.

## கற்றல் நோக்கங்கள்

இந்த பாடம் முடிவில், நீங்கள்:

- பல நிரலை மொழிகளில் அதிகாரப்பூர்வ SDKகளை பயன்படுத்தி MCP தீர்வுகளை நிறைவேற்ற முடியும்
- MCP சேவைகளை முறையாக டிபக் மற்றும் சோதனை செய்ய முடியும்
- சேவையின் அம்சங்களை (வளங்கள், தூண்டுகோற்கள் மற்றும் கருவிகள்) உருவாக்கி பயன்படுத்த முடியும்
- சிக்கலான பணிகளுக்கான சிறந்த MCP பணிமுறைகளை வடிவமைக்க முடியும்
- செயலாற்றல் மற்றும் நம்பகத்தன்மைக்கான MCP நிறைவேற்றங்களை மேம்படுத்த முடியும்

## முதன்மை சேவையகம் அம்சங்கள்

MCP சேவைகள் பின்வரும் அம்சங்களில் எதையாவது அல்லது ஒன்றுக்கும் மேற்பட்டவற்றை நிறைவேற்ற முடியும்:

### வளங்கள்

வளங்கள் பயனர் அல்லது AI மாதிரிக்கு பயன்படுத்தப்படுவதற்கான சூழலும் தரவுகளையும் அளிக்கின்றன:

- ஆவண சேமிப்பகம்
- அறிவுத் தளங்கள்
- கட்டமைக்கப்பட்ட தரவு மூலங்கள்
- கோப்பு அமைப்புகள்

### தூண்டுகோற்கள்

தூண்டுகோற்கள் பயனர்களுக்கான முன்கூறிய செய்திகள் மற்றும் பணிமுறைகளாகும்:

- முன்பிரிக்கப்பட்ட உரையாடல் வடிவங்கள்
- வழிகாட்டப்பட்ட தொடர்பாடல் முறைபாடுகள்
- சிறப்பு உரையாடல் கட்டமைப்புகள்

### கருவிகள்

கருவிகள் AI மாதிரி இயக்கு வேண்டிய செயல்பாடுகளாகும்:

- தரவு செயலாக்கக் கருவிகள்
- வெளி API ஒருங்கிணைப்புகள்
- கணக்கீட்டு திறன்கள்
- தேடல் செயல்பாட்டு திறன்

## மாதிரி நிறைவேற்றங்கள்: C# நிறைவேற்றம்

அதிகாரப்பூர்வ C# SDK இருப்பகம் MCP-ன் வேவ்வேறு அம்சங்களை காட்டும் சில மாதிரி நிறைவேற்றங்களை கொண்டுள்ளது:

- **அடிப்படை MCP கிளையன்ட்**: MCP கிளையன்டை உருவாக்கி கருவிகளை அழைக்கும் எளிய உதாரணம்
- **அடிப்படை MCP சேவையகம்**: அடிப்படை கருவி பதிவு கொண்ட குறைந்தபட்ச சேவையகம் நிறைவேற்றம்
- **மேம்பட்ட MCP சேவையகம்**: கருவி பதிவு, அங்கீகாரம் மற்றும் பிழை கையாளுதலுடன் முழுமையான சேவையகம்
- **ASP.NET ஒருங்கிணைவு**: ASP.NET கோர் உடன் ஒருங்கிணைப்பை காட்டும் உதாரணங்கள்
- **கருவி நிறைவேற்ற முறைபாடுகள்**: வெவ்வேறு சிக்கல் நிலைகளுக்கான கருவிகளை நிறைவேற்ற பல முறைப்பாடுகள்

MCP C# SDK முன்னோட்ட நிலையில் உள்ளது மற்றும் APIகள் மாற்றப்படக்கூடும். SDK வளர்ச்சியுடன் இந்த வலைப்பதிவை தொடர்ந்து புதுப்பிப்போம்.

- பிரபலமான இயந்திர கற்பித்தல் நூலகங்களுடன் இயல்புநிலை ஒருங்கிணைவு

முழு Python நிறைவேற்ற மாதிரி க்காக, மாதிரி கோப்புறையிலுள்ள [Python மாதிரி](samples/python/README.md) ஐ பார்வையிடவும்.

## API மேலாண்மை

Azure API மேலாண்மை MCP சேவைகளை எவ்வாறு பாதுகாப்பது என நல்ல பதிலாகும். உங்கள் MCP சேவையக 앞் ஒரு Azure API மேலாண்மை உதவிக்கூடிய இணைப்பை வைத்திருப்பது, பின்வரும் போன்ற வசதிகளை கையாளச் செய்வது:

- வீத வரம்பு விதிப்பது
- டோக்கன் மேலாண்மை
- கண்காணிப்பு
- சுமை சமச்சீர்
- பாதுகாப்பு

### Azure மாதிரி

## Azure இல் Remote MCP சேவையக பரப்பு

இப்போதும் முன் கூறிய மாதிரியை நாம் பரப்ப முடியுமா பார்ப்போம்:

1. பாதுகாப்பு கிடைக்கவும்

    ```bash
    git clone https://github.com/Azure-Samples/remote-mcp-apim-functions-python.git
    cd remote-mcp-apim-functions-python
    ```

1. `Microsoft.App` வள வழங்குநரை பதிவு செய்யவும்.

   - Azure CLI பயன்படுத்துவோர், `az provider register --namespace Microsoft.App --wait` என்று ஓட்டவும்.
   - Azure PowerShell பயன்படுத்துவோர், `Register-AzResourceProvider -ProviderNamespace Microsoft.App` ஓட்டவும். பின்னர் சில நேரம் கழித்து `(Get-AzResourceProvider -ProviderNamespace Microsoft.App).RegistrationState` ஓட்டி பதிவு நிறைவு ஆனதா என்று பாருங்கள்.

1. இந்த [azd](https://aka.ms/azd) கட்டளை மூலம் API மேலாண்மை சேவை, செயற்பாட்டு செயலி (குறியீடு உடன்) மற்றும் மற்ற அனைத்து Azure வளங்களை ஏற்படுத்தவும்

    ```shell
    azd up
    ```

    இந்த கட்டளைகள் Azure மேக வளங்களைக் 모두 நிறுவும்.

### MCP இன்ஸ்பெக்டர் மூலம் சேவையகத்தை சோதனை செய்தல்

1. புதிய **விண்ணப்ப சாளரத்தில்**, MCP இன்பெக்டரை நிறுவி இயக்கவும்

    ```shell
    npx @modelcontextprotocol/inspector
    ```

    நீங்கள் இதுபோன்ற இடைமுகத்தை காண்பீர்கள்:

    ![Connect to Node inspector](../../../translated_images/ta/connect.141db0b2bd05f096.webp)

1. MCP இன்ஸ்பெக்டர் வலை செயலியை செயலி வழங்கும் URL-இல் (எ.கா. [http://127.0.0.1:6274/#resources](http://127.0.0.1:6274/#resources)) CTRL கிளிக் செய்து திறக்கவும் 
1. கடத்தல் வகையை `SSE` ஆக அமைக்கவும் 
1. `azd up` போல இயக்கப்பட்ட உங்கள் API மேலாண்மை SSE நிறைவுத் தள முகவரியை URL ஆக அமைத்து **Connect** செய்க:

    ```shell
    https://<apim-servicename-from-azd-output>.azure-api.net/mcp/sse
    ```

1. **கருவி பட்டியல்**: கருவியை கிளிக் செய்து **கருவியை இயக்கவும்**.

எல்லா படிகளும் வெற்றி பெறின், நீங்கள் இப்போது MCP சேவையகத்துடன் இணைக்கப்பட்டுள்ளீர்கள் மற்றும் ஒரு கருவியை அழைக்க முடிந்துள்ளது.

## Azure க்கான MCP சேவைகள்

[Remote-mcp-functions](https://github.com/Azure-Samples/remote-mcp-functions-dotnet): இந்த இருப்பகங்கள் Python, C# .NET அல்லது Node/TypeScript உடன் Azure Functions பயன்படுத்தி தனிப்பயன் MCP (Model Context Protocol) சேவைகளை விரைவில் தொடக்க உருவாக்கவும் பரப்பவும் உதவும் வடிவமைப்புப் மாதிரிகள்.

இந்த மாதிரிகள் மேம்படுத்துநர்களுக்கு முழுமையான தீர்வை வழங்குகிறது, இது:

- உள்ளகமாக உருவாக்கி இயக்குதல்: உள்ளூர் கணினியில் MCP சேவையகத்தை உருவாக்கி டிபக் செய்ய
- Azure க்கு பரப்புதல்: `azd up` எளிய கட்டளையால் மேகத்திற்கு பரப்ப
- கிளையன்ட்களிலிருந்து இணைப்பு: VS Code இன் Copilot முகவரி முறை மற்றும் MCP இன்ஸ்பெக்டர் கருவி உட்பட பல்வேறு கிளையன்ட்களிலிருந்து சேவையகத்துடன் இணையும்

### முக்கிய அம்சங்கள்

- வடிவமைப்பின் பாதுகாப்பு: MCP சேவையகம் விசைகளும் HTTPS-இலும் பாதுகாக்கப்பட்டுள்ளது
- அங்கீகார விருப்பங்கள்: உள்ளமைந்த அங்கீகாரம் மற்றும்/அல்லது API மேலாண்மை மூலம் OAuth ஆதரவு
- நெட்வொர்க் தனிமைப்படுத்தல்: Azure Virtual Networks (VNET) மூலம் நெட்வொர்க் தனிமைப்படுத்தலை அனுமதிக்கிறது
- சேவையில்லா கட்டமைப்பு: Azure Functions மூலம் அளவுக்கு ஏற்ப, நிகழ்வு சார்ந்த செயல்பாடு
- உள்ளக வளர்ச்சி: விரிவான உள்ளக வளர்ச்சி மற்றும் டிபக் ஆதரவு
- எளிதான பரப்பு: Azureக்கு நேர்த்தியான பரப்பும் செயல்முறை

இறுதியில் MCP சேவையக உற்பத்தி-தயாராக விரைவாக தொடங்க தேவையான அனைத்து கட்டமைப்பு கோப்புகள், மூலக் குறியீடு மற்றும் வளச் வரையறைகளையும் கொண்டுள்ளது.

- [Azure Remote MCP Functions Python](https://github.com/Azure-Samples/remote-mcp-functions-python) - Python கொண்டு Azure Functions பயன்படுத்தி MCP மாதிரி நிறைவேற்றம்

- [Azure Remote MCP Functions .NET](https://github.com/Azure-Samples/remote-mcp-functions-dotnet) - C# .NET கொண்டு Azure Functions பயன்படுத்தி MCP மாதிரி நிறைவேற்றம்

- [Azure Remote MCP Functions Node/Typescript](https://github.com/Azure-Samples/remote-mcp-functions-typescript) - Node/TypeScript கொண்டு Azure Functions பயன்படுத்தி MCP மாதிரி நிறைவேற்றம்

## முக்கிய எடுத்துக்காட்டுகள்

- MCP SDKகள் தமக்கு உரிய மொழிகளுக்கு வலுவான MCP தீர்வுகளை உருவாக்க கருவிகளை வழங்குகிறது
- டிபக்கும் சோதனையும் MCP செயலிகளுக்குப் நம்பகத்தன்மைக்கு மிக அவசியம்
- மீண்டும் பயன்படுத்தக்கூடிய தூண்டுகோறு வடிவமைப்புகள் சீரான AI தொடர்புகளை உதவுகிறது
- நன்றாக வடிவமைக்கப்பட்ட பணிமுறைகள் பல கருவிகளை பயன்படுத்தி சிக்கலான பணிகளை ஒருங்கிணைக்க முடியும்
- MCP தீர்வுகளை நிறைவேற்ற வழியாக பாதுகாப்பு, செயல்திறன் மற்றும் பிழை கையாளுதலை கவனிக்க வேண்டும்
