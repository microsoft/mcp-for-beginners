# 🚀 மாடல் கான்டெக்ஸ் ப்ரோட்டோகால் (MCP) முதன்மை பயிற்சி திட்டம்

## **C#, Java, JavaScript, Rust, Python மற்றும் TypeScript இல் கையொப்ப உதாரணங்களுடன் MCP கற்றுக்கொள்ளுங்கள்**

## 🧠 மாடல் கான்டெக்ஸ் ப்ரோட்டோகால் பாடத்திட்டத்தின் ஒவ்வொன்றும்
மாடல் கான்டெக்ஸ் ப்ரோட்டோகால் பயணத்திற்கு உங்களை வரவேற்கிறோம்! நீங்கள் எப்படிக் AI பயன்பாடுகள் பல்வேறு கருவிகள் மற்றும் சேவைகளுடன் தொடர்பு கொள்கின்றன என்று ஏதாவது சந்தேகமாக இருந்தால், இங்கே அதன் அழகான தீர்வு உங்களை காத்திருக்கிறது, இது எப்படி டெவலப்பர்கள் புத்திசாலி அமைப்புகளை உருவாக்குகின்றனர் என்பதை மாற்றி அமைக்கிறது.

MCP என்பதை AI பயன்பாடுகளுக்கான ஒருமையான மொழிபெயர்ப்பாளராக நினைக்கலாம் - உங்களின் கணினியில் எந்த சாதனத்தையும் இணைக்க USB போர்ட்கள் உதவுகிறதைப் போலவே, MCP AI மாடல்களை எந்த கருவி அல்லது சேவையுடன் இருந்தாலும் ஒரு நிலையான முறையில் இணைக்க உதவுகிறது. நீங்கள் உங்கள் முதலில் பேசும் பாட்டன் பிளாட்பாம் உருவாக்கினாலும் அல்லது சிக்கலான AI வேலைதிட்டங்களில் செயல்படினாலும், MCP பற்றி புரிந்துகொள்வது உங்கள் பயன்பாடுகளை அதிக சக்தி வாய்ந்ததும் வளைவுள்ளதும் ஆக உருவாக்குவதற்கு சக்தியளிக்கும்.

இந்த பாடத்திட்டம் உங்கள் கற்றல் பயணத்திற்கு பொறுமையுடன் மற்றும் கவனத்துடன் வடிவமைக்கப்பட்டுள்ளது. நாம் உங்களுக்குத் தெரிந்த எளிய கருத்துக்களிலிருந்து தொடங்கி, உங்கள் விருப்பமான நிரலாக்க மொழியில் கையொப்ப முறைபாடுகளை தனிப்படுத்தியாக பயிற்சி மூலம் மனம் வைக்கிறோம். ஒவ்வொரு படியும் தெளிவான விளக்கங்களும், பயன்பாட்டு உதாரணங்களும், மற்றும் பல்வேறு ஊக்கங்களுடனும் கூடியதாக இருக்கும்.

இந்த பயணத்தை முடிக்கும்போது, உங்கள் சொந்த MCP சேவையகங்களை உருவாக்கும் திறன், புகழ்பெற்ற AI தளங்களுடன் இணைக்கும் அறிவு, மற்றும் இந்த தொழில்நுட்பம் AI மேம்பாட்டின் எதிர்காலத்தை எவ்வாறு மறுசீரமைக்கிறது என்பதையும் புரிந்துகொள்ளும் மனநிலை உண்டாகும். இந்த வாய் ஆர்வமுள்ள பயணத்தை நாம் ஒன்றாகத் தொடங்குவோம்!

### அதிகாரப்பூர்வ ஆவணங்கள் மற்றும் குறிப்புகள்

இந்த வளங்கள் உங்களுடைய புரிதல் வளரும் பொழுதில் மேலும் முக்கியமாக மாறும், ஆனால் உடனடியாக ஒவ்வொன்றையும் படிக்குமாறு அழுத்தாதீர்கள். முதலில் உங்களுக்கு மிகவும் ஆர்வமுள்ள பகுதிகளிலிருந்து துவங்குங்கள்!

## கற்றல் குறிக்கோள்கள்

இந்த பாடத்திட்டத்தை முடிக்கும் போது, நீங்கள் உங்கள் புதிய திறன்களில் நம்பிக்கை மற்றும் உற்சாகம் கொண்டிருப்பீர்கள். இங்கே நீங்கள் அடையும் திறன்கள்:

• **MCP அடிப்படைகளை புரிந்து கொள்ளுதல்**: மாடல் கான்டெக்ஸ் ப்ரோட்டோகால் என்றால் என்ன என்பதையும், அது எப்படித் தந்திரத்துடன் AI பயன்பாடுகள் ஒருங்கிணைக்கப்படுகிறன என புரிந்துகொள்ள, விளக்கங்கள் மற்றும் உதாரணங்களுடன் கற்றுக்கொள்வீர்கள்.

• **முதலில் MCP சேவையகத்தை உருவாக்குதல்**: நீங்கள் உங்கள் விருப்பமான நிரலாக்க மொழியில் செயல்படும் MCP சேவையகத்தை உருவாக்கி கற்றுக்கொள்வீர்கள், எளிய உதாரணங்களிலிருந்து பயிற்சி செய்து திறன்களை விரிவாக்கிக் கொள்வீர்கள்.

• **AI மாதிரிகளை உண்மையான கருவிகளுடன் இணைத்தல்**: AI மாதிரிகளும் உண்மையான சேவைகளும் ஒருங்கிணைக்கப் பெறுவது எப்படி என்று அறிந்து, உங்கள் பயன்பாடுகளை வலிமையான புதிய திறன்களுடன் கூடியதாக மாற்றுவீர்கள்.

## அடிப்படைக் கருவிகள்

MCP விஷயங்களில் மூழ்கும் முன், சில அடிப்படை கருத்துக்களில் அதி நம்பகமாக உணர்ந்துகொள்ளவும். நீங்கள் இவை பற்றியில் வல்லுனர் இல்லாவிட்டாலும் கவலை கொள்ள வேண்டாம் - தேவையான அனைத்தையும் நாம் விளக்கிக் கொடுக்கப்போகிறோம்!

### ப்ரோட்டோகால்களைப் புரிந்துகொள் (அடித்தளம்)

ப்ரோட்டோகால் என்பது உரையாடல் விதிமுறைகள் மாதிரியானது என்று நினைத்துக் கொள்ளுங்கள். நண்பரை அழைக்கும் போது, இருவரும் "வணக்கம்" என்று சொல்ல வேண்டும், பேசும் நேரம் ஒழுங்காக பகிர்ந்துகொள்ள வேண்டும், முடிவில் "பிரியாவிட" நம்பிக்கை வேணும். கணினி நிரல்கள் திறம்பட தொடர்பு கொள்ள இது போன்ற விதிமுறைகள் தேவைப்படும்.

MCP என்பது ஒரு ப்ரோட்டோகால் - AI மாதிரிகள் மற்றும் பயன்பாடுகள் கருவிகளுடன் மற்றும் சேவைகளுடன் பயனுள்ள "உரையாடல்களை" மேற்கொள்ள ஒப்புக்கொண்ட விதிமுறைகளின் தொகுப்பு. மனித உரையாடல் விதிமுறைகள் தொடர்பை மென்மையாக்குவது போலவே, MCP AI பயன்பாடுகளுக்கு உறுதியான மற்றும் சக்திவாய்ந்த தொடர்பை வழங்குகிறது.

### கிளையன் சேவையக உறவு (நிரல்கள் ஒன்றாக செயல்படுவது)

நீங்கள் தினமும் கிளையன்-சேவையக உறவுகளைப் பயன்படுத்துகிறீர்கள்! இணைய உலாவி (கிளையன்) மூலம் ஒரு இணையதளத்துக்கு செல்வது போல், நீங்கள் அப்பக்கத்தில் உள்ள தகவலை அனுப்பலாம். உலாவி கேட்கும் முறையை அறிந்து கொண்டுள்ளது, சேவையகம் பதில் அளிக்கும் முறையை அறிவது போன்றது.

MCP-இல் இதே மாதிரியான மாற்று உறவு உள்ளது: AI மாதிரிகள் கிளையனாக செயல்பட்டு தகவல் அல்லது செயல் வேண்டுகிறார்கள், MCP சேவையகங்கள் அதற்கான திறன்களை வழங்குகின்றன. இது உதவியாளர் (சேவையகம்) போல ஆகும், AI குறிப்பிட்ட பணிகளை செய்யவே கேட்கும் இடமாக.

### நிலையானதாக செய்வது ஏன் முக்கியம் (ஒன்றாக செயல்படுவதே)

ஒவ்வொரு கார் உற்பத்தியாளரும் வேறுபட்ட வடிவுடைய எரிபொருள் குழாய்களைப் பயன்படுத்தினால் யாரும் ஒரு ஒன்றும் பொருந்தாத அடாப்டர் தேவைப்படும்! ஒரே மாதிரியான முறைக்கு ஏற்ப ஒப்புக்கொள்வது வெகுவாக வாய்ப்பு தரும்.

MCP இது AI பயன்பாடுகளுக்கு ஒரு நிலையான வழியைக் கொடுக்கிறது. ஒவ்வொரு AI மாதிரிக்கும் தனிப்பட்ட குறியீடு எழுதுவதற்கு பதிலாக, MCP அவற்றுக்கு ஒன்றாக தொடர்பு கொள்ளும் வழியை உருவாக்குகிறது. இதனால் டெவலப்பர்களுக்கு ஒருமுறை கருவிகள் உருவாக்கி பல AI அமைப்புகளைச் சேர்ந்த பயன்பாடுகளில் பயன்படுத்தக் கூடியது.

## 🧭 உங்கள் கற்றல் பாதை அறிமுகம்

### 🌱 அடித்தளம் கட்டம்: அடிப்படைகளை புரிந்து கொள்வது (அலகுகள் 0-2)

இங்கே உங்கள் சாகசம் தொடங்குகிறது! MCP கருத்துக்களை பரிச்சயமான ஒப்புமைகளும் எளிய உதாரணங்களும் கொண்டு விளக்கியிருப்போம். MCP என்றால் என்ன, இது ஏன் அவசியம், மற்றும் இது AI மேம்பாட்டின் பெரிய உலகத்தில் எவ்வாறு பொருந்துகிறது என்பதைக் புரிந்துகொள்வீர்கள்.

• **அலகு 2 - MCP இல் பாதுகாப்பு**: பாதுகாப்பு கடுமையாக தோன்றலாம், ஆனால் MCP என்பது உள்ளமைக்கப்பட்ட பாதுகாப்பு அம்சங்களுடன் வருகிறது, மேலும் உங்கள் பயன்பாடுகளுக்கு பக்குவமாய் பாதுகாப்பு வழங்கும் சிறந்த நடைமுறைகளை கற்றுக் கொள்வோம்.

### 🔨 கட்டும் கட்டம்: உங்கள் முதற் MCP செயல்பாடுகளை உருவாக்குதல் (அலகு 3)

### 🌟 திறமை நிலை: சமூகமும் சிறப்புப்பாடும் (தொகுதிகள் 6-11)

தொகுதி 11 சிறப்பு குறிப்பிடத்தக்கது - இது முழுமையான 13-பயிற்சி கைமுறைகளை கொண்ட கற்கும் பாதை ஆகும், இது PostgreSQL ஒருங்கிணைப்புடன் உற்பத்தி தயார் MCP சர்வர்களை கட்டமைக்க கற்றுச் செல்லும். இது நீங்கள் கற்ற அனைத்தையும் ஒன்றிணைக்கும் வாய்ப்பு போல உள்ளது!

## 📚 படிப்புத் தளவமைப்பு மற்றும் வளங்கள்  

இந்த உள்நுழைவிடம் நீங்கள் வழிமுறைக் கொள்கைகளை அனுசரித்து வேலை செய்யவும் கற்றுக்கொள்ளவும் பல வளங்களை உள்ளடக்கியது:  

### படிப்புத் தளவமைப்பு  

ஒரு விரிவான [படிப்புத் தளவமைப்பு](./study_guide.md) உள்நுழைவிடம் உள்ளது, இது நீங்கள் இந்த உள்நுழைவிடம் முழுமையாக வழிமொழிய உதவுகிறது. இந்த காட்சி பாடத்திட்ட வரைபடம் அனைத்து தலைப்புகளும் எப்படி தொடர்புடையவை என்பதை காட்டுகிறது மற்றும் மாதிரிப் திட்டங்களை எவ்வாறு பயன்படுத்துவது என்று வழிகாட்டுகிறது. நீங்கள் ஒரு காட்சி கற்றலர் என்றால் இது மிகவும் உதவும், பெரிய படத்தை பார்க்க விரும்புபவர்களுக்கு சிறந்தது.  

இந்த வழிகாட்டி கொண்டுள்ளது:  
- அனைத்து தலைப்புகளையும் காட்டும் காட்சி பாடத்திட்ட வரைபடம்  
- ஒவ்வொரு உள்நுழைவிடம் பகுதியின் விரிவான உட்சேதங்கள்  
- மாதிரிப் திட்டங்களை எவ்வாறு பயன்படுத்துவது என்று வழிகாட்டல்  
- வெவ்வேறு திறன் நிலைகளுக்கான பரிந்துரைக்கப்பட்ட கற்றல் பாதைகள்  
- உங்கள் கற்றல் பயணத்தை மேம்படுத்த கூடுதல் வளங்கள்

### தொடங்க தயாரா?  

உங்கள் MCP பயணம் இப்போதே துவங்குகிறது! முதலில் செயல்பாட்டிற்கான MCP அனுபவங்களுக்காக Module 0 ஐத் தொடங்கவும், அல்லது நீங்கள் என்ன கட்டியமைப்பீர்கள் என்பதைப் பார்க்க மாதிரிப் திட்டங்களை ஆராயவும். நினைவில் கொள்ளுங்கள் - ஒவ்வொரு நிபுணரும் இப்போல் தான் துவங்கியவர்கள், பொறுமை மற்றும் பயிற்சியோடு நீங்கள் சாதனை காண்பீர்கள்.  

Model Context Protocol வளர்ச்சியின் உலகிற்கு வரவேற்பு. ஒன்றாக அருமையாக ஒன்றை கட்டுவோம்!  

## 🤝 கற்றல் சமூகத்திற்கு பங்களித்தல்  

இந்த பாடத்திட்டம் உங்களைப் போன்ற கற்றலர்களின் பங்களிப்புகளால் வலுவடைகிறது! நீங்கள் ஒரு உள்ளீட்டு பிழையை திருத்தினாலும், தெளிவான விளக்கத்தை பரிந்துரைத்தாலும், அல்லது புதிய உதாரணத்தைச் சேர்த்தாலும் உங்கள் பங்களிப்பு ஆரம்பக்காரர்களுக்கு வெற்றியடையும் உதவியாக இருக்கும்.  

Microsoft மதிப்பிடப்பட்ட நிபுணர் [Shivam Goyal](https://www.linkedin.com/in/shivam2003/) அவர்களுக்கு குறியீடு உதாரணங்களுக்குப் பங்களித்ததற்கும் நன்றி.  

பங்களிப்பு செயல்முறை வரவேற்கத்தக்கதும் ஆதரவானதுமானதுதான். பெரும்பாலான பங்களிப்புகளுக்கு Contributor License Agreement (CLA) தேவை, ஆனால் தானியங்கி கருவிகள் உங்களை சீராக வழிநடத்தும்.

## 🤝 பங்களிப்பு வழிகாட்டல்கள்  

இந்த திட்டம் பங்களிப்புகள் மற்றும் பரிந்துரைகளை வரவேற்கிறது. பெரும்பாலான பங்களிப்புகளுக்கு நீங்கள் உங்கள் பங்களிப்புக்கு உரிமை உண்டு என்றும் அதனை நமக்கு வழங்குகிறீர்கள் என்றும் உறுதி செய்ய CLA ஒப்பந்தம் தேவை. விரிவுகளுக்கு: <https://cla.opensource.microsoft.com>.  

நீங்கள் ஒரு pull request சமர்ப்பிக்கும் போது, CLA பாட்டம் உங்கள் CLA தேவையை தானாக தீர்மானித்து போர்வைப் பெறும் (எ.கா., நிலை சரிபார்ப்பு, கருத்து). பாட்டின் வழிகாட்டுதல்களை பின் தொடர்ந்து செய்யுங்கள். ஒரு முறை மட்டும் இந்த நடவடிக்கையை செய்ய வேண்டியிருக்கும்.  

இந்தத் திட்டம் [Microsoft திறந்த மூல நடத்தைக் குறியீட்டை](https://opensource.microsoft.com/codeofconduct/) ஏற்றுக்கொண்டுள்ளது. கூடுதல் தகவலுக்கு [நடத்தைக் குறியீட்டு FAQ](https://opensource.microsoft.com/codeofconduct/faq/) பார்வையிடவும் அல்லது [opencode@microsoft.com](mailto:opencode@microsoft.com) தொடர்புகொள்ளவும்.
