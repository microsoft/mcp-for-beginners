# 🚀 ஆரம்பக்காரர்களுக்கான மாதிரி சூழல் நெறிமுறை (MCP) பாடத்திட்டம்

## **C#, Java, JavaScript, Rust, Python மற்றும் TypeScript இல் கைமுறை குறியீடு உதாரணங்களுடன் MCP கற்றுக்கொள்ளுங்கள்**

## 🧠 மாதிரி சூழல் நெறிமுறை பாடத்திட்டத்தின் ஒட்டுமொத்த நோக்கு
மாதிரி சூழல் நெறிமுறையின் பயணத்தில் உங்களை அன்புடன் வரவேற்கிறோம்! AI செயலிகள் வெவ்வேறு கருவிகள் மற்றும் சேவைகளுடன் எவ்வாறு தொடர்பு கொள்கின்றன என்று நீங்கள் யோசித்திருந்தால், இப்போது நீங்கள் புரிந்துகொள்ளப் போகிறீர்கள், இது தொழிலாளர்களுக்கு நுண்ணறிவு ஆன அமைப்புகளை உருவாக்கும் முறையை மாற்றி அமைக்கிறது.

MCP ஐ அனைத்து AI செயலிகளுக்கும் பொதுவான மொழிபெயர்ப்பாளராக நினைக்கவும் — USB போர்ட்கள் உங்கள் கணினியில் எந்த சாதனத்தையும் இணைக்க அனுமதிப்பது போல், MCP AI மாடல்களுக்கு எந்த கருவி அல்லது சேவையையும் ஒரே முறை முறையில் இணைக்கிறது. உங்கள் முதல் chatbot ஐ உருவாக்கிக் கொண்டிருந்தாலோ அல்லது சிக்கலான AI வேலைநிரலை உருவாக்கி கொண்டிருந்தாலோ MCP பற்றி அறிந்துக்கொள்வது, நீங்கள் திறமை வாய்ந்த மற்றும் நெகிழ்வான செயலிகளை உருவாக்க சக்தியை வழங்கும்.

உங்கள் கற்றல் பயணத்திற்கு இந்த பாடத்திட்டம் கவனமாக அமைக்கப்பட்டுள்ளது. நாம் நீங்கள் ஏற்கனவே அறிவதும் கீழ் இருந்து ஆரம்பிக்கின்றோம், மேலும் உங்கள் விருப்பமான நிரலாக்க மொழியில் நடைமுறை செயல்முறை மூலம் உங்கள் திறனை அடி அடி அதிகரிக்கிறோம். ஒவ்வொரு படியும் தெளிவான விளக்கங்கள், நடைமுறை உதாரணங்கள், மற்றும் நிறைந்த ஊக்கத்துடன் கொண்டு செல்லப்படும்.

இந்த பயணத்தை முடிக்கும் போது, உங்கள் சொந்த MCP சேவையகங்களை உருவாக்குவதற்கும், அவற்றை பிரபலமான AI மையங்களுடன் இணைப்பதற்கும் மற்றும் இந்த தொழில்நுட்பம் AI மேம்பாட்டின் எதிர்காலத்தை எப்படி மாற்றிக் கொண்டு இருக்கிறது என்பதை புரிந்துகொள்வதற்கும் தைரியமாக இருப்பீர்கள். இந்த மாபெரும் பயணத்தை ஒன்றாகத் துவங்குவோம்!

## அவசியமான பின்னணி

MCP நுட்பங்களை முன்னதாக அறியும்வரை, சில அடிப்படை கொள்கைகள் மீதான நம்பகத்தைப் பெற்றுக்கொள்வோம். நீங்கள் இதில் நிபுணர் இல்லையென்றால் கவலைப்பட வேண்டாம் - நாங்கள் தேவைப்பட்ட எல்லாவற்றையும் விளக்குவோம்!

### நெறிமுறைகளை புரிந்துகொள்ளுதல் (அடித்தளம்)

ஒரு உரையாடலுக்கான விதிமுறைகள் போன்றவை நெறிமுறை. நீங்கள் நண்பரை அழைக்கும் போது நாம் எப்படி "வணக்கம்" என்று பேசுவோம், இருவரும் பேசும் முறை மற்றும் "போ" என்று சொல்வது போன்றவை இருக்கும். கணினித் திட்டங்களும் இதுபோன்று விதிமுறைகளை அறிந்து பேச்சுவார்த்தை நடத்த வேண்டும்.

MCP என்பது நெறிமுறை - அது AI மாடல்கள் மற்றும் செயலிகள் கருவிகளுடனும் சேவைகளுடனும் பயனுள்ள உரையாடல்களை நடத்த உதவும் ஒப்பந்த விதிகள் தொகுப்பு. பேசும் விதிமுறைகள் மனிதர்களுக்குச் சீரான தொடர்பை ஏற்படுத்துவது போல், MCP AI செயலிகள் இடையேயான சரியான, திறமையான தொடர்பை வழங்குகிறது.

### கிளையண்ட்-சேவையகம் உறவுகள் (திட்டங்கள் எப்படி ஒன்றாக வேலை செய்கின்றன)

நீங்கள் தினமும் கிளையண்ட்-சேவையகம் உறவுகளைப் பயன்படுத்துகிறீர்கள்! ஒரு இணைய உலாவியை (கிளையண்ட்) பயன்படுத்தி ஒரு இணையதளத்தைப் பார்க்கும்போது, அது அந்த இணையதள சேவையகத்துடன் இணைக்கப்படுகிறது மற்றும் பக்க உள்ளடக்கத்தை பெறுகிறது. உலாவி தகவல்களை கேட்கும் முறையை தெரிந்துகொள்கிறது, சேவையகம் எப்படி பதில் கொடுக்குமென்று அறிந்துள்ளது.

MCP இல் இதே மாதிரி உறவு உள்ளது: AI மாடல்கள் கேள்விகள் கேட்டு செயல் கோருகிற கிளையண்ட்களாக செயற்படுகின்றன, MCP சேவையகங்கள் அந்த திறன்களை வழங்குகின்றன. இது உதவியாளராக (சேவையகம்) இருப்பது போல், AI கேள்விகளை நிறைவேற்றுவதற்கு உதவுகிறது.

## 📚 படிப்புத் தொகுப்பு மற்றும் வளங்கள்

இந்த சேமிப்பகம் உங்களுக்கு வழிகாட்டி மற்றும் பயிற்சியைச் செய்ய பல வளங்களை கொண்டுள்ளது:

### படிப்புத் தொகுப்பு

இல்லாட்சிய [படிப்புத் தொகுப்பு](./study_guide.md) இந்த சேமிப்பகத்தை விளங்க பயனுள்ளதாக உள்ளது. இந்த காட்சிப்படுத்தப்படும் பாடத்திட்ட வரைபடம் அனைத்து தலைப்புகளும் எப்படி இணைக்கப்பட்டுள்ளதென்றும் உதவி அளிக்கிறது மற்றும் மாதிரிக் கட்டமைப்புகளை எப்படிப் பயன்படுத்துவது என்று வழிகாட்டுகிறது. பெரிய படத்தை காண விரும்பும் காட்சித் திறன் கொண்டவர்களுக்கு இது சிறப்பாக உதவும்.

இந்த வழிகாட்டி உள்ளடக்கம்:
- அனைத்து தலைப்புகளையும் காட்டும் காட்சிப்படுத்தப்பட்ட பாடத்திட்ட வரைபடம்
- ஒவ்வொரு சேமிப்பக பகுதியின் விரிவான பகுப்பாய்வு
- மாதிரி திட்டங்களை எவ்வாறு பயன்படுத்துவது என்ற வழிகாட்டி
- விதவிதமான திறமை நிலைகளுக்கான பரிந்துரைக்கப்பட்ட கற்றல் பாதைகள்
- உங்கள் கற்றலை இணைக்கும் கூடுதல் வளங்கள்

### மாற்றக் குறிப்புகள்

நாங்கள் பாடத்திட்டப் பொருட்களின் அனைத்து முக்கிய புதுப்பிப்புகளையும் கண்காணிக்கும் விரிவான [மாற்றக் குறிப்புகள்](./changelog.md) வைத்திருக்கிறோம், அதனால் உடனடி மேம்பாடுகளுடன் உங்களுக்கு இருப்பது உறுதி.
- புதிய உள்ளடக்க மேலோட்டங்கள்
- கட்டமைப்புச் சீர்மாற்றங்கள்
- அம்ச மேம்பாடுகள்
- ஆவண புதுப்பிப்புகள்

இந்த அறிமுகப்பாகம் மூலம், நீங்கள் MCP அறிவுக்கூறின் அடித்தளத்தை உருவாக்கத் தொடங்கியுள்ளீர்கள். ### எதிர்காலப் பயணம்

உங்கள் MCP பயணம் இப்போது துவங்குகிறது! உங்கள் முதல் MCP அனுபவங்களுக்கு நுழைய Module 0-ஐத் தொடங்கவும் அல்லது நீங்கள் உருவாக்கப்போகும் மாதிரி திட்டங்களை ஆராயவும். நினைவில் வைக்கவும் - ஒவ்வொரு நிபுணரும் தற்போது நீங்கள் உள்ள இடத்தில்தான் துவங்கியவர்கள், பொறுமையும் பயிற்சியும் மூலம் நீங்கள் சாதிக்க முடியாது என்று நினைத்ததை யோசிக்க முடியாமல் போவீர்கள்.

Model Context Protocol மேம்பாட்டின் உலகிற்கு வரவேற்கிறோம். ஒன்றாக ஒரு அற்புதமானதைக் கட்டுவோம்!
