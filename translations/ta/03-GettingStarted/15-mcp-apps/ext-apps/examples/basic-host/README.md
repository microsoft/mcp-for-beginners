# உதாரணம்: அடிப்படை ஹோஸ்ட்

MCP சர்வர்களுடன் இணைக்கவும், பாதுகாப்புடைய சென்ட்பாக்ஸில் கருவி UIகளை அமைக்கவும் எப்படி MCP ஹோஸ்ட் பயன்பாட்டை கட்டமைப்பதைக் காட்டும் ஒரு குறிப்பு செயலாக்கம்.

இந்த அடிப்படை ஹோஸ்ட்டை உள்ளூர் மேம்பாட்டின் போது MCP செயலிகளைக் காண்பதற்கும் பயன்படுத்தலாம்.

## முக்கிய கோப்புகள்

- [`index.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/index.html) / [`src/index.tsx`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/index.tsx) - கருவி தேர்வு, அளவுரு உள்ளீடு மற்றும் ஐபிரேம் மேலாண்மையுடன் React UI ஹோஸ்ட்
- [`sandbox.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/sandbox.html) / [`src/sandbox.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/sandbox.ts) - பாதுகாப்பு சரிபார்ப்பு மற்றும் இருமுக தகவல் பரிமாற்றத்துடன் வெளியே உள்ள ஐபிரேம் பிரதிநிதி
- [`src/implementation.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/implementation.ts) - முதன்மை தர்க்கம்: சர்வர் இணைப்பு, கருவி அழைப்பு மற்றும் AppBridge அமைப்பு

## தொடக்கமாக

```bash
npm install
npm run start
# http://localhost:8080 ஐ திறக்கவும்
```

பொதுவாக, ஹோஸ்ட் பயன்பாடு `http://localhost:3001/mcp` என்ற MCP சர்வருடன் இணைக்க முயறும். இந்த நடத்தை `SERVERS` சூழல் மாறியை சர்வர் URLகளின் JSON வரிசையாக அமைப்பதன் மூலம் மாற்ற முடியும்:

```bash
SERVERS='["http://localhost:1234/mcp", "http://localhost:5678/mcp"]' npm run start
```

## கட்டமைப்பு

பாதுகாப்பான UI தனியுரிமைக்காக இந்த உதாரணம் இரட்டை-ஐபிரேம் சென்ட்பாக்ஸ் முறைப்பாட்டைப் பயன்படுத்துகிறது:

```
Host (port 8080)
  └── Outer iframe (port 8081) - sandbox proxy
        └── Inner iframe (srcdoc) - untrusted tool UI
```

**ஐரு ஐபிரேம்கள் ஏன்?**

- வெளியே உள்ள ஐபிரேம் தனித்த ஒரிஜினில் (ஓட்டம் 8081) இயங்கி, ஹோஸ்ட்டிற்கு நேரடி அணுகலை தடுக்கும்
- உள்ளே உள்ள ஐபிரேம் `srcdoc` மூலம் HTML பெற்றுக்கொள்கிறது மற்றும் சென்ட்பாக்ஸ் பண்புகளால் கட்டுப்படுத்தப்படுகிறது
- செய்திகள் வெளியே உள்ள ஐபிரேமின் வழியே செல்லும், அது அவற்றை சரிபார்த்து இருமுகமாக பரிமாறுகிறது

இந்த கட்டமைப்பு, கருவி UI குறியீடு தீங்கு செய்யும் போது கூட, அது ஹோஸ்ட் பயன்பாட்டின் DOM, குக்கீஸ் அல்லது ஜாவாஸ்கிரிப்ட் சூழலை அணுக முடியாமல் உறுதி செய்கிறது.
