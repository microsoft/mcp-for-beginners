## தொடக்கம்  

[![உங்கள் முதல் MCP சர்வரை உருவாக்குங்கள்](../../../translated_images/ta/04.0ea920069efd979a.webp)](https://youtu.be/sNDZO9N4m9Y)

_(இந்த பாடத்தின் காணொளியை பார்வையிட மேலே உள்ள படத்தை கிளிக் செய்யவும்)_

இந்த பகுதி பல பாடங்களைக் கொண்டுள்ளது:

- **1 உங்கள் முதல் சர்வர்**, இந்த ஆரம்ப பாடத்தில், நீங்கள் உங்கள் முதல் சர்வரை உருவாக்கும் முறையையும், அதைப் பரிசோதக கருவியுடன் (inspector tool) ஆய்வு செய்வதையும் கற்கபோகிறீர்கள், இது உங்கள் சர்வரை சோதிக்கவும் பிழையுணர்த்தவும் மதிப்புமிக்க வழி ஆகும், [பாடத்திற்கு](01-first-server/README.md)

- **2 கிளையண்ட்**, இந்த பாடத்தில், உங்கள் சர்வருடன் இணைக்க கூடிய கிளையண்டை எவ்வாறு எழுதுவது என்பதை கற்கபோகிறீர்கள், [பாடத்திற்கு](02-client/README.md)

- **3 LLM உடன் கிளையண்ட்**, கிளையண்டை இன்னும் சிறப்பாக எழுதுவதற்கான மேலும் ஒரு வழி, அதில் LLM சேர்க்கப்படுகிறது, அது உங்கள் சர்வருடன் "பரிமாற்றம்" செய்ய உதவுகிறது, [பாடத்திற்கு](03-llm-client/README.md)

- **4 Visual Studio Code இல் GitHub Copilot Agent விதியில் சர்வர் பயன்படுத்தல்**. இங்கே, நாம் MCP சர்வரை Visual Studio Code உள்ளே இயங்க வலிமையுடன் பார்க்கிறோம், [பாடத்திற்கு](04-vscode/README.md)

- **5 stdio மடந்த சர்வர்** stdio மடந்தம் என்பது உள்ளூர் MCP சர்வர்-கிளையண்ட் தொடர்புக்கு பரிந்துரைக்கப்பட்ட தரநிலை ஆகும், பாதுகாப்பான subprocess அடிப்படையிலான தொடர்பு மற்றும் நிரல் முக்கிய தனி தன்மைகளை வழங்குகிறது [பாடத்திற்கு](05-stdio-server/README.md)

- **6 MCP உடன் HTTP ஸ்ட்ரீமிங் (Streamable HTTP)**. இந்தப் பாடத்தில் நவீன HTTP ஸ்ட்ரீமிங் மடந்தத்தை கற்று கொள்ளலாம் (தொலை MCP சர்வர்களுக்கு பரிந்துரைக்கபட்ட முறை [MCP கொள்கை 2025-11-25](https://spec.modelcontextprotocol.io/specification/2025-11-25/basic/transports/#streamable-http) படி), முன்னேற்ற அறிவிப்புகள், மற்றும் Streamable HTTP பயன்படுத்தி விரிவான, நேரடி MCP சர்வர் மற்றும் கிளையண்டுகளை எப்படி செயல்படுத்துவது என்பதையும் கற்கலாம். [பாடத்திற்கு](06-http-streaming/README.md)

- **7 VSCode க்கான AI கருவி தொகுப்பைப் பயன்படுத்தல்** MCP கிளையண்ட்கள் மற்றும் சர்வர்கள் சோதனை செய்ய மற்றும் பயன்படுத்த [பாடத்திற்கு](07-aitk/README.md)

- **8 சோதனை**. இங்கே நாம் குறிப்பாக எவ்வாறு நம்முடைய சர்வர் மற்றும் கிளையண்டை வித்தியாசமான முறைகளில் சோதனை செய்யலாம் என்பதை கவனிப்போம், [பாடத்திற்கு](08-testing/README.md)

- **9 பணியிடல்**. இந்த அத்தியாயம் உங்கள் MCP தீர்வுகளை பிரித்தெடுக்கும் பல வழிகளைப் பார்க்கும், [பாடத்திற்கு](09-deployment/README.md)

- **10 மேம்பட்ட சர்வர் பயன்பாடு**. இந்த அத்தியாயம் மேம்பட்ட சர்வர் பயன்பாட்டை உள்ளடக்கியது, [பாடத்திற்கு](./10-advanced/README.md)

- **11 அங்கீகாரம்**. இந்த அத்தியாயம் எளிய அங்கீகாரத்தை சேர்க்க எப்படி, Basic Auth முதல் JWT மற்றும் RBAC பயன்படுத்துதல் வரை கதைபடுகிறது. நீங்கள் இங்கே தொடங்க ஊக்கம் பெறுகிறீர்கள் பிறகு அத்தியாயம் 5ல் உள்ள மேம்பட்ட தலைப்புகளையும் அத்தியாயம் 2ல் பரிந்துரைக்கும் கூடுதல் பாதுகாப்பு உறுதிப்படுத்தல்களையும் பார்க்கவும், [பாடத்திற்கு](./11-simple-auth/README.md)

- **12 MCP ஹோஸ்ட்கள்**. Claude Desktop, Cursor, Cline மற்றும் Windsurf போன்ற பிரபல MCP ஹோஸ்ட் கிளையண்டுகளை கட்டமைக்கவும் பயன்படுத்தவும். மடந்த வகைகள் மற்றும் பிழைத்திருத்தத்தையும் கற்றுக்கொள்ளவும், [பாடத்திற்கு](./12-mcp-hosts/README.md)

- **13 MCP பரிசோதகர்**. MCP பரிசோதகர் கருவியை பயன்படுத்தி உங்கள் MCP சர்வர்களை தொடர்புகொண்டு பிழைத்திருத்தம் மற்றும் சோதனை செய்யவும். கருவிகள், வளங்கள் மற்றும் நெறிமுறை செய்திகள் பற்றி அறியவும், [பாடத்திற்கு](./13-mcp-inspector/README.md)

- **14 மாதிரிப்பெறு**. LLM சம்பந்தப்பட்ட பணி தொடர்பாக MCP கிளையண்ட்களுடன் இணைந்து செயல்படும் MCP சர்வர்களை உருவாக்கவும். [பாடத்திற்கு](./14-sampling/README.md)

- **15 MCP செயலிகள்**. MCP சர்வர்களை UI அறிவுரைகளுடன் பதிலளிக்கும் முறையில் உருவாக்கவும், [பாடத்திற்கு](./15-mcp-apps/README.md)

Model Context Protocol (MCP) என்பது செயலிகள் LLM களுக்கு context வழங்கும் முறையை ஒருங்கிணைக்கும் திறந்த நெறிமுறை ஆகும். MCP ஐ USB-C போர்ட்டாகக் கற்பனை செய்யலாம், இது AI செயலிக்களுக்கு பல்வேறு தரவுத் தளங்களுக்கும் கருவிகளுக்கும் இணைக்க ஒரு தரநிலை அமைப்பை வழங்குகிறது.

## கற்கும் நோக்கங்கள்

இந்த பாடம் பூர்த்தி செய்யும்போது நீங்கள்:

- C#, ஜாவா, பைதான், டைப்ஸ்கிரிப்ட் மற்றும் ஜாவாவிரிப்பில் MCP உருவாக்கப்படும் மேம்பாட்டு சூழலை அமைக்க தெரிந்து கொள்வீர்கள்
- தனிப்பட்ட அம்சங்கள் (வளங்கள், அறிவுறுத்தல்கள் மற்றும் கருவிகள்) கொண்ட அடிப்படையான MCP சர்வரை கட்டி பணியிட முடியும்
- MCP சர்வர்களுக்கு இணைக்க கூடிய ஹோஸ்ட் செயலிகளை உருவாக்க முடியும்
- MCP அடிப்படைகளை சோதனை செய்து பிழைத்திருத்தம் செய்யும் திறன் கற்றுக் கொள்வீர்கள்
- பொதுவான அமைவு சிக்கல்கள் மற்றும் அதற்கு தீர்வுகளை புரிந்துகொள்வீர்கள்
- MCP செயலிகளை பிரபலமான LLM சேவைகளுடன் இணைக்க முடியும்

## உங்கள் MCP சூழலை அமைத்தல்

MCP உடன் பணிபுரிய தொடங்கும் முன், உங்கள் மேம்பாட்டு சூழலை தயார் செய்யவும் அடிப்படையான பணிச்செயற்குழுவைப் புரிந்துகொள்ளவும் முக்கியம். இந்த பகுதி நீங்கள் MCP யுடன் மென்மையான தொடக்கத்தை உறுதி செய்ய ஆரம்ப அமைப்பு படிகளிலிருந்து வழிகாட்டும்.

### தேவைகள்

MCP உருவாக்குதலுக்கு முன்னர், கீழ்கண்டவை உறுதி செய்யவும்:

- **மேம்பாட்டு சூழல்**: தேர்ந்தெடுத்த மொழிக்கான (C#, Java, Python, TypeScript அல்லது JavaScript)
- **IDE/ஏடிட்டர்**: Visual Studio, Visual Studio Code, IntelliJ, Eclipse, PyCharm அல்லது எந்த முன்னேற்றிய குறியீட்டு தொகுப்பாளர்
- **பின்னணி மேலாளர்கள்**: NuGet, Maven/Gradle, pip, அல்லது npm/yarn
- **API விசைகள்**: உங்கள் ஹோஸ்ட் செயலிகளில் பயன்படுத்தும் AI சேவைகளுக்கானவை

### அதிகாரப்பூர்வ SDKக்கள்

இருக்கும் அத்தியாயங்களில் Python, TypeScript, Java மற்றும் .NET பயன்படுத்தி தீர்வுகள் உருவாக்கப்படும். இங்கு அனைத்து அதிகாரப்பூர்வமாக ஆதரிக்கப்படும் SDKக்கள் உள்ளன.

MCP பல மொழிகளுக்கான அதிகாரப்பூர்வ SDKக்களை வழங்குகிறது ([MCP கொள்கை 2025-11-25](https://spec.modelcontextprotocol.io/specification/2025-11-25/) உடன் இணக்கமானது):
- [C# SDK](https://github.com/modelcontextprotocol/csharp-sdk) - Microsoft உடன் ஒத்துழைத்து பராமரிக்கப்படுகிறது
- [ஜாவா SDK](https://github.com/modelcontextprotocol/java-sdk) - Spring AI உடன் ஒத்துழைத்து பராமரிக்கப்படுகிறது
- [டைப்ஸ்கிரிப்ட் SDK](https://github.com/modelcontextprotocol/typescript-sdk) - அதிகாரப்பூர்வ டைப் ஸ்கிரிப்ட் செயலாக்கம்
- [பைதான் SDK](https://github.com/modelcontextprotocol/python-sdk) - அதிகாரப்பூர்வ பைதான் செயலாக்கம் (FastMCP)
- [கொட்லின் SDK](https://github.com/modelcontextprotocol/kotlin-sdk) - அதிகாரப்பூர்வ கொட்லின் செயலாக்கம்
- [ஸ்விஃப்ட் SDK](https://github.com/modelcontextprotocol/swift-sdk) - Loopwork AI உடன் ஒத்துழைத்து பராமரிக்கப்படுகிறது
- [ரஸ்ட் SDK](https://github.com/modelcontextprotocol/rust-sdk) - அதிகாரப்பூர்வ ரஸ்ட் செயலாக்கம்
- [கோ SDK](https://github.com/modelcontextprotocol/go-sdk) - அதிகாரப்பூர்வ கோ செயலாக்கம்

## முக்கியக் குறிப்புகள்

- MCP மேம்பாட்டு சூழல் அமைவு மொழிச் சாலைகளால் எளிமையாக நடைபெறும்
- MCP சர்வர்களை உருவாக்கும்போது தெளிவான வடிவமைப்புகளுடன் கருவிகளை உருவாக்கி பதிவு செய்ய வேண்டும்
- MCP கிளையண்ட்கள் சர்வர்களுக்கும் மாதிரிகளுக்கும் இணைக்க இந்த விரிவாக்கப்பட்ட திறன்களை சுமந்துள்ளன
- நம்பகமான MCP செயலாக்கங்கள் க்காக சோதனை மற்றும் பிழைத்திருத்தம் அவசியம்
- பணியிடல் தேர்வுகள் உள்ளூர் மேம்பாட்டிலிருந்து மேக அடிப்படையிலான தீர்வுகளுக்கு பரவலாக உள்ளன
