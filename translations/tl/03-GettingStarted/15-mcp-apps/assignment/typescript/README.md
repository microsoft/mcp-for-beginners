Rock paper scissor sample

## Kung paano nakaayos ang solusyon na ito

Ang solusyon ay binubuo lamang ng bahagi ng code na mahalaga, ibig sabihin ay ang markup, event wire up at mga tampok ng server.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

## Paano patakbuhin

Tingnan ang [Code](../../code/typescript/README.md) at idagdag ang nilalaman mula sa mga file na ito sa kani-kanilang mga file.
