## Kako je ovo rješenje strukturirano

Rješenje se sastoji samo od dijela koda koji je bitan, tj. markup, povezivanje događaja i serverske značajke.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

## Kako pokrenuti

Pogledajte [Code](../../code/typescript/README.md) i dodajte sadržaj iz ovih datoteka u svaku odgovarajuću datoteku.
