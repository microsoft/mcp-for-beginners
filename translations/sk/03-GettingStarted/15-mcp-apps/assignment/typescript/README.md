## Ako je toto riešenie štruktúrované

Riešenie pozostáva iba z časti kódu, ktorá je podstatná, t.j. zo značkovania, prepojenia udalostí a serverových funkcií.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
