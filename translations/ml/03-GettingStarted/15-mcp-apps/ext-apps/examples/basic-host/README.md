# ഉദാഹരണം: അടിസ്ഥാന ഹോസ്റ്റ്

MCP സര്‍വറുകളുമായി സംയോജിപ്പിച്ച് സുരക്ഷിതമായ സാൻഡ്ബോക്സിൽ ടൂൾ UI കളെ അവതരിപ്പിക്കുന്ന MCP ഹോസ്റ്റ് അപ്ലിക്കേഷൻ നിർമ്മിക്കുന്നതിനുള്ള റഫറൻസ് നടപ്പാക്കല്‍.

ലോകൽ വികസനത്തിനിടെ MCP ആപ്പുകൾ പരിശോധിക്കാന് ഈ അടിസ്ഥാന ഹോസ്റ്റും ഉപയോഗിക്കാവുന്നതാണ്.

## പ്രധാന ഫയലുകൾ

- [`index.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/index.html) / [`src/index.tsx`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/index.tsx) - ടൂൾ തെരഞ്ഞെടുത്ത്, പാരാമീറ്റർ നൽകുകയും iframe മാനേജ്‌മെന്റ് ചെയ്തും നടത്തുന്ന React UI ഹോസ്റ്റ്
- [`sandbox.html`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/sandbox.html) / [`src/sandbox.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/sandbox.ts) - സുരക്ഷാ പരിശോധനയും ഇരുവഴിക്കുമാറ്റം സന്ദേശ പ്രക്ഷേപണവും ഉള്ള പുറമേ iframe പ്രോക്സി
- [`src/implementation.ts`](../../../../../../../03-GettingStarted/15-mcp-apps/ext-apps/examples/basic-host/src/implementation.ts) - മെയിൻ ലൊജിക്: സർവർ കണക്ഷൻ, ടൂൾ കോൾ ചെയ്യൽ, AppBridge നിർമ്മാണം

## തുടങ്ങുന്ന വിധം

```bash
npm install
npm run start
# തുറക്കുക http://localhost:8080
```

ഡിഫോൾട്ടായി, ഹോസ്റ്റ് അപ്ലിക്കേഷൻ `http://localhost:3001/mcp` എന്ന MCP സർവറുമായി കണക്റ്റ് ചെയ്യാൻ ശ്രമിക്കും. `SERVERS` എൻവയോൺമെന്റ് വേരിയബിൾ സെറ്റ് ചെയ്ത് ഒരു JSON അടങ്ങിയ സർവർ URL കളുടെ ശ്രേണി ക്രമീകരിക്കാം:

```bash
SERVERS='["http://localhost:1234/mcp", "http://localhost:5678/mcp"]' npm run start
```

## വാസ്തുശില്പം

സുരക്ഷിത UI ഐസൊലേഷനായി ഈ ഉദാഹരണം ഇരട്ട iframe സാൻഡ്ബോക്സ് മാതൃക ഉപയോഗിക്കുന്നു:

```
Host (port 8080)
  └── Outer iframe (port 8081) - sandbox proxy
        └── Inner iframe (srcdoc) - untrusted tool UI
```

**എന്തിനാണ് രണ്ട് iframe കളാണത്?**

- പുറത്തെ iframe വേറെ ഉറവിടത്തിൽ (പോർട്ട് 8081) പ്രവർത്തിച്ച് ഹോസ്റ്റിലേക്ക് നേരിട്ട് ആക്സസ് തടയുന്നു
- ഉള്ളിലെ iframe `srcdoc` വഴി HTML സ്വീകരിക്കുന്നു, സാൻഡ്ബോക്സ് ആട്രിബ്യൂട്ടുകൾ കൊണ്ട് നിയന്ത്രിക്കപ്പെടുന്നു
- സന്ദേശങ്ങൾ പുറത്തെ iframe വഴി ഒഴുകുന്നു, അത് പരിശോധിച്ച് ഇരുവഴി പ്രക്ഷേപണം നടത്തുന്നു

ഈ വാസ്തുശില്പം ഉറപ്പാക്കുന്നു ടൂൾ UI കോഡ് ദുഷ്ഫലമായാലും ഹോസ്റ്റ് അപ്ലിക്കേഷന്റെ DOM, കുക്കീസ്, ജാവാസ്ക്രിപ്റ്റ് സാന്ദർഭം എന്നിവ ആക്സസ് ചെയ്യാൻ അയാൾക്ക് കഴിയുകയില്ല.
