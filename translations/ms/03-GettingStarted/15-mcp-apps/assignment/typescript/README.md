## Bagaimana penyelesaian ini disusun

Penyelesaian ini terdiri hanya daripada bahagian kod yang penting, iaitu markup, sambungan acara dan ciri pelayan.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
