## Kako je ta rešitev strukturirana

Rešitev vsebuje samo tisti del kode, ki je pomemben, tj. označevalec, povezavo dogodkov in strežniške funkcije.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```

## Kako zagnati

Oglejte si [Koda](../../code/typescript/README.md) in dodajte vsebino iz teh datotek v vsako ustrezno datoteko.
