## Cách giải pháp này được tổ chức

Giải pháp chỉ bao gồm phần mã quan trọng, tức là phần đánh dấu, kết nối sự kiện và các tính năng máy chủ.

```text
my-app
  server.ts -- the server functionality
  src
    mcp-app.ts -- UI, event wire up
  mcp-app.html -- UI markup
```
