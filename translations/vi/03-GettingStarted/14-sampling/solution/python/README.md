# Chạy mẫu này

Bạn nên cài đặt `uv` nhưng không bắt buộc, xem [hướng dẫn](https://docs.astral.sh/uv/#highlights)

## -0- Tạo môi trường ảo

```bash
python -m venv venv
```

## -1- Kích hoạt môi trường ảo

```bash
venv\Scripts\activate
```

## -2- Cài đặt các phụ thuộc

```bash
pip install "mcp[cli]" openai
```

## -3- Chạy mẫu


```bash
python client.py
```
