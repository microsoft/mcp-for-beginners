## ਦਸੰਬਰ 18, 2025

### ਸੁਰੱਖਿਆ ਦਸਤਾਵੇਜ਼ ਅੱਪਡੇਟ - MCP ਵਿਸ਼ੇਸ਼ਤਾ 2025-11-25

#### MCP ਸੁਰੱਖਿਆ ਬਿਹਤਰ ਅਭਿਆਸ (02-Security/mcp-best-practices.md) - ਵਿਸ਼ੇਸ਼ਤਾ ਵਰਜਨ ਅੱਪਡੇਟ
- **ਪ੍ਰੋਟੋਕੋਲ ਵਰਜਨ ਅੱਪਡੇਟ**: ਨਵੀਂ MCP ਵਿਸ਼ੇਸ਼ਤਾ 2025-11-25 (ਜੋ 25 ਨਵੰਬਰ, 2025 ਨੂੰ ਜਾਰੀ ਹੋਈ) ਨੂੰ ਦਰਸਾਉਂਦਾ ਅੱਪਡੇਟ ਕੀਤਾ
  - ਸਾਰੇ ਵਿਸ਼ੇਸ਼ਤਾ ਵਰਜਨ ਸੰਦਰਭ 2025-06-18 ਤੋਂ 2025-11-25 ਤੱਕ ਅੱਪਡੇਟ ਕੀਤੇ
  - ਦਸਤਾਵੇਜ਼ ਦੀਆਂ ਤਾਰੀਖਾਂ ਅਗਸਤ 18, 2025 ਤੋਂ ਦਸੰਬਰ 18, 2025 ਤੱਕ ਅੱਪਡੇਟ ਕੀਤੀਆਂ
  - ਸਾਰੇ ਵਿਸ਼ੇਸ਼ਤਾ URLs ਨੂੰ ਮੌਜੂਦਾ ਦਸਤਾਵੇਜ਼ੀਕਰਨ ਵੱਲ ਸਹੀ ਤਰੀਕੇ ਨਾਲ ਪੁਆਇੰਟ ਕੀਤਾ ਗਿਆ
- **ਸਮੱਗਰੀ ਦੀ ਪੁਸ਼ਟੀ**: ਨਵੀਂਆਂ ਮਿਆਰਾਂ ਦੇ ਮੁਤਾਬਕ ਸੁਰੱਖਿਆ ਬਿਹਤਰ ਅਭਿਆਸਾਂ ਦੀ ਵਿਸਥਾਰਪੂਰਵਕ ਪੁਸ਼ਟੀ ਕੀਤੀ
  - **ਮਾਈਕ੍ਰੋਸਾਫਟ ਸੁਰੱਖਿਆ ਹੱਲ**: ਪ੍ਰੰਪਟ ਸ਼ੀਲਡਜ਼ (ਪਹਿਲਾਂ "ਜੇਲਬ੍ਰੇਕ ਖਤਰੇ ਦੀ ਪਹਿਚਾਣ"), ਅਜ਼ੂਰ ਕੰਟੈਂਟ ਸੇਫਟੀ, ਮਾਈਕ੍ਰੋਸਾਫਟ ਐਂਟਰਾ ID, ਅਤੇ ਅਜ਼ੂਰ ਕੀ ਵਾਲਟ ਲਈ ਮੌਜੂਦਾ ਟਰਮੀਨੋਲੋਜੀ ਅਤੇ ਲਿੰਕਾਂ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ
  - **OAuth 2.1 ਸੁਰੱਖਿਆ**: ਨਵੀਂ OAuth ਸੁਰੱਖਿਆ ਬਿਹਤਰ ਅਭਿਆਸਾਂ ਨਾਲ ਸਹਿਮਤੀ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ
  - **OWASP ਮਿਆਰ**: LLMs ਲਈ OWASP ਟੌਪ 10 ਸੰਦਰਭ ਮੌਜੂਦਾ ਹਨ
  - **ਅਜ਼ੂਰ ਸੇਵਾਵਾਂ**: ਸਾਰੇ ਮਾਈਕ੍ਰੋਸਾਫਟ ਅਜ਼ੂਰ ਦਸਤਾਵੇਜ਼ੀਕਰਨ ਲਿੰਕ ਅਤੇ ਬਿਹਤਰ ਅਭਿਆਸਾਂ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ
- **ਮਿਆਰਾਂ ਨਾਲ ਸਹਿਮਤੀ**: ਸਾਰੇ ਸੰਦਰਭਿਤ ਸੁਰੱਖਿਆ ਮਿਆਰ ਮੌਜੂਦਾ ਹਨ
  - NIST AI ਰਿਸਕ ਮੈਨੇਜਮੈਂਟ ਫਰੇਮਵਰਕ
  - ISO 27001:2022
  - OAuth 2.1 ਸੁਰੱਖਿਆ ਬਿਹਤਰ ਅਭਿਆਸ
  - ਅਜ਼ੂਰ ਸੁਰੱਖਿਆ ਅਤੇ ਅਨੁਕੂਲਤਾ ਫਰੇਮਵਰਕ
- **ਲਾਗੂ ਕਰਨ ਵਾਲੇ ਸਰੋਤ**: ਸਾਰੇ ਲਾਗੂ ਕਰਨ ਵਾਲੇ ਗਾਈਡ ਲਿੰਕ ਅਤੇ ਸਰੋਤਾਂ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ
  - ਅਜ਼ੂਰ API ਮੈਨੇਜਮੈਂਟ ਪ੍ਰਮਾਣਿਕਤਾ ਪੈਟਰਨ
  - ਮਾਈਕ੍ਰੋਸਾਫਟ ਐਂਟਰਾ ID ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਗਾਈਡ
  - ਅਜ਼ੂਰ ਕੀ ਵਾਲਟ ਸੀਕ੍ਰੇਟ ਮੈਨੇਜਮੈਂਟ
  - DevSecOps ਪਾਈਪਲਾਈਨ ਅਤੇ ਮਾਨੀਟਰਿੰਗ ਹੱਲ

### ਦਸਤਾਵੇਜ਼ ਗੁਣਵੱਤਾ ਯਕੀਨੀਕਰਨ
- **ਵਿਸ਼ੇਸ਼ਤਾ ਅਨੁਕੂਲਤਾ**: ਸਾਰੇ ਜ਼ਰੂਰੀ MCP ਸੁਰੱਖਿਆ ਲੋੜਾਂ (MUST/MUST NOT) ਨੂੰ ਨਵੀਂ ਵਿਸ਼ੇਸ਼ਤਾ ਨਾਲ ਸਹਿਮਤ ਕੀਤਾ
- **ਸਰੋਤ ਮੌਜੂਦਗੀ**: ਸਾਰੇ ਬਾਹਰੀ ਲਿੰਕ ਮਾਈਕ੍ਰੋਸਾਫਟ ਦਸਤਾਵੇਜ਼ੀਕਰਨ, ਸੁਰੱਖਿਆ ਮਿਆਰਾਂ ਅਤੇ ਲਾਗੂ ਕਰਨ ਵਾਲੇ ਗਾਈਡਾਂ ਵੱਲ ਸਹੀ ਹਨ
- **ਬਿਹਤਰ ਅਭਿਆਸ ਕਵਰੇਜ**: ਪ੍ਰਮਾਣਿਕਤਾ, ਅਧਿਕਾਰ, AI-ਵਿਸ਼ੇਸ਼ ਖਤਰੇ, ਸਪਲਾਈ ਚੇਨ ਸੁਰੱਖਿਆ, ਅਤੇ ਉਦਯੋਗ ਪੈਟਰਨਾਂ ਦੀ ਵਿਸਥਾਰਪੂਰਵਕ ਕਵਰੇਜ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ

## ਅਕਤੂਬਰ 6, 2025

### ਸ਼ੁਰੂਆਤ ਸੈਕਸ਼ਨ ਦਾ ਵਿਸਥਾਰ – ਉੱਨਤ ਸਰਵਰ ਵਰਤੋਂ ਅਤੇ ਸਧਾਰਣ ਪ੍ਰਮਾਣਿਕਤਾ

#### ਉੱਨਤ ਸਰਵਰ ਵਰਤੋਂ (03-GettingStarted/10-advanced)
- **ਨਵਾਂ ਅਧਿਆਇ ਸ਼ਾਮਲ ਕੀਤਾ**: ਉੱਨਤ MCP ਸਰਵਰ ਵਰਤੋਂ ਲਈ ਵਿਸਥਾਰਪੂਰਵਕ ਗਾਈਡ, ਜਿਸ ਵਿੱਚ ਨਿਯਮਤ ਅਤੇ ਨੀਵੀਂ ਪੱਧਰ ਦੀਆਂ ਸਰਵਰ ਆਰਕੀਟੈਕਚਰਾਂ ਨੂੰ ਕਵਰ ਕੀਤਾ ਗਿਆ।
  - **ਨਿਯਮਤ ਵਿਰੁੱਧ ਨੀਵੀਂ ਪੱਧਰ ਸਰਵਰ**: ਦੋਹਾਂ ਲਈ ਪਾਇਥਨ ਅਤੇ ਟਾਈਪਸਕ੍ਰਿਪਟ ਵਿੱਚ ਵਿਸਥਾਰਪੂਰਵਕ ਤੁਲਨਾ ਅਤੇ ਕੋਡ ਉਦਾਹਰਣਾਂ।
  - **ਹੈਂਡਲਰ-ਆਧਾਰਿਤ ਡਿਜ਼ਾਈਨ**: ਸਕੇਲਬਲ ਅਤੇ ਲਚਕੀਲੇ ਸਰਵਰ ਲਾਗੂ ਕਰਨ ਲਈ ਹੈਂਡਲਰ-ਆਧਾਰਿਤ ਟੂਲ/ਸਰੋਤ/ਪ੍ਰੰਪਟ ਪ੍ਰਬੰਧਨ ਦੀ ਵਿਆਖਿਆ।
  - **ਵਿਆਵਹਾਰਿਕ ਪੈਟਰਨ**: ਅਸਲੀ ਦੁਨੀਆ ਦੇ ਸੰਦਰਭ ਜਿੱਥੇ ਨੀਵੀਂ ਪੱਧਰ ਸਰਵਰ ਪੈਟਰਨ ਉੱਨਤ ਫੀਚਰਾਂ ਅਤੇ ਆਰਕੀਟੈਕਚਰ ਲਈ ਲਾਭਦਾਇਕ ਹਨ।

#### ਸਧਾਰਣ ਪ੍ਰਮਾਣਿਕਤਾ (03-GettingStarted/11-simple-auth)
- **ਨਵਾਂ ਅਧਿਆਇ ਸ਼ਾਮਲ ਕੀਤਾ**: MCP ਸਰਵਰਾਂ ਵਿੱਚ ਸਧਾਰਣ ਪ੍ਰਮਾਣਿਕਤਾ ਲਾਗੂ ਕਰਨ ਲਈ ਕਦਮ-ਦਰ-ਕਦਮ ਗਾਈਡ।
  - **ਪ੍ਰਮਾਣਿਕਤਾ ਸੰਕਲਪ**: ਪ੍ਰਮਾਣਿਕਤਾ ਅਤੇ ਅਧਿਕਾਰ ਦੇ ਵਿਚਕਾਰ ਸਪਸ਼ਟ ਵਿਆਖਿਆ ਅਤੇ ਪ੍ਰਮਾਣ ਪੱਤਰ ਸੰਭਾਲ।
  - **ਮੂਲ ਪ੍ਰਮਾਣਿਕਤਾ ਲਾਗੂ ਕਰਨ**: ਪਾਇਥਨ (Starlette) ਅਤੇ ਟਾਈਪਸਕ੍ਰਿਪਟ (Express) ਵਿੱਚ ਮਿਡਲਵੇਅਰ-ਆਧਾਰਿਤ ਪ੍ਰਮਾਣਿਕਤਾ ਪੈਟਰਨ, ਕੋਡ ਉਦਾਹਰਣਾਂ ਸਮੇਤ।
  - **ਉੱਨਤ ਸੁਰੱਖਿਆ ਵੱਲ ਪ੍ਰਗਤੀ**: ਸਧਾਰਣ ਪ੍ਰਮਾਣਿਕਤਾ ਨਾਲ ਸ਼ੁਰੂਆਤ ਕਰਨ ਅਤੇ OAuth 2.1 ਅਤੇ RBAC ਵੱਲ ਅੱਗੇ ਵਧਣ ਲਈ ਮਾਰਗਦਰਸ਼ਨ, ਉੱਨਤ ਸੁਰੱਖਿਆ ਮਾਡਿਊਲਾਂ ਦੇ ਸੰਦਰਭਾਂ ਸਮੇਤ।

ਇਹ ਸ਼ਾਮਲੀਆਂ MCP ਸਰਵਰ ਲਾਗੂ ਕਰਨ ਲਈ ਵਿਆਵਹਾਰਿਕ, ਹੱਥ-ਵਿੱਚ ਗਾਈਡ ਪ੍ਰਦਾਨ ਕਰਦੀਆਂ ਹਨ, ਜੋ ਮੂਲ ਸੰਕਲਪਾਂ ਨੂੰ ਉੱਨਤ ਉਤਪਾਦਨ ਪੈਟਰਨਾਂ ਨਾਲ ਜੋੜਦੀਆਂ ਹਨ।

#### ਮੁੱਖ ਧਾਰਣਾਵਾਂ (01-CoreConcepts/) - ਪੂਰੀ ਤਰ੍ਹਾਂ ਆਧੁਨਿਕੀਕਰਨ
- **ਪ੍ਰੋਟੋਕੋਲ ਵਰਜ਼ਨ ਅਪਡੇਟ**: ਮੌਜੂਦਾ MCP Specification 2025-06-18 ਨੂੰ ਸੰਦਰਭਿਤ ਕੀਤਾ ਗਿਆ, ਤਾਰੀਖ-ਆਧਾਰਿਤ ਵਰਜ਼ਨਿੰਗ (YYYY-MM-DD ਫਾਰਮੈਟ)
- **ਆਰਕੀਟੈਕਚਰ ਸੁਧਾਰ**: ਮੌਜੂਦਾ MCP ਆਰਕੀਟੈਕਚਰ ਪੈਟਰਨਾਂ ਨੂੰ ਦਰਸਾਉਂਦੇ ਹੋਏ ਹੋਸਟ, ਕਲਾਇੰਟ, ਅਤੇ ਸਰਵਰਾਂ ਦੇ ਵੇਰਵੇ ਸੁਧਾਰੇ ਗਏ
  - ਹੋਸਟ ਹੁਣ ਸਪਸ਼ਟ ਤੌਰ 'ਤੇ AI ਐਪਲੀਕੇਸ਼ਨਾਂ ਵਜੋਂ ਪਰਿਭਾਸ਼ਿਤ ਜੋ ਕਈ MCP ਕਲਾਇੰਟ ਕਨੈਕਸ਼ਨਾਂ ਦਾ ਸਹਿਯੋਗ ਕਰਦੀਆਂ ਹਨ
  - ਕਲਾਇੰਟ ਪ੍ਰੋਟੋਕੋਲ ਕਨੈਕਟਰ ਵਜੋਂ ਵਰਣਿਤ ਜੋ ਇੱਕ-ਤੋਂ-ਇੱਕ ਸਰਵਰ ਸੰਬੰਧਾਂ ਨੂੰ ਬਰਕਰਾਰ ਰੱਖਦੇ ਹਨ
  - ਸਰਵਰਾਂ ਨੂੰ ਸਥਾਨਕ ਅਤੇ ਦੂਰੀ ਵਾਲੇ ਤਾਇਨਾਤੀ ਦ੍ਰਿਸ਼ਾਂਕਾਂ ਨਾਲ ਸੁਧਾਰਿਆ ਗਿਆ
- **ਪ੍ਰਿਮਿਟਿਵ ਪੁਨਰਸੰਰਚਨਾ**: ਸਰਵਰ ਅਤੇ ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵਾਂ ਦਾ ਪੂਰਾ ਨਵੀਨੀਕਰਨ
  - ਸਰਵਰ ਪ੍ਰਿਮਿਟਿਵ: ਸਰੋਤ (ਡਾਟਾ ਸਰੋਤ), ਪ੍ਰਾਂਪਟ (ਟੈਮਪਲੇਟ), ਟੂਲ (ਚਲਾਉਣਯੋਗ ਫੰਕਸ਼ਨ) ਵਿਸਥਾਰਿਤ ਵਿਆਖਿਆ ਅਤੇ ਉਦਾਹਰਣਾਂ ਨਾਲ
  - ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵ: ਸੈਂਪਲਿੰਗ (LLM ਪੂਰਨਤਾ), ਇਲਿਸੀਟੇਸ਼ਨ (ਉਪਭੋਗਤਾ ਇਨਪੁੱਟ), ਲੌਗਿੰਗ (ਡਿਬੱਗਿੰਗ/ਨਿਗਰਾਨੀ)
  - ਮੌਜੂਦਾ ਖੋਜ (`*/list`), ਪ੍ਰਾਪਤੀ (`*/get`), ਅਤੇ ਚਲਾਉਣ (`*/call`) ਵਿਧੀ ਪੈਟਰਨਾਂ ਨਾਲ ਅਪਡੇਟ ਕੀਤਾ ਗਿਆ
- **ਪ੍ਰੋਟੋਕੋਲ ਆਰਕੀਟੈਕਚਰ**: ਦੋ-ਪੱਧਰੀ ਆਰਕੀਟੈਕਚਰ ਮਾਡਲ ਪੇਸ਼ ਕੀਤਾ
  - ਡਾਟਾ ਲੇਅਰ: JSON-RPC 2.0 ਬੁਨਿਆਦ ਨਾਲ ਜੀਵਨਚੱਕਰ ਪ੍ਰਬੰਧਨ ਅਤੇ ਪ੍ਰਿਮਿਟਿਵ
  - ਟ੍ਰਾਂਸਪੋਰਟ ਲੇਅਰ: STDIO (ਸਥਾਨਕ) ਅਤੇ Streamable HTTP ਨਾਲ SSE (ਦੂਰੀ) ਟ੍ਰਾਂਸਪੋਰਟ ਮਕੈਨਿਜ਼ਮ
- **ਸੁਰੱਖਿਆ ਫਰੇਮਵਰਕ**: ਸਪਸ਼ਟ ਉਪਭੋਗਤਾ ਸਹਿਮਤੀ, ਡਾਟਾ ਗੋਪਨੀਯਤਾ ਸੁਰੱਖਿਆ, ਟੂਲ ਐਗਜ਼ੀਕਿਊਸ਼ਨ ਸੁਰੱਖਿਆ, ਅਤੇ ਟ੍ਰਾਂਸਪੋਰਟ ਲੇਅਰ ਸੁਰੱਖਿਆ ਸਮੇਤ ਵਿਆਪਕ ਸੁਰੱਖਿਆ ਸਿਧਾਂਤ
- **ਸੰਚਾਰ ਪੈਟਰਨ**: ਸ਼ੁਰੂਆਤ, ਖੋਜ, ਚਲਾਉਣ, ਅਤੇ ਸੂਚਨਾ ਪ੍ਰਵਾਹ ਦਿਖਾਉਂਦੇ ਪ੍ਰੋਟੋਕੋਲ ਸੁਨੇਹੇ ਅਪਡੇਟ ਕੀਤੇ
- **ਕੋਡ ਉਦਾਹਰਣ**: ਮੌਜੂਦਾ MCP SDK ਪੈਟਰਨਾਂ ਨੂੰ ਦਰਸਾਉਂਦੇ ਹੋਏ ਬਹੁ-ਭਾਸ਼ਾਈ ਉਦਾਹਰਣ (.NET, ਜਾਵਾ, Python, ਜਾਵਾਸਕ੍ਰਿਪਟ) ਤਾਜ਼ਾ ਕੀਤੇ

#### ਸੁਰੱਖਿਆ (02-Security/) - ਵਿਆਪਕ ਸੁਰੱਖਿਆ ਨਵੀਨੀਕਰਨ  
- **ਮਿਆਰ ਅਨੁਕੂਲਤਾ**: MCP Specification 2025-06-18 ਸੁਰੱਖਿਆ ਲੋੜਾਂ ਨਾਲ ਪੂਰੀ ਤਰ੍ਹਾਂ ਅਨੁਕੂਲ
- **ਪ੍ਰਮਾਣਿਕਤਾ ਵਿਕਾਸ**: ਕਸਟਮ OAuth ਸਰਵਰਾਂ ਤੋਂ ਬਾਹਰੀ ਪਹਚਾਣ ਪ੍ਰਦਾਤਾ ਡੈਲੀਗੇਸ਼ਨ (Microsoft Entra ID) ਤੱਕ ਵਿਕਾਸ ਦਸਤਾਵੇਜ਼ ਕੀਤਾ
- **AI-ਵਿਸ਼ੇਸ਼ ਧਮਕੀ ਵਿਸ਼ਲੇਸ਼ਣ**: ਆਧੁਨਿਕ AI ਹਮਲੇ ਦੇ ਰਸਤੇ ਵਧੇਰੇ ਕਵਰੇਜ
  - ਅਸਲੀ ਦੁਨੀਆ ਦੇ ਉਦਾਹਰਣਾਂ ਨਾਲ ਵਿਸਥਾਰਿਤ ਪ੍ਰਾਂਪਟ ਇੰਜੈਕਸ਼ਨ ਹਮਲੇ ਦੇ ਦ੍ਰਿਸ਼
  - ਟੂਲ ਜਹਿਰੀਲਾ ਕਰਨ ਦੇ ਮਕੈਨਿਜ਼ਮ ਅਤੇ "ਰੱਗ ਪੁੱਲ" ਹਮਲੇ ਦੇ ਪੈਟਰਨ
  - ਸੰਦਰਭ ਵਿੰਡੋ ਜਹਿਰੀਲਾ ਕਰਨ ਅਤੇ ਮਾਡਲ ਗੁੰਝਲਦਾਰ ਹਮਲੇ
- **Microsoft AI ਸੁਰੱਖਿਆ ਹੱਲ**: Microsoft ਸੁਰੱਖਿਆ ਪਰਿਸਰ ਦੀ ਵਿਆਪਕ ਕਵਰੇਜ
  - AI Prompt Shields ਨਾਲ ਉन्नਤ ਪਤਾ ਲਗਾਉਣ, ਸਪੌਟਲਾਈਟਿੰਗ, ਅਤੇ ਡੈਲੀਮੀਟਰ ਤਕਨੀਕਾਂ
  - Azure Content Safety ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਪੈਟਰਨ
  - ਸਪਲਾਈ ਚੇਨ ਸੁਰੱਖਿਆ ਲਈ GitHub Advanced Security
- **ਉন্নਤ ਧਮਕੀ ਰੋਕਥਾਮ**: ਵਿਸਥਾਰਿਤ ਸੁਰੱਖਿਆ ਨਿਯੰਤਰਣ
  - MCP-ਵਿਸ਼ੇਸ਼ ਸੈਸ਼ਨ ਹਾਈਜੈਕਿੰਗ ਹਮਲੇ ਦੇ ਦ੍ਰਿਸ਼ ਅਤੇ ਕ੍ਰਿਪਟੋਗ੍ਰਾਫਿਕ ਸੈਸ਼ਨ ID ਲੋੜਾਂ ਨਾਲ
  - MCP ਪ੍ਰਾਕਸੀ ਦ੍ਰਿਸ਼ਾਂਕਾਂ ਵਿੱਚ Confused Deputy ਸਮੱਸਿਆਵਾਂ ਸਪਸ਼ਟ ਸਹਿਮਤੀ ਲੋੜਾਂ ਨਾਲ
  - ਲਾਜ਼ਮੀ ਪ੍ਰਮਾਣਿਕਤਾ ਨਿਯੰਤਰਣਾਂ ਨਾਲ ਟੋਕਨ ਪਾਸਥਰੂ ਕਮਜ਼ੋਰੀਆਂ
- **ਸਪਲਾਈ ਚੇਨ ਸੁਰੱਖਿਆ**: ਫਾਊਂਡੇਸ਼ਨ ਮਾਡਲ, ਐਂਬੈਡਿੰਗ ਸੇਵਾਵਾਂ, ਸੰਦਰਭ ਪ੍ਰਦਾਤਾ, ਅਤੇ ਤੀਜੀ-ਪੱਖੀ API ਸਮੇਤ AI ਸਪਲਾਈ ਚੇਨ ਕਵਰੇਜ ਵਧਾਇਆ
- **ਫਾਊਂਡੇਸ਼ਨ ਸੁਰੱਖਿਆ**: ਜ਼ੀਰੋ ਟਰੱਸਟ ਆਰਕੀਟੈਕਚਰ ਅਤੇ Microsoft ਸੁਰੱਖਿਆ ਪਰਿਸਰ ਸਮੇਤ ਉਦਯੋਗ ਸੁਰੱਖਿਆ ਪੈਟਰਨਾਂ ਨਾਲ ਸੁਧਾਰਿਆ ਇੰਟੀਗ੍ਰੇਸ਼ਨ
- **ਸਰੋਤ ਸੰਗਠਨ**: ਕਿਸਮ (ਅਧਿਕਾਰਿਕ ਦਸਤਾਵੇਜ਼, ਮਿਆਰ, ਖੋਜ, Microsoft ਹੱਲ, ਲਾਗੂ ਕਰਨ ਵਾਲੇ ਗਾਈਡ) ਅਨੁਸਾਰ ਵਿਆਪਕ ਸਰੋਤ ਲਿੰਕ ਵਰਗੀਕ੍ਰਿਤ
